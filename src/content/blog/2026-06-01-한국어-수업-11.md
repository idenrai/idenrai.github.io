# 수업 11. 왜요? 그래서요?

文をつなげる接続表現を学ぶ。これを覚えると一気に表現の幅が広がる。

## 1. ~고（〜して / 〜で）

2つの動作や状態を並列につなぐ。日本語の「〜して」「〜で」に相当。
語幹にそのまま`고`をつけるだけ。形による変化なし。

- 밥을 먹`고` 커피를 마셔요.（ご飯を食べ`て`コーヒーを飲みます。）
- 샤워하`고` 잠을 자요.（シャワーを浴び`て`寝ます。）
- 크`고` 예뻐요.（大き`くて`きれいです。）

> `~고`は前後の順序を表すこともあるが、単純な並列にも使える。時制は文末だけで表す。

- 밥을 먹고 커피를 `마셨어요`.（ご飯を食べてコーヒーを飲み`ました`。— 過去は文末だけ）

---

## 2. ~아서/어서（〜なので / 〜して）

**理由**または**先行動作**を表す。

語幹の最後の母音で使い分ける。
- 母音が`ㅏ/ㅗ` → `아서`
- それ以外 → `어서`
- `하다` → `해서`

### 理由・原因（〜なので）

- `피곤해서` 일찍 잤어요.（疲れ`たので`早く寝ました。）
- `바빠서` 못 갔어요.（忙しく`て`行けませんでした。）
- `맛있어서` 많이 먹었어요.（おいしく`て`たくさん食べました。）
- `비가 와서` 집에 있었어요.（雨が降っ`たので`家にいました。）

### 先行動作（〜して、それから）

- 역에 `가서` 친구를 만났어요.（駅に行`って`友達に会いました。）
- 집에 `와서` 밥을 먹었어요.（家に帰`って`ご飯を食べました。）

> **重要**：`~아서/어서`は`~고`と違い、**前節の時制を変えてはいけない**。
> ❌ `피곤했어서 잤어요` → ⭕ `피곤해서 잤어요`

---

## 3. ~고 と ~아서/어서 の比較

| | ~고 | ~아서/어서 |
| ---- | ---- | ---- |
| 意味 | 〜して（並列・順序） | 〜なので / 〜して（理由・原因） |
| 前節の時制変化 | 文末のみ | **前節は変えない** |
| 例 | 밥을 먹고 잤어요. | 배가 불러서 잤어요. |

---

## 4. 접속사（接続詞）

文と文をつなぐ接続詞。会話の流れを作るのに必須。

| 韓国語 | 意味 | 使う場面 |
| ---- | ---- | ---- |
| 그래서 | だから / それで | 前の文が理由 |
| 그런데 | でも / ところで | 逆接・話題転換 |
| 그리고 | そして / それから | 並列・追加 |
| 그러면 / 그럼 | それなら / じゃあ | 条件・話の展開 |
| 그래도 | それでも | 逆接（譲歩） |

- 배가 고파요. `그래서` 밥을 먹었어요.（お腹がすいています。`だから`ご飯を食べました。）
- 오늘 바빠요. `그런데` 내일은 괜찮아요.（今日は忙しい。`でも`明日は大丈夫。）
- 한국어를 공부했어요. `그리고` 드라마도 봤어요.（韓国語を勉強しました。`そして`ドラマも見ました。）
- 비가 와요. `그래도` 갈 거예요.（雨が降っています。`それでも`行くつもりです。）

---

## 5. 理由だけ答えるとき

理由だけを答える場合、`~아서/어서요`で文末を止めるのが自然。

- 왜 한국어를 공부해요?（なぜ韓国語を勉強してるんですか？）
  - 한국 드라마를 좋아`해서요`.（韓国ドラマが好き`だからです`。）
  - 한국에 가고 싶`어서요`.（韓国に行きたい`からです`。）
  - 한국 가족이 있`어서요`.（韓国に家族がいる`からです`。）

---

## 練習

2つの文を`~고`または`~아서/어서`でつなげてみましょう。

| 2つの文 | つなげ方 |
| ---- | ---- |
| 밥을 먹었어요 + 커피를 마셨어요 | ~고 |
| 피곤했어요 + 일찍 잤어요 | ~아서/어서 |
| 마트에 갔어요 + 장을 봤어요 | ~아서/어서 |
| 날씨가 좋아요 + 기분이 좋아요 | ~아서/어서 |

**정답**

1. 밥을 먹고 커피를 마셨어요.
2. 피곤해서 일찍 잤어요.
3. 마트에 가서 장을 봤어요.
4. 날씨가 좋아서 기분이 좋아요.

---

## Tips

### ~아서/어서 の前に過去形を使わない理由

`~아서/어서`は前後の文が**因果関係**で結ばれているため、時制は文全体の末尾で一度だけ表現する。

| ❌ 間違い | ⭕ 正しい |
| ---- | ---- |
| 피곤**했**어서 잤어요 | 피곤**해서** 잤어요 |
| 바**빴**어서 못 갔어요 | 바**빠서** 못 갔어요 |

この規則は`~아서/어서`特有のもの。`~고`には適用されない。
