# 수업 8. 날씨가 어때요?

今回は**形容詞**を学ぶ。  
韓国語の形容詞は動詞とほぼ同じルールで활용（活用）する。現在形・過去形・疑問形もそのまま使えるので、動詞を覚えていれば応用は簡単。

---

## 1. 형용사의 해요체（形容詞の丁寧形）

動詞と同様に、語幹の最後の母音で`아요/어요`を使い分ける。

| 原型 | 해요체 | 意味 |
|------|--------|------|
| 크다 | 커요 | 大きいです |
| 작다 | 작아요 | 小さいです |
| 많다 | 많아요 | 多いです |
| 적다 | 적어요 | 少ないです |
| 좋다 | 좋아요 | 良いです / 好きです |
| 나쁘다 | 나빠요 | 悪いです |
| 예쁘다 | 예뻐요 | 綺麗です / かわいいです |
| 귀엽다 | 귀여워요 | 可愛いです |
| 멋있다 | 멋있어요 | かっこいいです |
| 재미있다 | 재미있어요 | 面白いです |
| 재미없다 | 재미없어요 | つまらないです |
| 맛있다 | 맛있어요 | おいしいです |
| 맛없다 | 맛없어요 | まずいです |

---

## 2. 날씨 표현（天気の表現）

### 날씨가 어때요?（天気はどうですか？）

| 韓国語 | 意味 |
|--------|------|
| 더워요 | 暑いです |
| 추워요 | 寒いです |
| 따뜻해요 | 暖かいです |
| 시원해요 | 涼しいです |
| 맑아요 | 晴れています |
| 흐려요 | 曇っています |
| 비가 와요 | 雨が降っています |
| 눈이 와요 | 雪が降っています |
| 바람이 불어요 | 風が吹いています |

```text
오늘 날씨가 어때요?      → 今日の天気はどうですか？
  → 너무 더워요.         → とても暑いです。
  → 맑고 시원해요.       → 晴れていて涼しいです。
  → 비가 와요.           → 雨が降っています。
```

## 3. 형용사 과거형（形容詞の過去形）

動詞と同様に`았어요/었어요`をつける。

| 現在形 | 過去形 | 意味 |
|--------|--------|------|
| 더워요 | 더웠어요 | 暑かったです |
| 추워요 | 추웠어요 | 寒かったです |
| 좋아요 | 좋았어요 | 良かったです |
| 맛있어요 | 맛있었어요 | おいしかったです |
| 재미있어요 | 재미있었어요 | 面白かったです |
| 예뻐요 | 예뻤어요 | 綺麗でした |

---

## 4. 어때요?（どうですか？）

感想や状態を聞く万能フレーズ。

```text
어때요?                  → どうですか？
이거 어때요?             → これどうですか？
한국어 공부가 어때요?    → 韓国語の勉強はどうですか？
한국 음식은 어때요?      → 韓国料理はどうですか？
```

### 答え方

```text
재미있어요!              → 面白いです！
조금 어려워요.           → 少し難しいです。
생각보다 쉬워요.         → 思ったより簡単です。
너무 매워요.             → とても辛いです。
```

---

## 5. 비교 표현（比較の表現）

### 더（もっと）/ 덜（より少なく）

```text
이게 더 커요.            → こっちの方が大きいです。
이게 더 싸요.            → こっちの方が安いです。
이게 덜 매워요.          → こっちの方が辛くないです。
```

### A보다 B가 더 ~（AよりBの方が〜）

```text
한국어가 일본어보다 더 쉬워요.
→ 韓国語は日本語より簡単です。

오늘이 어제보다 더 더워요.
→ 今日は昨日より暑いです。
```

## Tips

### 좋아요의 두 가지 의미

`좋아요`は文脈によって意味が変わる。

```text
날씨가 좋아요.           → 天気がいいです。（状態）
한국어가 좋아요.         → 韓国語が好きです。（好み）
좋아요!                  → いいですよ！/ OKです！（同意）
```

### ~고（〜くて / 〜で）

形容詞を2つつなげるときは`고`を使う。

```text
크고 예뻐요.             → 大きくて綺麗です。
맛있고 싸요.             → おいしくて安いです。
따뜻하고 맑아요.         → 暖かくて晴れています。
```
