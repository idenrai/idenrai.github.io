# 수업 27. 제가 도와줄게요

오늘은 대화 상대방에게 자신의 행동이나 의지를 다짐하고 약속할 때 쓰는 **~(으)ㄹ게요** 와,
동사를 명사로 만들어 약속할 때 쓰는 **~기 약속해요**를 배운다.

---

## 1. 의지/약속 관련 어휘 (의지・約束의 語彙)

| 한국어 | 일본어 | 예문 |
|--------|--------|------|
| 약속 (약속하다) | 約束 (約束する) | 친구와 약속을 했어요. |
| 전화 (전화하다) | 電話 (電話する) | 이따가 부모님께 전화할게요. |
| 돕다 | 돕는 것 / 手伝う | 제가 청소를 도와줄게요. |
| 문자 (문자를 보내다) | メッセージ (メッセージを送る) | 도착하면 문자를 보내 주세요. |
| 연락 (연락하다) | 連絡 (連絡する) | 나중에 다시 연락할게요. |

---

## 2. ~(으)ㄹ게요 (〜합니다 / 〜할게요 - 話し手の意思・約束)

말하는 이가 상대방에게 자신의 의지를 알리거나, 어떤 행동을 하겠다고 확실하게 **약속**할 때 사용한다. (오직 주어가 **1인칭(나, 저)**일 때만 쓸 수 있으며, 문장의 끝에 물음표(?)를 붙여 질문할 수 없다.)

- 받침이 없을 때: **~ㄹ게요** (예: 가다 → 갈게요)
- 받침이 있을 때: **~을게요** (예: 먹다 → 먹을게요)
- 'ㄹ' 받침으로 끝날 때: **~ㄹ게요** (예: 만들다 → 만들게요)
- 'ㅂ' 불규칙 동사: **~울게요** (예: 돕다 → 도울게요 / 도와줄게요)

```text
제가 오늘 저녁밥을 만들게요.
  → 私が今日の晩ご飯を作ります。（作ると約束します。）

내일 꼭 9시까지 갈게요.
  → 明日必ず9時までに行きます。

이따가 연락할게요.
  → 後で連絡します。
```

---

## 3. ~기 약속해요 / ~기로 해요 (〜することを約束します / 〜することにします)

동사 뒤에 `~기`를 붙여 동사를 명사 형태로 만든 후, 상대방과 약속이나 다짐을 할 때 쓴다.

### 1) ~기 약속해요 (〜することを約束します)
동사 어간 + **~기 약속해요** (예: 늦지 않다 → 늦지 않기 약속해요)

```text
매일 한국어를 공부하기 약속해요.
  → 毎日韓国語を勉強することを約束します。

내일은 늦지 않기 약속해요.
  → 明日は遅れないことを約束しましょう。
```

### 2) ~기로 해요 (〜することにしましょう - 決定・約束)
동사 어간 + **~기로 해요** (예: 만나다 → 만나기로 해요, 먹다 → 먹기로 해요)

```text
내일 서울역에서 만나기로 해요.
  → 明日ソウル駅で会うことにしましょう。

오늘 저녁은 외식하기로 해요.
  → 今日の夕食は外食することにしましょう。
```

## 4. 회화 연습

**상황: 저녁 준비와 약속 (夕食の準備と約束)**

```text
아내: 오늘 회사 일이 너무 많아서 피곤하네요.
     (今日は仕事が多すぎて疲れますね。)

남편: 그래요? 그럼 오늘 저녁 요리는 제가 할게요.
     (そうですか？じゃあ今日の夕食の料理は私がします。)

아내: 정말요? 고마워요. 그럼 제가 설거지를 할게요.
     (本当ですか？ありがとうございます。じゃあ私が皿洗いをします。)

남편: 아니에요, 설거지도 제가 할게요. 유나 씨는 그냥 거실에서 쉬세요.
     (いいえ, 皿洗いも私がします。ユナさんはただリビングで休んでください。)

아내: 미안해서 어쩌죠? 대신 주말에는 제가 맛있는 음식을 만들어 줄게요.
     (申し訳なくてどうしましょう？代わりに週末には私が美味しいものを作ってあげます。)

남편: 와, 신난다! 주말에 꼭 만들어 주기 약속해요.
     (わあ, 楽しみ！週末に必ず作ってくれることを約束してね。)

아내: 네, 약속할게요!
     (はい, 約속할게요!)
```

## 練習

**문장의 빈칸에 들어갈 알맞은 단어를 만들어 보세요. (文章의 空欄に入る正しい단어를 만들어 보세요.)**

1. 제가 먼저 집으로 ________. (가다 - 의지/약속)
2. 내일부터 담배를 안 ________ 약속해요. (피우다 - 약속)
3. 주말에 같이 미술관에 ________ 해요. (가다 - 결정)
4. 걱정하지 마세요. 제가 꼭 ________. (도와주다 - 의지/약속)

**정답 (答え)**
1. 갈게요
2. 피우기
3. 가기로
4. 도와줄게요

## Tips

### '~ㄹ게요'와 '~ㄹ 거예요'의 차이
- **~ㄹ 거예요** (수업 13 복습): 말하는 이의 미래 **계획이나 예정**을 전달한다. (상대방에 대한 약속의 뉘앙스가 약함. 제3자도 주어로 올 수 있음. 예: 비가 올 거예요.)
- **~ㄹ게요**: 듣는 사람에 대한 확실한 **다짐이나 약속**을 의미한다. (오직 주어가 '내'가 될 때만 씀.)
