# 수업 23. 저랑 같이 커피 마실래요?

오늘은 부드러운 제안이나 상대방의 의사를 물어볼 때 쓰는 **~(으)ㄹ래요?** 와,
미안함을 나타내는 표현, 그리고 행동을 금지하는 **~지 마세요**를 배운다.

---

## 1. 약속/거절 관련 어휘 (約束・断りの語彙)

| 한국어 | 일본어 | 예문 |
|--------|--------|------|
| 바쁘다 | 忙しい | 요즘 회사 일이 너무 바빠요. |
| 괜찮다 | 大丈夫だ | 내일은 시간 괜찮아요. |
| 미안하다 / 죄송하다 | 申し訳ない / すみません | 늦어서 정말 미안해요. |
| 약속이 있다/없다 | 予定がある / ない | 오늘 약속이 있어서 못 만나요. |
| 시간이 있다/없다 | 時間がある / ない | 지금 대화할 시간 있어요? |

---

## 2. ~(으)ㄹ래요? (〜しますか？ / 〜したいですか？ - 意思・提案)

주로 친밀한 사이에서 상대방에게 어떤 일을 같이 하자고 제안하거나, 상대방의 의사·선호를 부드럽게 물어볼 때 사용한다. (질문자 본인의 의사를 상대방에게 말할 때도 쓴다.)

- 받침이 없을 때: **~ㄹ래요?** (예: 마시다 → 마실래요?)
- 받침이 있을 때: **~을래요?** (예: 먹다 → 먹을래요?)
- 'ㄹ' 받침으로 끝날 때: **~ㄹ래요?** (예: 놀다 → 놀래요? - 'ㄹ' 중복 생략)

```text
오늘 저녁에 저랑 같이 영화 볼래요?
  → 今日、私と一緒に 영화 볼래요?

A: 뭐 마실래요?
B: 저는 따뜻한 아메리카노 마실래요.

내일 우리 집에 놀러 올래요?
  → 明日、私の家に遊びに来ませんか？
```

---

## 3. ~아/어서 죄송해요 / 미안해요 (〜してすみません)

이유를 나타내는 `~아/어서`(수업 19 복습)와 죄송하다/미안하다를 결합해, 미안한 이유를 구체적으로 정중하게 밝힐 때 쓴다.

```text
늦어서 정말 죄송합니다.
  → 遅れて本当に申し訳ありません。

약속을 어겨서 미안해요.
  → 約束を破ってごめんなさい。

전화가 늦어서 미안해요.
  → 電話が遅くなってごめんなさい。
```

※ 과거형으로 쓰지 않도록 주의하자! (❌ ~~늦었어서 미안해요~~ → ✅ **늦어서 미안해요**)

---

## 4. ~지 마세요 (〜しないでください - 禁止)

상대방에게 특정한 행동을 하지 말라고 부드럽게 지시하거나 요청할 때 쓴다.
동사 어간에 받침 유무와 상관없이 **~지 마세요**를 붙인다.

- 가다 → 가지 마세요
- 걱정하다 → 걱정하지 마세요
- 먹다 → 먹지 마세요

```text
길이 미끄러우니까 뛰지 마세요.
  → 道が滑りやすいので走らないでください。

아직 결정하지 마세요.
  → 아직 결정하지 마세요.

걱정하지 마세요. 제가 도와줄게요.
  → 心配しないでください。私が手伝います。
```

## 5. 회화 연습

**상황: 제안과 거절 (提案と断り)**

```text
남편: 유나 씨, 오늘 저녁에 퇴근하고 삼겹살 먹으러 갈래요?
     (ユナさん、今日の夜仕事帰りにサムギョプサル食べに行きませんか？)

아내: 아, 정말 가고 싶지만 오늘은 약속이 있어서 안 돼요.
     (아, 정말 가고 싶지만 오늘은 약속이 있어서 안 돼요.)

남편: 아, 그래요? 무슨 약속이 있어요?
     (아, 그래요? 何の予定があるんですか？)

아내: 오늘 친구 생일이라서 저녁에 만나기로 했어요. 같이 못 가서 미안해요.
     (今日は友達の誕生日なので、夜に会うことになりました。一緒に行けなくてごめんなさい。)

남편: 아니에요, 미안해하지 마세요! 친구랑 즐거운 시간 보내세요.
     (いいえ、申し訳なく思わないでください！友達と楽しい時間を送ってくださいね。)

아내: 고마워요. 내일 저녁에는 시간 괜찮으니까 내일 같이 마실래요?
     (ありがとうございます。明日の夜は時間が大丈夫なので、明日一緒に飲みますか？)

남편: 좋아요! 내일 만나요.
     (いいですね！明日会いましょう。)
```

## 練習

**문장의 빈칸에 들어갈 알맞은 단어를 만들어 보세요. (文章の空欄に入る正しい単語を作ってみましょう。)**

1. 저는 매운 음식을 안 좋아해서 안 ________. (먹다 - 의사)
2. 약속 시간에 늦어서 ________. (미안하다 - 이유)
3. 극장 안에서는 사진을 ________ 마세요. (찍다 - 금지)
4. 오늘 우리 집에 놀러 ________? (오다 - 제안)

**정답 (答え)**
1. 먹을래요
2. 미안해요 (또는 죄송해요)
3. 찍지
4. 올래요

---

## Tips

### '죄송해요'와 '미안해요'의 차이
- **죄송합니다 / 죄송해요**: 윗사람이나 격식을 차려야 하는 관계(예: 직장 상사, 시부모님)에 쓴다.
- **미안합니다 / 미안해요**: 동료, 나이가 비슷하거나 아랫사람, 혹은 친밀한 관계(예: 부부, 친구)에 쓴다.
