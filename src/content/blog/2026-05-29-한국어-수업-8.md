# 수업 8. 날씨가 어때요?

形容詞の해요체と否定表現を学ぶ。動詞の해요체と作り方は同じ。

## 1. 형용사 해요체（形容詞の해요체）

形容詞も動詞と同じく、語幹の最後の母音で`아요/어요`を使い分ける。

| 原型 | 해요체 | 意味 |
| ---- | ---- | ---- |
| 크다 | 커요 | 大きいです |
| 작다 | 작아요 | 小さいです |
| 좋다 | 좋아요 | いいです |
| 나쁘다 | 나빠요 | 悪いです |
| 많다 | 많아요 | 多いです |
| 적다 | 적어요 | 少ないです |
| 비싸다 | 비싸요 | 高い（値段）です |
| 싸다 | 싸요 | 安いです |
| 맛있다 | 맛있어요 | おいしいです |
| 맛없다 | 맛없어요 | まずいです |

### 하다 形容詞

| 原型 | 해요체 | 意味 |
| ---- | ---- | ---- |
| 깨끗하다 | 깨끗해요 | きれいです（清潔） |
| 조용하다 | 조용해요 | 静かです |
| 따뜻하다 | 따뜻해요 | 暖かいです |
| 시원하다 | 시원해요 | 涼しいです / さっぱりです |

---

## 2. 날씨 표현（天気の表現）

| 韓国語 | 意味 |
| ---- | ---- |
| 날씨가 좋아요. | 天気がいいです。 |
| 날씨가 나빠요. | 天気が悪いです。 |
| 더워요. | 暑いです。 |
| 추워요. | 寒いです。 |
| 비가 와요. | 雨が降っています。 |
| 눈이 와요. | 雪が降っています。 |
| 바람이 불어요. | 風が吹いています。 |
| 흐려요. | 曇っています。 |
| 맑아요. | 晴れています。 |

> `더워요`は原型`덥다`、`추워요`は原型`춥다`。
> いずれもㅂ불규칙（ㅂ不規則活用）で、今後수업 17で詳しく学ぶ。

---

## 3. 감정/상태 형용사（感情・状態の形容詞）

人の状態を表す形容詞。

| 原型 | 해요체 | 意味 |
| ---- | ---- | ---- |
| 기쁘다 | 기뻐요 | 嬉しいです |
| 슬프다 | 슬퍼요 | 悲しいです |
| 피곤하다 | 피곤해요 | 疲れています |
| 졸리다 | 졸려요 | 眠いです |
| 배고프다 | 배고파요 | お腹が空いています |
| 배부르다 | 배불러요 | お腹がいっぱいです |
| 바쁘다 | 바빠요 | 忙しいです |
| 무섭다 | 무서워요 | 怖いです |

---

## 4. 부정 표현（否定表現）

### 안 ~ （〜しない / 〜でない）

動詞・形容詞の前に`안`を置く。

- `안` 먹어요.（食べません。）
- `안` 추워요.（寒くないです。）
- `안` 바빠요.（忙しくないです。）

> `하다`動詞は「`안` 공부해요」ではなく、`공부` `안` `해요`と言うのが自然。

### ~지 않아요（〜ではありません）

より丁寧・書き言葉寄りの否定表現。動詞の語幹に`지 않아요`をつける。

- 먹`지 않아요`.（食べません。）
- 춥`지 않아요`.（寒くありません。）

> 日常会話では`안`が圧倒的に多い。

---

## 5. 정도 표현（程度の表現）

| 韓国語 | 意味 | 例 |
| ---- | ---- | ---- |
| 아주 / 매우 | とても | 아주 좋아요.（とてもいいです。） |
| 너무 | 〜すぎる / とても | 너무 더워요.（暑すぎます。） |
| 정말 / 진짜 | 本当に | 진짜 맛있어요.（本当においしい。） |
| 조금 / 좀 | 少し | 좀 추워요.（ちょっと寒い。） |
| 별로 | あまり（〜ない） | 별로 안 비싸요.（あまり高くない。） |

## 練習

以下の日本語を韓国語にしてみましょう。

```text
1. 今日は天気がいいです。
2. この食べ物はおいしいです。
3. 今日はあまり寒くないです。
4. とても疲れています。
5. 少しお腹が空いています。
```

**정답**

1. 오늘은 날씨가 좋아요.
2. 이 음식은 맛있어요.
3. 오늘은 별로 안 추워요.
4. 너무 피곤해요.
5. 좀 배고파요.

## Tips

### 형용사と동사の違い

韓国語の形容詞は文法的には「動詞」に近い。活用方法がほぼ同じ。
ただし形容詞は**進行形にできない**。

| | 動詞 | 形容詞 |
| ---- | ---- | ---- |
| 活用 | 同じ | 同じ |
| 進行形 | ⭕ 먹고 있어요 | ❌ 크고 있어요 |
| 過去形 | 갔어요 | 좋았어요 |
