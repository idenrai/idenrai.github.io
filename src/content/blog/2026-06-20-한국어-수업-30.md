# 수업 30. 그동안 수고하셨습니다! 종합 복습

いよいよ最後の授業である **授業 30** に到達した！
これまで授業1から授業29まで学んだ核心的な韓国語文法と便利なパターンを総合整理して復習してみよう。

## 1. 핵심 조사 및 기본 어미 정리（核心助詞と基本語尾の整理）

| 文法項目 | 表現形態 | 主な使い道 | 例文 |
| ---- | ---- | ---- | ---- |
| 名詞/叙述 | ~예요 / 이에요 | 〜です | 저는 유나**예요**. |
| 主題の提示 | ~은 / 는 | 〜は | 고양이**는** 있어요. |
| 目的語 | ~을 / 를 | 〜を | 커피**를** 마셔요. |
| 場所 | ~에 / 에서 | 〜に / 〜で | 2층**에** 있어요. / 회사**에서** 일해요. |
| 動詞の基本形 | ~아요 / 어요 | 해요体 (丁寧語) | 밥을 먹**어요**. |

## 2. 주요 시제 및 조동사 정리（主要な時制と助動詞）

これまで学習した核心文法の形態と意味を比較しながら思い出してみよう。

- **과거형 (過去形)**: ~았어요 / 었어요 / 했어요 (수업 7)
  - 어제 영화를 `봤어요`.（昨日映画を見ました。）
- **미래/예정 (未来/予定)**: ~(으)ㄹ 거예요 (수업 13)
  - 내일 도서관에 `갈 거예요`.（明日図書館に行くつもりです。）
- **희망/욕구 (希望/欲求)**: ~고 싶어요 (수업 9)
  - 비빔밥을 `먹고 싶어요`.（ビビンバが食べたいです。）
- **가능/불가능 (可能/不可能)**: ~(으)ㄹ 수 있어요 / 없어요 (수업 10)
  - 저는 운전`할 수 있어요`.（私は運転できます。）
- **의무 (義務)**: ~아야 / 어야 해요 (수업 12)
  - 집에서 푹 쉬`어야 해요`.（家でゆっくり休まなければなりません。）
- **경험 (経験)**: ~(으)ㄴ 적이 있다 / 없다 (수업 25)
  - 한라산에 가 `본 적이 있어요`.（ハルラ山に行ったことがあります。）

---

## 3. 유용한 연결어미와 상황 표현（便利な接続語尾と状況表現）

時間の順序、理由、条件などをつなぐときに使う文法。

- **나열 / 병렬 (羅列/並列)**: ~고 (수업 11)
  - 밥을 먹`고` 커피를 마셨어요.（ご飯を食べてコーヒーを飲みました。）
- **이유 / 원인 (理由/原因)**: ~아서/어서 (수업 11, 19), ~기 때문에 (수업 22)
  - 피곤`해서` 일찍 잤어요.（疲れたので早く寝ました。）
  - 비가 오`기 때문에` 추워요.（雨が降っているため寒いです。）
- **역접 / 대조 (逆接/対照)**: ~지만, ~는데 (수업 14)
  - 한국어는 어렵`지만` 재미있어요.（韓国語は難しいですが面白いです。）
- **가정 / 조건 (仮定/条件)**: ~(으)면 (수업 26)
  - 돈이 많`으면` 차를 살 거예요.（お金がたくさんあれば車を買います。）
- **간접화법 (間接話法)**: ~다고 하다 / ~대요 (수업 28)
  - 친구가 부산 바다가 예쁘`대요`.（友達が釜山の海がきれいだと言っていました。）
