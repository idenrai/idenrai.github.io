# 수업 2. 책이 어디에 있어요?

존재・所有を表す`있어요/없어요`と、場所の表現を学ぶ。

## 1. 있어요 / 없어요（います・あります / いません・ありません）

原型は`있다 / 없다`。会話では`있어요 / 없어요`を使う。
人にも物にも両方使える点が、日本語の「いる/ある」の使い分けと異なる。

### 존재（存在）

人や物がどこかに「ある・いる」ことを表す。

- 고양이`가` 있어요.（猫`が`います。）
- 강아지`가` 없어요.（犬`が`いません。）
- 책`이` 있어요.（本`が`あります。）
- 열쇠`가` 없어요.（鍵`が`ありません。）

### 소유（所有）

「〜を持っている・〜がある」という所有を表す。

- 저는 시간`이` 있어요.（私は時間`が`あります。）
- 저는 돈`이` 없어요.（私はお金`が`ありません。）
- 오빠`가` 있어요.（兄`が`います。）
- 형제`가` 없어요.（兄弟`が`いません。）

---

## 2. 어디에 있어요?（どこにありますか？）

### 어디 + 에

`어디`は疑問詞で、日本語の「どこ」と一致する。
場所を表す助詞`에`と組み合わせて使う。

- `어디에` 있어요?（`どこに`ありますか？ / いますか？）

### 答え方

`[場所] + 에 있어요`で答える。

- 화장실`이` 어디에 있어요?（トイレはどこですか？）
  - 저기`에` 있어요.（あそこ`に`あります。）
  - 2층`에` 있어요.（2階`に`あります。）

> `어디에 있어요?`は日常でとても頻繁に使う。まずこの一文を丸ごと覚えておこう。

---

## 3. 위치 명사（位置を表す名詞）

場所の名詞の後ろに`에`をつけて「〜に」を表す。

| 韓国語 | 日本語 | 例文 |
| ---- | ---- | ---- |
| 위 | 上 | 책상 위에 있어요.（机の上にあります。） |
| 아래 / 밑 | 下 | 침대 아래에 있어요.（ベッドの下にあります。） |
| 앞 | 前 | 학교 앞에 있어요.（学校の前にあります。） |
| 뒤 | 後ろ | 건물 뒤에 있어요.（建物の後ろにあります。） |
| 옆 | 横 | 가게 옆에 있어요.（お店の横にあります。） |
| 안 | 中 | 가방 안에 있어요.（カバンの中にあります。） |
| 밖 | 外 | 집 밖에 있어요.（家の外にあります。） |

## 5. 긍정 / 부정（肯定・否定の応答）

수업 1で学んだ`네 / 아니요`と組み合わせる。

- 고양이가 있어요?（猫はいますか？）
  1. `네`, 있어요.（`はい`、います。）
  2. `아니요`, 없어요.（`いいえ`、いません。）

- 시간이 있어요?（時間はありますか？）
  1. `네`, 있어요.（`はい`、あります。）
  2. `아니요`, `지금은` 없어요.（`いいえ`、`今は`ありません。）

> `지금은`（今は）のように`은/는`を使うと「今は（他は知らないが）」というニュアンスになる。

## 練習

以下の単語を使って、文章を作成してみましょう。

```text
강아지（犬）, 고양이（猫）, 물고기（魚）
지갑（財布）, 핸드폰（スマホ）, 우산（傘）
책상 위（机の上）, 서랍 안（引き出しの中）, 가방 안（カバンの中）
```

## Tips

### 이/가 と 은/는 の違い

| | 使う場面 | 例 |
| -- | ---- | ---- |
| 이/가 | 新しい情報・強調 | 고양이`가` 있어요. |
| 은/는 | 話題の提示・対比 | 고양이`는` 없어요, 강아지`는` 있어요. |

初級段階では `있어요/없어요`の前は`이/가`を使うと覚えておけばよい。

### 助詞まとめ（수업 1・2）

| 意味 | 母音終わり | 子音終わり |
| ---- | ---- | ---- |
| は | 는 | 은 |
| が | 가 | 이 |
| に（場所） | 에 | 에 |
