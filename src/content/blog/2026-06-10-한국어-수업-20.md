# 수업 20. 가족·관계 표현

오늘은 가족과 관계를 나타내는 표현을 배운다.
특히 한국은 가족 호칭이 발달해 있고, 시댁(夫の実家) 식구를 부르는 명칭이 중요하므로 잘 익혀두자!

---

## 1. 기본 가족 호칭 (基本の家族呼称)

한국어는 부르는 사람이 '남자'인지 '여자'인지에 따라 형제자매를 부르는 호칭이 다르다.
아내(여성)의 입장에서 쓰는 호칭을 중심으로 알아보자.

| 한국어 (여성 입장) | 일본어 |
|-------------------|--------|
| 오빠              | 兄 |
| 언니              | 姉 |
| 남동생            | 弟 |
| 여동생            | 妹 |
| 아빠 / 아버지     | お父さん / 父 |
| 엄마 / 어머니     | お母さん / 母 |
| 할아버지          | おじいさん |
| 할머니            | おばあさん |

※ 남성 입장에서는 형(兄), 누나(姉)를 사용한다.

---

## 2. 남편의 가족 부르기 (夫の家族の呼び方)

시댁(夫の実家) 식구들을 부를 때는 특별한 호칭과 높임말을 쓴다.

| 대상 | 한국어 호칭 | 일본어 |
|------|-------------|--------|
| 남편의 아버지 | 아버님 | お義父さん |
| 남편의 어머니 | 어머님 / 시어머니 | お義母さん |
| 남편의 형 | 아주버님 | 義理の兄 |
| 남편의 누나 | 형님 | 義理の姉 |
| 남편의 남동생 | 도련님 (미혼) / 서방님 (기혼) | 義理の弟 |
| 남편의 여동생 | 아가씨 | 義理の妹 |

```text
어머님, 안녕하세요. 건강하시죠?
  → お義母さん、こんにちは。お元気ですか？

아버님, 식사하셨어요?
  → お義父さん、お食事されましたか？
```

---

## 3. 가족 소개하기 (家族の紹介)

가족을 소개할 때는 `제 (저의)` 또는 `우리 (우리의)`를 사용한다. 한국에서는 '우리(私たち)'라는 표현을 자주 쓴다. (우리 남편, 우리 가족 등)

```text
우리 남편이에요.      → 私の夫です。
제 여동생이에요.      → 私の妹です。
저희 부모님이세요.    → 私の両親です。（부모님은 높임말이므로 이/가 아니에요 대신 이세요를 씀）
```

### 가족 인원수 말하기

사람을 셀 때는 `명(名)`을 쓴다. 숫자에는 고유어 수사(하나, 둘, 셋, 넷...)를 사용한다.

- 1명: 한 명
- 2명: 두 명
- 3명: 세 명
- 4명: 네 명
- 5명: 다섯 명

```text
가족이 몇 명이에요?
  → 家族は何人ですか？

우리 가족은 네 명이에요. 아버지, 어머니, 저, 그리고 여동생이 있어요.
  → 私の家族は4人です。父、母、私、そして妹がいます。
```

---

## 4. 높임말 (敬語)

가족, 특히 부모님이나 시부모님에 대해 말할 때는 동사에 `~시/으시`를 넣어 높임말을 만든다.

```text
어머님이 요리를 하세요. (하다 → 하시다 → 하세요)
  → お義母さんが料理をなさいます。

아버님이 신문을 읽으세요. (읽다 → 읽으시다 → 읽으세요)
  → お義父さんが新聞をお読みになります。
```

---

## 5. 회화 연습

**상황: 시댁 방문 (義実家への訪問)**

```text
시어머니: 아이고, 우리 며느리 왔어? 피곤하지?
         (あら、うちの嫁が来たの？疲れたでしょう？)

아내: 아니에요, 어머님. 잘 지내셨어요?
     (いいえ、お義母さん。お元気でしたか？)

시어머니: 그럼. 아버님은 거실에 계셔. 가서 인사해.
         (もちろん。お義父さんはリビングにいらっしゃるわ。挨拶してきなさい。)

아내: 네. 아버님, 안녕하세요! 
     (はい。お義父さん、こんにちは！)

시아버지: 오, 그래. 먼 길 오느라 고생 많았다.
         (おお、遠いところよく来たね。お疲れ様。)
```

## 練習

**빈칸에 알맞은 호칭을 넣어 보세요. (空欄に合う呼称を入れてみましょう。)**

1. 남편의 아버지를 부를 때는 _________ (이)라고 합니다.
2. 남편의 어머니를 부를 때는 _________ (이)라고 합니다.
3. 남편의 형은 _________ (이)라고 부릅니다.
4. 남편의 누나는 _________ (이)라고 부릅니다.

**정답 (答え)**
1. 아버님
2. 어머님
3. 아주버님
4. 형님

---

## Tips

한국에서는 남편을 부를 때 "오빠", "자기야", "여보" 등 다양한 호칭이 쓰인다.
시부모님 앞에서는 "남편이~", "그이가~", 또는 자녀가 있다면 "~아빠가" 라고 지칭하는 것이 일반적이다.
