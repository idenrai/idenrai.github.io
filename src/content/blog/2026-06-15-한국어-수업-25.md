# 수업 25. 한국에 가 본 적이 있어요?

오늘은 살면서 겪어본 과거의 경험을 물어보고 표현할 때 쓰는 **~(으)ㄴ 적이 있다/없다** 와,
시도나 경험을 나타내는 **~아/어 보다**의 복습 및 심화 표현을 배운다.

---

## 1. 경험 관련 어휘 (経験の語彙)

| 한국어 | 일본어 | 예문 |
|--------|--------|------|
| 경험 | 経験 | 여러 가지 경험을 했어요. |
| 타다 | (乗り物に)乗る | 비행기를 탄 적이 있어요. |
| 가 보다 | 가 보는 것 / 行ってみる | 한국 제주도에 가 봤어요. |
| 먹어 보다 | 먹어 보는 것 / 食べてみる | 비빔밥을 먹어 본 적이 있어요. |
| 만나다 | 만나는 것 / 会う | 한국 연예인을 만난 적이 있어요. |

---

## 2. ~(으)ㄴ 적이 있다/없다 (〜한 적이 있다/없다 - 経験)

과거에 어떤 일을 해본 경험이 "있다" 또는 "없다"를 나타낼 때 사용한다.

- 받침이 없을 때: **~ㄴ 적이 있다/없다** (예: 가다 → 간 적이 있다)
- 받침이 있을 때: **~은 적이 있다/없다** (예: 먹다 → 먹은 적이 있다)
- 'ㄹ' 받침으로 끝날 때: **~ㄴ 적이 있다/없다** (예: 만들다 → 만든 적이 있다 - 'ㄹ' 탈락)

```text
한국에 간 적이 있어요?
  → 韓国に行ったことがありますか?

저는 매운 음식을 먹은 적이 없어요.
  → 私は辛い食べ物を食べたことがありません。

말을 탄 적이 있어요?
  → 馬に乗ったことがありますか？
```

---

## 3. ~아/어 보다 (〜해 보다 - 試み・経験)

수업 18에서 배웠던 `~아/어 보다`의 과거형인 **~아/어 봤어요**는 이미 시도해서 완료된 **경험**을 구체적으로 강조할 때 자주 쓰인다.
앞에서 배운 `~(으)ㄴ 적이 있다/없다`와 결합하여 **~아/어 본 적이 있다/없다** 형태로 매우 자주 쓰인다.

```text
제주도에 가 봤어요.
  → 済州島に行ってみました。（行ったことがあります。）

제주도에 가 본 적이 있어요. (더 명확하게 '경험'임을 강조)
  → 済州島に行ってみたことがあります。

김치를 먹어 봤어요?
  → キムチを食べてみましたか？
```

---

## 4. 회화 연습

**상황: 과거 경험 묻기 (過去の経験についての会話)**

```text
남편: 유나 씨는 한국의 제주도에 가 본 적이 있어요?
     (ユ나 씨는 한국의 제주도에 가 본 적이 있어요?)

아내: 아니요, 제주도에는 가 본 적이 없어요. 서울에만 가 봤어요.
     (いいえ、済州島には行ったことがありません。ソウルにだけ行ってみました。)

남편: 그래요? 제주도는 정말 아름다운 섬이에요. 다음에 꼭 같이 가 봐요.
     (そうですか？ 済州島は本当に美しい島です。今度ぜひ一緒に行ってみましょう。)

아내: 네, 좋아요. 참, 남편은 한라산(제주도의 산)에 올라가 봤어요?
     (はい, いいですね。そういえば、夫はハルラ山(済州島の山)に登ってみましたか？)

남편: 네, 등산해 본 적이 있어요. 조금 힘들었지만 경치가 아주 좋았어요.
     (はい、登山したことがあります。少し大変でしたが景色がとても良かったです。)

아내: 부럽네요. 저도 한라산에 가 보고 싶어요.
     ( 羨ましいですね。저도 한라산에 가 보고 싶어요.)
```

## 練習

**문장의 빈칸에 들어갈 알맞은 단어를 만들어 보세요. (文章의 空欄に入る正しい단어를 만들어 보세요.)**

1. 저는 한국 비행기를 ________ 적이 없어요. (타다 - 경험)
2. 비빔밥은 진짜 맛있으니까 한 번 ________ 보세요. (먹다 - 시도)
3. 예전에 한국 친구를 ________ 적이 있어요. (만나다 - 경험)
4. 김치를 맛________ 적이 있습니까? (보다 - 경험)

**정답 (答え)**
1. 탄
2. 먹어
3. 만난
4. 본 (맛본 - '맛보다'의 활용형)

---

## Tips

### '가 봤어요'와 '간 적이 있어요'의 미묘한 차이
두 표현 모두 "갔다 온 경험이 있다"를 뜻하지만 구어에서는 **가 봤어요**가 훨씬 가볍고 일상적으로 자주 쓰인다.
**간 적이 있어요**는 조금 더 객관적인 경험의 사실 유무를 나타낸다.
