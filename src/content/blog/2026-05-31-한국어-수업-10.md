# 수업 10. 한국어 할 수 있어요?

可能・不可能を表す表現を学ぶ。

## 1. ~(으)ㄹ 수 있어요 / 없어요（〜できます / できません）

動詞の語幹に`(으)ㄹ 수 있어요`をつける。

- 語幹が母音またはㄹで終わる → `ㄹ 수 있어요`
- 語幹が子音で終わる → `을 수 있어요`

| 原型 | ~ㄹ/을 수 있어요 | 意味 |
| ---- | ---- | ---- |
| 가다 | 갈 수 있어요 | 行けます |
| 오다 | 올 수 있어요 | 来られます |
| 먹다 | 먹을 수 있어요 | 食べられます |
| 마시다 | 마실 수 있어요 | 飲めます |
| 하다 | 할 수 있어요 | できます |
| 보다 | 볼 수 있어요 | 見られます |
| 운전하다 | 운전할 수 있어요 | 運転できます |
| 수영하다 | 수영할 수 있어요 | 泳げます |

### 없어요（できません）

`있어요`を`없어요`に変えるだけ。

- `갈 수 없어요.`（行けません。）
- `먹을 수 없어요.`（食べられません。）
- `할 수 없어요.`（できません。）

---

## 2. 못 との違い

「できない」を表すには`못`と`~ㄹ/을 수 없어요`の二通りがある。

| 表現 | ニュアンス | 例 |
| ---- | ---- | ---- |
| 못 + 동사 | 短く「できない」 | 못 가요. |
| ~ㄹ/을 수 없어요 | 丁寧・理由ありの感じ | 갈 수 없어요. |

- `못 먹어요.`（食べられません。— アレルギーなど）
- `먹을 수 없어요.`（食べられません。— 丁寧）
- `오늘 못 가요.`（今日行けません。）
- `오늘은 갈 수 없어요.`（今日は行けません。— 丁寧）

> 日常会話では`못`の方がよく使われる。

---

## 3. 질문과 응답（質問と応答）

文末を上げるだけで疑問文になる。

- `한국어 할 수 있어요?`（韓国語できますか？）
  - 네, 조금 할 수 있어요.（はい、少しできます。）
  - 아직 잘 못해요.（まだうまくできません。）

- `이거 먹을 수 있어요?`（これ食べられますか？）
  - 네, 먹을 수 있어요.（はい、食べられます。）
  - 아니요, 매워서 못 먹어요.（いいえ、辛くて食べられません。）

---

## 4. 과거형（過去形）

`있어요 → 있었어요`に変える。

- 어렸을 때 수영`할 수 있었어요.`（小さいころ泳げました。）
- 예전에는 매운 거 `먹을 수 있었어요.`（以前は辛いもの食べられました。）

---

## 5. 程度を表す副詞と組み合わせ

| 副詞 | 意味 | 例 |
| ---- | ---- | ---- |
| 잘 | うまく | 잘 할 수 있어요.（うまくできます。） |
| 조금 | 少し | 조금 할 수 있어요.（少しできます。） |
| 별로 | あまり | 별로 못해요.（あまりできません。） |
| 전혀 | 全然 | 전혀 못해요.（全然できません。） |

---

## 練習

以下の文を韓国語にしてみましょう。

```text
1. 日本語が話せますか？
2. はい、少し話せます。
3. 辛いものは食べられません。
4. 明日来られますか？
5. 以前はもっとうまくできました。
```

**정답**

1. 일본어 할 수 있어요? / 일본어 말할 수 있어요?
2. 네, 조금 할 수 있어요.
3. 매운 건 못 먹어요. / 매운 걸 먹을 수 없어요.
4. 내일 올 수 있어요?
5. 예전에는 더 잘할 수 있었어요.
