간단하게 디렉토리 구성도 작성하기

簡単に「ディレクトリ構成図」を作成する

맥에서 사용할 경우엔 인스톨 필요

Macで使いたい場合はインストール必要

```
brew install tree
```

\- cmd에서 폴더의 트리 구조 출력

\- コマンドプロンプトでフォルダのツリー構造出力

**tree 경로**

**tree 経路**

폴더 내의 파일까지 보고 싶을 때엔 **/f** 추가

フォルダ内のファイルまで見たい場合は、「**/f**」を追加

그걸 파일로 출력하고 싶다면 **\> ~~.txt**

それをファイルで出力したい場合は、「 **\> ~~.txt**」

ex)

**tree /f C:\\Users\\kwon\\git\\Project > DirectoryTree.txt**

이후 해당 txt파일을 엑셀에 붙여넣기.

後、そのtxtファイル（**DirectoryTree.txt**）をエクセルに貼り付け

엑셀에서의 칸 구분을 위한 설정은 다음과 같다.

エクセルでの枠区分のための設定

1\. 칸 나눌 라인 선택

　枠を分けるラインを選択

2\. データ의 区切り位置 클릭

　データの区切り位置をクリック

3\. カンマやタブなどの区切り文字によってフィールドごとにっ区切られたデータ(D) 선택

　カンマやタブなどの区切り文字によってフィールドごとにっ区切られたデータ(D) 選択

4\. 区切り文字를 タブ와 スペース로 선택 후 완료

　区切り文字をタブとスペースで選択して完了
