# 수업 15. 이거 주세요!

今回は**식당에서 음식 주문하기**（レストランでの注文）を学ぶ。  
韓国に行ったとき、また韓国料理屋さんでそのまま使える実用フレーズ集。

---

## 1. 주문할 때 기본 표현 (注文の基本)

```text
이거 주세요.          → これをください。
저거 주세요.          → あれをください。
이걸로 할게요.        → これにします。
추천이 뭐예요?        → おすすめは何ですか？
메뉴판 주세요.        → メニューをください。
```

### 수량 표현 (数量の言い方)

韓国語の数量表現は **고유어 수** (一、二、三…) を使う。

| 수 | 고유어 | 예 |
|----|--------|----|
| 1 | 하나 | 하나 주세요 |
| 2 | 둘 | 둘 주세요 |
| 3 | 셋 | 셋 주세요 |
| 4 | 넷 | 넷 주세요 |
| 5 | 다섯 | 다섯 주세요 |

※ 명사 앞에서는 **한, 두, 세, 네** 로 변한다.

```text
삼겹살 2인분 주세요.     → サムギョプサル2人前ください。
소주 한 병 주세요.       → 焼酎1本ください。
물 두 잔 주세요.         → 水2杯ください。
```

---

## 2. 식당에서 자주 쓰는 표현 (よく使う表現)

```text
얼마나 기다려요?         → どのくらい待ちますか？
조금 매워요?             → 少し辛いですか？
안 맵게 해 주세요.       → 辛くしないでください。
이거 빼 주세요.          → これを抜いてください。
포장해 주세요.           → 持ち帰りにしてください。（包んでください。）
영수증 주세요.           → 領収書ください。
계산해 주세요.           → お会計お願いします。
```

### 맛 표현 (味の表現)

| 한국어 | 일본어 |
|--------|--------|
| 맵다 | 辛い |
| 달다 | 甘い |
| 짜다 | 塩辛い |
| 싱겁다 | 薄い（味が薄い） |
| 시다 | 酸っぱい |
| 쓰다 | 苦い |
| 고소하다 | こうばしい・香ばしい |
| 담백하다 | さっぱりしている |

```text
이게 좀 매운데 괜찮아요?    → これちょっと辛いですが大丈夫ですか？
너무 짜지 않아요?            → 塩辛すぎませんか？
고소하고 맛있어요!           → 香ばしくて美味しいです！
```

## 3. 회화 연습

**상황：부부가 한국 식당에서**（夫婦が韓国料理屋で）

```text
직원: 어서 오세요! 몇 분이세요?
     （いらっしゃいませ！何名様ですか？）

남편: 두 명이요.
     （2名です。）

직원: 이쪽으로 앉으세요. 주문 받을까요?
     （こちらにお座りください。ご注文をお受けしましょうか？）

아내: 삼겹살 2인분이랑 된장찌개 하나 주세요.
     （サムギョプサル2人前とテンジャンチゲ一つください。）

직원: 음료는요?
     （お飲み物は？）

남편: 소주 한 병이랑 물 두 잔 주세요.
     （焼酎1本と水2杯ください。）

아내: 혹시 많이 매워요?
     （もしかして、すごく辛いですか？）

직원: 아니요, 별로 안 매워요.
     （いいえ、あまり辛くないです。）

아내: 그럼 됐어요. 감사합니다!
     （じゃあ大丈夫です。ありがとうございます！）
```

## 練習

**다음 상황에 맞게 한국어로 말해 보세요。（状況に合わせて韓国語で言ってみましょう。）**

1. 냉면을 2개 주문하고 싶다.
2. 음식이 너무 짜서 불평하고 싶다.
3. 계산하고 싶다.
4. 포장해 달라고 하고 싶다.

**정답 (答え)**

```text
1. 냉면 두 개 주세요.
2. 좀 짠데요… 또는 너무 짜요.
3. 계산해 주세요.
4. 포장해 주세요.
```

## Tips

### ~(이)랑 （〜と）

注文などで複数のものを並べるときは `(이)랑` が便利。  
（丁寧な書き言葉では `과/와` だが、会話では `(이)랑` が自然）

```text
삼겹살이랑 냉면이랑 주세요.   → サムギョプサルと冷麺をください。
소주랑 맥주 주세요.            → 焼酎とビールをください。
```
