# 수업 18. 이거 얼마예요?

買い物の表現（쇼핑 표현）を学ぶ。市場や服屋など、さまざまな場面で使えるフレーズ。

## 1. 가격 묻기와 흥정（値段の聞き方と値切り）

韓国の伝統市場や地下商店街などでは、値段を聞いたり値切ったりすることがある。

- `이거 얼마예요?`（これいくらですか？）
- `모두 얼마예요?`（全部でいくらですか？）

### 흥정 표현（値切り表現）

- `너무 비싸요.`（高すぎます。）
- `좀 깎아 주세요.`（少し安くしてください。）

> デパートや大型マート、チェーン店では値引きはできないので注意。
> 市場や東大門の服屋などで「좀 깎아 주세요」と言うと、おまけしてくれたり少し安くしてくれたりする。

---

## 2. 옷가게에서（服屋で）

服を買うときに必須の表現。

### 입어 보다（着てみる）

- `이거 입어 봐도 돼요?`（これ着てみてもいいですか？）
  - 수업 12で学んだ `~아/어도 돼요`（〜してもいいですか）の応用。
- `탈의실이 어디예요?`（試着室はどこですか？）

### 사이즈·색상（サイズ・色）

- `다른 색깔 있어요?`（別の色はありますか？）
- `더 큰 사이즈 있어요?`（もっと大きいサイズはありますか？）
- `더 작은 사이즈 있어요?`（もっと小さいサイズはありますか？）

---

## 3. 결제할 때（お会計のとき）

- `계산해 주세요.`（お会計お願いします。）
- `카드로 결제할게요.`（カードで支払います。）
- `현금으로 할게요.`（現金で払います。）
- `영수증 주세요.`（レシートください。）
- `봉투 주세요.`（袋をください。）

> 韓国では「봉투(袋)」は有料（環境保護のため）であることが多い。
> `봉투 필요하세요?`（袋は必要ですか？）と聞かれたら `네, 주세요` または `아니요, 괜찮아요` と答える。

---

## 4. ~아/어 보다（〜してみる）

「〜してみる」という経験や試みを表す。買い物の場面でよく使う。

- 입다（着る） → `입어 보세요.`（着てみてください。）
- 신다（履く） → `신어 보세요.`（履いてみてください。）
- 먹다（食べる） → `먹어 보세요.`（食べてみてください。）
- 하다（する） → `해 보세요.`（やってみてください。）

---

## 5. 회화 연습

**상황：동대문 옷가게에서**（東大門の服屋で）

- 유나: 아저씨, 이 바지 얼마예요?（おじさん、このズボンいくらですか？）
- 점원: 삼만 오천 원이에요.（3万5千ウォンです。）
- 유나: 다른 색깔은 없어요?（別の色はないですか？）
- 점원: 검은색이랑 하얀색이 있어요. 한번 입어 보세요.（黒色と白色があります。一度着てみてください。）
- 유나: (입어 본 후) 마음에 들어요. 근데 너무 비싸요. 조금 깎아 주세요!
       (着てみた後) 気に入りました。でも高すぎます。少し安くしてください！
- 점원: 현금으로 하면 삼만 원에 줄게요.（現金で払うなら3万ウォンにしますよ。）
- 유나: 네, 그럼 현금으로 할게요. 봉투에 넣어 주세요.（はい、じゃあ現金で払います。袋に入れてください。）

## 練習

以下の状況でどう言うか韓国語で考えてみましょう。

```text
1. 値段を聞きたいとき。
2. 他の色があるか聞きたいとき。
3. 靴（구두）を履いてみてもいいか聞きたいとき。
4. 少し安くしてほしいとき。
```

**정답**

1. 이거 얼마예요?
2. 다른 색깔 있어요?
3. 구두 신어 봐도 돼요?
4. 좀 깎아 주세요.

## Tips

### 착용 동사（身につける動詞）

日本語は「着る・履く・被る・はめる」など分かれているが、韓国語も同様に分かれている。

| 韓国語 | 日本語 | 対象の例 |
| ---- | ---- | ---- |
| 입다 | 着る | 옷(服)、바지(ズボン)、치마(スカート) |
| 신다 | 履く | 신발(靴)、양말(靴下) |
| 쓰다 | 被る・かける | 모자(帽子)、안경(メガネ) |
| 끼다 | はめる | 장갑(手袋)、반지(指輪) |
| 차다 | 着ける | 시계(時計) |
| 매다 | 結ぶ・締める | 넥타이(ネクタイ) |
