# 수업 26. 비가 오면 집에서 쉴 거예요

오늘은 가정이나 조건을 나타내는 **~(으)면** 표현과,
어떤 일이 일어나는 시간을 나타내는 **~(으)ㄹ 때** 표현을 배운다.

---

## 1. 조건/날씨 관련 어휘 (조건・天気の語彙)

| 한국어 | 일본어 | 예문 |
|--------|--------|------|
| 만약 | もしも | 만약 내일 비가 오면 약속을 취소해요. |
| 날씨 | 天気 | 오늘 날씨가 참 맑네요. |
| 비 / 눈 | 雨 / 雪 | 하늘에서 눈이 내려요. |
| 바람 | 風 | 바람이 강하게 불어요. |
| 기분 | 기분 / 気持ち | 기분이 좋을 때 노래를 불러요. |

---

## 2. ~(으)면 (〜하면 / 〜하면 - 仮定・条件)

어떤 상황을 가정하거나 앞 문장의 행동이 뒷 문장의 조건이 될 때 사용한다. (일본어의 `~ば`, `~たら`에 대응한다.)

- 받침이 없을 때: **~면** (예: 가다 → 가면)
- 받침이 있을 때: **~으면** (예: 먹다 → 먹으면)
- 'ㄹ' 받침으로 끝날 때: **~면** (예: 살다 → 살면 - 'ㄹ' 유지)

```text
돈이 있으면 가방을 살 거예요.
  → お金があればバッグを買います。

질문이 있으면 언제든지 말해 주세요.
  → 質問があればいつでも言ってください。

날씨가 좋으면 산책합시다.
  → 天気が良ければ散책합시다.
```

---

## 3. ~(으)ㄹ 때 (〜の時 - 時間)

어떤 행동이나 상태가 지속되는 시점이나 시간적 상황을 나타낼 때 사용한다. (명사 뒤에는 바로 **때**를 붙인다.)

- 받침이 없을 때: **~ㄹ 때** (예: 가다 → 갈 때)
- 받침이 있을 때: **~을 때** (예: 먹다 → 먹을 때)
- 'ㄹ' 받침으로 끝날 때: **~ㄹ 때** (예: 만들다 → 만들 때)
- 명사 뒤: **명사 + 때** (예: 방학 → 방학 때, 아침 → 아침 때 - 단, 아침에는 보통 '아침에'가 자연스럽다. '방학 때', '고등학교 때'가 자연스러움)

```text
비가 올 때 우산을 써요.
  → 雨が降る時に傘をさします。

밥을 먹을 때 조용히 하세요.
  → ご飯を食べる時、静かにしてください。

저는 슬플 때 매운 음식을 먹어요.
  → 私は悲しい時、辛い食べ物を食べます。

방학 때 한국 여행을 갈 거예요.
  → 休み（長期休暇）の時に韓国旅行に行くつもりです。
```

---

## 4. 회화 연습

**상황: 주말 날씨에 따른 계획 (週末の天気に合わせた予定)**

```text
남편: 유나 씨, 주말에 뭐 할 거예요? 날씨가 좋을까요?
     (ユナさん、週末に何をしますか？ 天気は良いでしょうか？)

아내: 일기예보를 보니까 토요일에 비가 온대요.
     (天気予報を見たら、土曜日に雨が降るそうです。)

남편: 그래요? 비가 오면 야외에서 등산을 못 하겠네요.
     ( 그렇습니까? 비가 오면 야외에서 등산을 못 하겠네요.)

아내: 네, 비가 오면 그냥 집에서 영화를 보면서 쉴래요.
     (はい, 비가 오면 그냥 집에서 영화를 보면서 쉴래요.)

남편: 그럼 일요일에는 어때요? 일요일엔 갠다고 하던데.
     (じゃあ、日曜日はどうですか？ 日曜日は晴れると言っていましたが。)

아내: 일요일에 날씨가 맑으면 공원에 피크닉 가요. 기분이 좋을 때 밖에서 노는 게 최고예요!
     (일요일에 날씨가 맑으면 공원에 피크닉 가요. 기분이 좋을 때 밖에서 노는 게 최고예요!)

남편: 좋아요! 맛있는 도시락을 만들어서 갑시다.
     (いいね！美味しいお弁当を作って行きましょう。)
```

## 練習

**문장의 빈칸에 들어갈 알맞은 단어를 만들어 보세요. (文章의 空欄に入る正しい단어를 만들어 보세요.)**

1. 모르는 단어가 ________ 사전을 찾으세요. (있다 - 조건)
2. 한국에 ________ 비행기표를 사야 해요. (가다 - 시간)
3. 너무 피곤________ 물을 많이 마시거나 주무세요. (하다 - 조건)
4. 어릴 ________ 무엇을 좋아했어요? (어리다 - 과거 시간, '어릴 때')

**정답 (答え)**
1. 있으면
2. 갈 때
3. 하면 (피곤하면)
4. 때 (어릴 때 - '어리다'의 형용사 어간 '어리' + ㄹ 때)

## Tips

### '만약'과 함께 쓰기
가정의 의미를 더 명확하게 나타내고 싶을 때 문장의 맨 앞에 **만약** 또는 **만일**을 붙여서 쓴다.
- **만약 내일 눈이 오면 좋겠어요.** (もしも明日雪이 오면 좋겠어요.)
