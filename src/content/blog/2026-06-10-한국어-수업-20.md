# 수업 20. 가족·관계 표현

家族や関係を表す言葉を学ぶ。韓国は家族の呼称が発達しており、特に義理の家族（시댁 식구）の呼び方が重要なので覚えておこう。

## 1. 기본 가족 호칭（基本の家族呼称）

韓国語は、呼ぶ人が「男性」か「女性」かによって兄弟姉妹の呼び方が異なる。
ここでは**妻（女性）の立場**で使う呼称を中心に学ぶ。

| 韓国語 (女性から見て) | 日本語 |
| ---- | ---- |
| 오빠 | 兄 |
| 언니 | 姉 |
| 남동생 | 弟 |
| 여동생 | 妹 |
| 아빠 / 아버지 | お父さん / 父 |
| 엄마 / 어머니 | お母さん / 母 |
| 할아버지 | おじいさん |
| 할머니 | おばあさん |

> ※ 男性からは兄を`형(ヒョン)`、姉を`누나(ヌナ)`と呼ぶ。

---

## 2. 남편의 가족 부르기（夫の家族の呼び方）

夫の実家（시댁）の家族を呼ぶときは、特別な呼称と敬語を使う。

| 対象 | 韓国語 | 日本語 |
| ---- | ---- | ---- |
| 夫の父 | 아버님 | お義父さん |
| 夫の母 | 어머님 / 시어머니 | お義母さん |
| 夫の兄 | 아주버님 | 義理の兄 |
| 夫の姉 | 형님 | 義理の姉 |
| 夫の弟 | 도련님 (未婚) / 서방님 (既婚) | 義理の弟 |
| 夫の妹 | 아가씨 | 義理の妹 |

- `어머님`, 안녕하세요. 건강하시죠?（お義母さん、こんにちは。お元気ですか？）
- `아버님`, 식사하셨어요?（お義父さん、お食事されましたか？）

---

## 3. 가족 소개하기（家族の紹介）

家族を紹介するときは `제 (私の)` ではなく **`우리 (私たちの)`** をよく使う。
（우리 남편, 우리 가족 など）

- `우리 남편`이에요.（私の夫です。）
- `제 여동생`이에요.（私の妹です。）
- `저희 부모님`이세요.（私の両親です。）

> `부모님`は敬語なので、`이에요`ではなく`이세요`を使う。

### 人数の数え方

人を数えるときは `명(名)` を使う。数字は**固有語数字**（하나, 둘, 셋...）を使う。

- 1명: 한 명
- 2명: 두 명
- 3명: 세 명
- 4명: 네 명
- 5명: 다섯 명

- 가족이 몇 명이에요?（家族は何人ですか？）
  - 우리 가족은 `네 명`이에요.（私の家族は4人です。）

---

## 4. 높임말（敬語の作り方）

両親や義両親について話すときは、動詞に **`~시/으시`** を入れて敬語を作る。
해요체（丁寧語）にすると `~세요/으세요` になる。

- 어머님이 요리를 `하세요`.（お義母さんが料理をなさいます。）
  - 하다 → 하시다 → 하세요
- 아버님이 신문을 `읽으세요`.（お義父さんが新聞をお読みになります。）
  - 읽다 → 읽으시다 → 읽으세요

---

## 5. 회화 연습

**상황：시댁 방문**（義実家への訪問）

- 시어머니: 아이고, 우리 며느리 왔어? 피곤하지?（あら、うちの嫁が来たの？疲れたでしょう？）
- 아내: 아니에요, 어머님. 잘 지내셨어요?（いいえ、お義母さん。お元気でしたか？）
- 시어머니: 그럼. 아버님은 거실에 계셔. 가서 인사해.（もちろん。お義父さんはリビングにいらっしゃるわ。挨拶してきなさい。）
- 아내: 네. 아버님, 안녕하세요!（はい。お義父さん、こんにちは！）
- 시아버지: 오, 그래. 먼 길 오느라 고생 많았다.（おお、遠いところよく来たね。お疲れ様。）

## Tips

### 남편을 부르는 호칭（夫の呼び方）

韓国では夫を呼ぶとき「오빠」「자기야」「여보」など多様な呼称が使われる。
しかし義両親の前では「남편이~」「그이가~」、または子どもがいれば「~아빠가」と呼ぶのが一般的。
