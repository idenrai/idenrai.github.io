# 수업 29. 고마워! 친구 사이의 반말 표현

오늘은 친구나 연인, 아랫사람처럼 아주 친하고 허물없는 사이에서 사용하는 **반말(기존 존댓말의 대응 형태)** 표현을 배운다.
한국어에서 반말을 쓰는 규칙과 자주 쓰이는 기본 어휘들을 정리해 보자!

---

## 1. 인칭 대명사 및 호칭의 변화 (人称代名詞と呼称の変化)

존댓말에서는 자신을 낮추고 상대방을 높여 말하지만, 반말에서는 격식을 낮춘 직접적인 대상을 사용한다.

| 한국어 (존댓말) | 한국어 (반말) | 일본어 |
|-----------------|--------------|--------|
| 저 (제) | **나 (내)** | 私 (僕 / 俺) |
| 당신 / 씨 (이름) | **너 (네 / 네가)** | 넌 / 君 / あなた |
| 저희 | **우리** | 私たち |
| 예 / 네 | **응** | うん |
| 아니요 | **아니** | ううん / いや |

※ `네가`는 말할 때 '니가'로 많이 발음한다. (나(내)와 너(네)의 발음이 비슷하여 혼동을 방지하기 위함.)

---

## 2. 해요체에서 반말 만들기 (해요체 → 반말)

해요체(~요)에서 단순히 **'요'를 빼면** 가장 기본적인 반말이 된다!

| 해요체 | 반말 | 의미 |
|--------|------|------|
| 가요 | **가** | 行く / 行って |
| 먹어요 | **먹어** | 食べる / 食べて |
| 공부해요 | **공부해** | 勉強する / 勉強して |
| 예뻐요 | **예뻐** | 綺麗だ / 可愛い |
| 맛있어요 | **맛있어** | 美味しい |
| 아니에요 | **아니야** | 違う / 違って |
| 예요 / 이에요 | **야 / 이야** | 〜だよ |

- 받침이 없을 때: **~야** (예: 친구야, 유나야)
- 받침이 있을 때: **~이야** (예: 남편이야, 선생님이야)

```text
A: 이거 뭐야? (이것은 무엇이에요?)
B: 이건 내 선물이야. (이것은 저의 선물이에요.)

A: 지금 어디 가? (지금 어디 가요?)
B: 나 지금 마트에 가. (저 지금 마트에 가요.)
```

---

## 3. 다양한 반말 종결어미 (様々なタメ口の語尾)

존댓말의 제안, 명령, 질문 형태에 대응하는 반말 규칙들이다.

### 1) 제안 / 청유 (~하자 - 〜しよう)
존댓말의 `~(으)ㅂ시다 / ~아요`에 대응하며, 동사 뒤에 **~자**를 붙인다.

- 가다 → 가자 (行こう)
- 먹다 → 먹자 (食べよう)

```text
우리 내일 만나자.
  → 私たち明日会おう。
```

### 2) 질문 (~니? / ~어? - 〜する？)
단순히 끝을 올려 말하거나, 질문의 뉘앙스를 담아 **~니?** 또는 **~어/아?**를 쓴다.

- 하다 → 하니? / 해?
- 먹다 → 먹니? / 먹어?

```text
밥 먹었어? / 밥 먹었니?
  → ご飯食べた？
```

### 3) 명령 (~아/어라 - 〜しろ / 〜しなさい)
단순히 해요체에서 '요'를 빼는 것보다 조금 더 명령이나 지시의 강도를 높일 때 쓴다.

- 보다 → 봐 / 봐라
- 자다 → 자 / 자라

```text
숙제 빨리 해라!
  → 宿題早くしなさい！
```

---

## 4. 자주 쓰는 반말 일상 표현

```text
안녕!               → 야호! / バイバイ! (만날 때와 헤어질 때 둘 다 씀)
잘 자!               → おやすみ！
고마워.              → ありがとう。
미안해.              → ごめんね。
맞아.               → その通り（だね）。
진짜?               → 本当？
축하해!              → おめでとう！
```

---

## 5. 회화 연습

**상황: 아침에 부부가 나눈 대화 (朝の夫婦の会話)**

```text
남편: 유나야, 잘 잤어?
     (ユナ、よく眠れた？)

아내: 응, 잘 잤어. 자기는? 오늘 몇 시에 출근해?
     (うん、よく眠れた。あなたは？今日何時に出勤する？)

남편: 나 오늘은 오전 10시에 출근해. 그래서 조금 더 자도 돼.
     (俺、 오늘은 오전 10시에 출근해. 그래서 조금 더 자도 돼.)

아내: 진짜? 잘됐다! 그럼 아침 같이 먹자. 내가 아침 만들게.
     (本当？よかった！じゃあ朝ごはん一緒に食べよう。私が朝ごはん作るね。)

남편: 고마워! 역시 유나가 최고야.
     (ありがとう！やっぱりユナが最高だよ。)

아내: 아침 다 먹은 후에 같이 청소하자.
     (朝ごはん全部食べた後に一緒に掃除しよう。)

남편: 응, 알았어. 내가 도와줄게.
     (うん、分かった。手伝うよ。)
```

## 練習

**존댓말 문장을 알맞은 반말 문장으로 고쳐 써 보세요. (敬語の文章を正しいタメ口의 文章に書き換えてください。)**

1. 저는 내일 학교에 가요. → 나 내일 학교에 ________.
2. 같이 점심 먹읍시다. → 같이 점심 ________.
3. 고맙습니다. 주말 잘 보내세요. → 고마워. 주말 잘 ________.
4. 이것은 제 핸드폰이에요. → 이건 내 핸드폰________.

**정답 (答え)**
1. 가
2. 먹자
3. 보내 / 보내라 (친구끼리는 '잘 보내!'가 자연스러움)
4. 이야

## Tips

### 반말을 쓸 때 주의할 점!
한국에서는 초면에 반말을 쓰면 큰 결례가 될 수 있다.
나이가 자신보다 어리더라도 상대방의 동의 없이 바로 반말을 쓰지 않고 보통 **"말 편하게 하셔도 돼요"** 또는 **"우리 이제 말 놓을까요?"**라는 제안을 거친 후에 반말을 사용한다.
부부 사이나 아주 친한 친구 사이에서는 반말을 통해 훨씬 더 큰 친근감을 느낄 수 있다!
