# 수업 12. 이거 먹어도 돼요?

許可・禁止・義務を表す3つの表現を学ぶ。日常のあらゆる場面で必要になる。

## 1. ~아도/어도 돼요（〜してもいいです）

許可を求めたり与えるときに使う。

| 原型 | ~아도/어도 돼요 | 意味 |
| ---- | ---- | ---- |
| 먹다 | 먹어도 돼요 | 食べてもいいです |
| 마시다 | 마셔도 돼요 | 飲んでもいいです |
| 가다 | 가도 돼요 | 行ってもいいです |
| 앉다 | 앉아도 돼요 | 座ってもいいです |
| 사용하다 | 사용해도 돼요 | 使ってもいいです |
| 찍다 | 찍어도 돼요 | 撮ってもいいです |

- `이거 먹어도 돼요?`（これ食べてもいいですか？）
  - 네, `먹어도 돼요.`（はい、食べてもいいです。）
  - 아니요, `안 돼요.`（いいえ、だめです。）
- `여기 앉아도 돼요?`（ここに座ってもいいですか？）
  - 네, `앉으세요.`（はい、どうぞ座ってください。）

---

## 2. ~(으)면 안 돼요（〜してはいけません）

禁止を表す表現。

- 語幹が母音またはㄹで終わる → `면 안 돼요`
- 語幹が子音で終わる → `으면 안 돼요`

- 여기서 담배 `피우면 안 돼요.`（ここでタバコを吸ってはいけません。）
- `늦으면 안 돼요.`（遅れてはいけません。）
- 사진 `찍으면 안 돼요.`（写真を撮ってはいけません。）

---

## 3. ~아야/어야 해요（〜しなければなりません）

義務・必要性を表す表現。

- 母音が`ㅏ/ㅗ` → `아야 해요`
- それ以外 → `어야 해요`
- `하다` → `해야 해요`

| 原型 | ~아야/어야 해요 | 意味 |
| ---- | ---- | ---- |
| 가다 | 가야 해요 | 行かなければなりません |
| 먹다 | 먹어야 해요 | 食べなければなりません |
| 자다 | 자야 해요 | 寝なければなりません |
| 공부하다 | 공부해야 해요 | 勉強しなければなりません |
| 일하다 | 일해야 해요 | 働かなければなりません |

- 내일 일찍 `일어나야 해요.`（明日早く起きなければなりません。）
- 약을 `먹어야 해요.`（薬を飲まなければなりません。）
- 지금 `가야 해요.`（今行かなければなりません。）

---

## 4. 3つの表現の比較

| 表現 | 意味 | 例 |
| ---- | ---- | ---- |
| ~아도/어도 돼요 | 〜してもいい（許可） | 가도 돼요. |
| ~(으)면 안 돼요 | 〜してはいけない（禁止） | 가면 안 돼요. |
| ~아야/어야 해요 | 〜しなければならない（義務） | 가야 해요. |

同じ動詞で比較すると分かりやすい。

- 약을 `먹어도 돼요.`（薬を飲んでもいいです。）
- 약을 `먹으면 안 돼요.`（薬を飲んではいけません。）
- 약을 `먹어야 해요.`（薬を飲まなければなりません。）

---

## 5. ~지 않아도 돼요（〜しなくてもいいです）

義務の否定 = 「しなくていい」は`안 해도 돼요`または`지 않아도 돼요`を使う。

- `안 가도 돼요.`（行かなくてもいいです。）
- `걱정 안 해도 돼요.`（心配しなくてもいいです。）
- 다 `먹지 않아도 돼요.`（全部食べなくてもいいです。）

## 練習

以下の場面に合う文を韓国語で作ってみましょう。

```text
1. 写真を撮ってもいいか聞く
2. ここでは電話してはいけないと伝える
3. 毎日練習しなければならないと伝える
4. 急がなくていいと伝える
5. ここに座ってもいいか聞く
```

**정답**

1. 사진 찍어도 돼요?
2. 여기서 전화하면 안 돼요.
3. 매일 연습해야 해요.
4. 서두르지 않아도 돼요. / 급하지 않아도 돼요.
5. 여기 앉아도 돼요?

## Tips

### 안 돼요 vs 없어요

- `안 돼요.`（だめです / できません — 禁止・不可）
- `없어요.`（ありません / いません — 存在しない）

例えば：
- 카드 `안 돼요.`（カードは使えません。— 非対応）
- 카드 `없어요.`（カードがありません。— 持っていない）

### 口語での縮約

`해야 해요`は口語で縮まることが多い。

- 가야 해요 → 가야 해 / `가야 돼`（口語）
- 먹어야 해요 → 먹어야 해 / `먹어야 돼`（口語）

`가야 돼요`も非常によく使われる自然な表現。
