# 수업 27. 제가 도와줄게요

話し手が自分の行動や意思を相手に約束するときに使う **~(으)ㄹ게요** と、動詞を名詞にして約束や決定を表す **~기로 해요** などを学ぶ。

## 1. 의지/약속 관련 어휘（意思・約束の語彙）

| 韓国語 | 日本語 | 例文 |
| ---- | ---- | ---- |
| 약속 (약속하다) | 約束 (約束する) | 친구와 `약속을 했어요`. |
| 전화 (전화하다) | 電話 (電話する) | 이따가 부모님께 `전화할게요`. |
| 돕다 | 手伝う / 助ける | 제가 청소를 `도와줄게요`. |
| 문자 (문자를 보내다) | メッセージ / メール | 도착하면 `문자를 보내 주세요`. |
| 연락 (연락하다) | 連絡 (連絡する) | 나중에 다시 `연락할게요`. |

---

## 2. ~(으)ㄹ게요（〜します / 〜しますね - 話し手の意思・約束）

話し手が相手に対し、自分の意思を知らせたり、ある行動をすると確実に**約束**するときに使う。（主語が**一人称（私）**のときのみ使え、疑問文では使えない。）

- パッチムがないとき: **~ㄹ게요**（例: 가다 → 갈게요）
- パッチムがあるとき: **~을게요**（例: 먹다 → 먹을게요）
- `ㄹ` パッチムのとき: **~ㄹ게요**（例: 만들다 → 만들게요）
- `ㅂ` 不規則動詞のとき: **~울게요**（例: 돕다 → 도울게요 / 도와줄게요）

- 제가 오늘 저녁밥을 `만들게요`.（私が今日の晩ご飯を作ります。※作るという約束）
- 내일 꼭 9시까지 `갈게요`.（明日必ず9時までに行きます。）
- 이따가 `연락할게요`.（後で連絡します。）

> **~ㄹ 거예요 との違い**
> `~ㄹ 거예요` (수업 13) は自分の未来の予定を単に伝達する。
> `~ㄹ게요` は**「相手に対する約束・配慮」**のニュアンスが強く含まれる。

---

## 3. ~기로 해요（〜することにしましょう - 決定・約束）

動詞の後ろに `~기` をつけて名詞の形にした後、相手と約束や決定事項を確認するときに使う。

- 動詞の語幹 + **~기로 해요**（例: 만나다 → 만나기로 해요, 먹다 → 먹기로 해요）

- 내일 서울역에서 `만나기로 해요`.（明日ソウル駅で会うことにしましょう。）
- 오늘 저녁은 외식하`기로 해요`.（今日の夕食は外食することにしましょう。）
- 내일은 늦지 않`기로 약속해요`.（明日は遅れないことを約束しましょう。）

---

## 4. 회화 연습

**상황：저녁 준비와 약속**（夕食の準備と約束）

- 유나: 오늘 회사 일이 너무 많아서 피곤하네요.（今日は仕事が多すぎて疲れますね。）
- 아빠: 그래요? 그럼 오늘 저녁 요리는 제가 할게요.（そうですか？じゃあ今日の夕食の料理は私がします。）
- 유나: 정말요? 고마워요. 그럼 제가 설거지를 할게요.（本当ですか？ありがとうございます。じゃあ私が皿洗いをします。）
- 아빠: 아니에요, 설거지도 제가 할게요. 유나 씨는 그냥 거실에서 쉬세요.（いいえ、皿洗いも私がします。ユナさんはただリビングで休んでください。）
- 유나: 미안해서 어쩌죠? 대신 주말에는 제가 맛있는 음식을 만들어 줄게요.（申し訳なくてどうしましょう？代わりに週末には私が美味しいものを作ってあげます。）
- 아빠: 와, 신난다! 주말에 꼭 만들어 주기 약속해요.（わあ、楽しみ！週末に必ず作ってくれることを約束してね。）
- 유나: 네, 약속할게요!（はい、約束します！）

## 練習

文章の空欄に入る正しい単語を作ってみましょう。

```text
1. 제가 먼저 집으로 ________. (가다 - 意思/約束)
2. 내일부터 담배를 안 ________ 약속해요. (피우다 - 約束)
3. 주말에 같이 미술관에 ________ 해요. (가다 - 決定)
4. 걱정하지 마세요. 제가 꼭 ________. (도와주다 - 意思/約束)
```

**정답**

1. 갈게요
2. 피우기로 / 피우기
3. 가기로
4. 도와줄게요
