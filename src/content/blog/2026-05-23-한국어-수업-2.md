# 수업 2. 책이 어디에 있어요?

前回は「~です」「~は」などの基本表現を学んだ。  
今回は**存在・所有**を表す`있어요/없어요`と、**場所**の表現を学ぶ。

---

## 1. 있어요 / 없어요（います・あります / いません・ありません）

### 意味

| 韓国語 | 日本語 |
|--------|--------|
| 있어요 | います / あります |
| 없어요 | いません / ありません |

原型は`있다 / 없다`。会話では`있어요 / 없어요`を使う。  
人にも物にも両方使える点が、日本語と異なる。

### 用法①：존재（存在）

人や物がどこかに「ある・いる」ことを表す。

```text
고양이가 있어요.       → 猫がいます。
강아지가 없어요.       → 犬がいません。
책이 있어요.           → 本があります。
열쇠가 없어요.         → 鍵がありません。
```

### 用法②：소유（所有）

「〜を持っている・〜がある」という所有を表す。

```text
저는 시간이 있어요.    → 私は時間があります。
저는 돈이 없어요.      → 私はお金がありません。
오빠가 있어요.         → 兄がいます（います）。
형제가 없어요.         → 兄弟がいません。
```

---

## 2. 어디에 있어요?（どこにありますか？）

### 어디 + 에

`어디`は疑問詞で、日本語の「どこ」と一致する。  
場所を表す助詞`에`と組み合わせて使う。

```text
어디에 있어요? → どこにありますか？ / どこにいますか？
```

### 答え方

`[場所] + 에 있어요`で答える。

```text
화장실이 어디에 있어요?        → トイレはどこですか？
  → 저기에 있어요.             → あそこにあります。
  → 2층에 있어요.              → 2階にあります。

편의점이 어디에 있어요?        → コンビニはどこですか？
  → 역 앞에 있어요.            → 駅の前にあります。
```

> ヒント：`어디에 있어요?`は日常でとても頻繁に使う。まずこの一文を丸ごと覚えよう。

---

## 3. 위치 명사（位置を表す名詞）

場所の名詞の後ろに`에`をつけて「〜に」を表す。

| 韓国語 | 日本語 | 例文 |
|--------|--------|------|
| 위 | 上 | 책상 위에 있어요.（机の上にあります。） |
| 아래 / 밑 | 下 | 침대 아래에 있어요.（ベッドの下にあります。） |
| 앞 | 前 | 학교 앞에 있어요.（学校の前にあります。） |
| 뒤 | 後ろ | 건물 뒤에 있어요.（建物の後ろにあります。） |
| 옆 | 横 | 가게 옆에 있어요.（お店の横にあります。） |
| 안 | 中 | 가방 안에 있어요.（カバンの中にあります。） |
| 밖 | 外 | 집 밖에 있어요.（家の外にあります。） |

## 5. 긍정 / 부정 질문（肯定・否定の応答）

前回学んだ`네 / 아니요`と組み合わせる練習をしよう。

```text
고양이가 있어요?                → 猫はいますか？
  네, 있어요.                   → はい、います。
  아니요, 없어요.               → いいえ、いません。

시간이 있어요?                  → 時間はありますか？
  네, 있어요.                   → はい、あります。
  아니요, 지금은 없어요.        → いいえ、今はありません。
```

> `지금은`（今は）のように`은/는`を使うと「今は（他は知らないが）」というニュアンスになる。

## 練習

以下の単語を使って、문장을 만들어 보세요（文章を作ってみましょう）。

```text
강아지（犬）, 고양이（猫）, 물고기（魚）
지갑（財布）, 핸드폰（スマホ）, 우산（傘）
책상 위（机の上）, 서랍 안（引き出しの中）, 가방 안（カバンの中）
```

**例**

- 제 지갑이 가방 안에 있어요.（私の財布はカバンの中にあります。）
- 강아지가 없어요.（犬はいません。）

## Tips

### 이/가 と 은/는 の違い（復習）

| | 使う場面 | 例 |
|--|--------|-----|
| 이/가 | 新しい情報・強調 | 고양이`가` 있어요. |
| 은/는 | 話題の提示・対比 | 고양이`는` 없어요, 강아지는 있어요. |

초보 단계에서는 `있어요/없어요`の前は`이/가`を使うと覚えておこう。

### 助詞まとめ（강의 1・2）

| 意味 | 母音終わり | 子音終わり |
|------|-----------|-----------|
| は | 는 | 은 |
| が | 가 | 이 |
| に（場所） | 에 | 에 |
