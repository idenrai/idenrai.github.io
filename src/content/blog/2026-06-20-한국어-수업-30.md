# 수업 30. 그동안 수고하셨습니다! 종합 복습

드디어 마지막 수업인 **수업 30**에 도달했다!
그동안 수업 1부터 수업 29까지 배운 핵심적인 한국어 문법과 유용한 패턴들을 종합 정리하고 복습해 보자.
완강을 축하하고 격려하는 마지막 정리를 시작한다!

## 1. 핵심 조사 및 기본 어미 정리 (핵심 조동사와 기본 어미 정리)

| 문법 항목 | 표현 형태 | 주요 쓰임새 | 예문 |
|-----------|-----------|-------------|------|
| 명사 주어/속성 | ~예요 / 이에요 | ~입니다 (명사 서술) | 저는 유나**예요**. |
| 대조 / 화제 | ~은 / 는 | ~은 / 는 (주제 표시) | 고양이**는** 있어요. |
| 목적격 조사 | ~을 / 를 | ~을 / 를 (목적어 표시) | 커피**를** 마셔요. |
| 처소 / 장소 | ~에 / 에서 | ~에 (존재/시간), ~에서 (행동) | 2층**에** 있어요. / 회사**에서** 일해요. |
| 동사 기본서술 | ~아요 / 어요 | 해요체 (일상 존댓말) | 밥을 먹**어요**. |

## 2. 주요 시제 및 조동사 정리 (主要な時制と助動詞)

그동안 학습한 핵심 문법들의 형태와 의미를 비교하며 떠올려 보자.

```text
1. 과거형: ~았어요 / 었어요 / 했어요 (수업 7)
   → 어제 영화를 봤어요. (昨日映画を見ました。)

2. 미래/예정: ~(으)ㄹ 거예요 (수업 13)
   → 내일 도서관에 갈 거예요. (明日図書館に行くつもりです。)

3. 희망/욕구: ~고 싶어요 (수업 9)
   → 비빔밥을 먹고 싶어요. (ビビンバが食べたいです。)

4. 가능/불가능: ~(으)ㄹ 수 있어요 / 없어요 (수업 10)
   → 저는 운전할 수 있어요. ( 저는 運転できます。)

5. 의무: ~아야 / 어야 해요 (수업 12)
   → 집에서 푹 쉬어야 해요. (家でゆっくり休まなければなりません。)

6. 경험: ~(으)ㄴ 적이 있다 / 없다 (수업 25)
   → 한라산에 가 본 적이 있어요. ( ハルラ山に行ったことがあります。)
```

## 3. 유용한 연결어미와 상황 표현 (便利な接続表記)

시간 순서, 이유, 조건 등을 연결할 때 사용하는 문법들이다.

```text
1. 나열 / 병렬: ~고 (수업 11)
   → 밥을 먹고 커피를 마셨어요. (ご飯を食べてコーヒーを飲みました。)

2. 이유 / 원인: ~아서/어서 (수업 19), ~기 때문에 (수업 22)
   → 피곤해서 일찍 잤어요. (疲れたので早く寝ました。)
   → 비가 오기 때문에 추워요. (雨が降っているため寒いです。)

3. 역접 / 대조: ~지만, ~는데 (수업 14)
   → 한국어는 어렵지만 재미있어요. (韓国語は難しいですが面白いです。)

4. 가정 / 조건: ~(으)면 (수업 26)
   → 돈이 많으면 차를 살 거예요. (お金がたくさんあれば車を買います。)

5. 목적 / 의도: ~(으)러 가다 (수업 28), ~(으)려고 (수업 28)
   → 영화를 보러 극장에 가요. ( 映画を見に映画館に行きます。)
   → 한국 여행을 하려고 환전해요. (韓国旅行をするために両替します。)
```

## 4. 격려와 작별의 대화 연습

**상황: 완강을 기념하며 부부가 나눈 대화 (完講を記念する会話)**

```text
남편: 유나야, 드디어 수업 30까지 모두 끝났어! 그동안 정말 수고했어.
     (ユナ、ついに授業30まで全部終わったよ！これまで本当にお疲れ様。)

아내: 고마워! 오빠가 매일 재미있게 가르쳐 줘서 포기하지 않고 공부할 수 있었어.
     (ありがとう！お兄さんが毎日楽しく教えてくれたから、諦めずに勉強することができたよ。)

남편: 처음에는 한글 읽기도 어려워했는데, 지금은 한국어로 대화도 잘하네. 정말 대단해!
     (最初はハングルを読むのも大変そうだったのに、今は韓国語で会話もよくできるね。本当にすごい！)

아내: 아직 조금 어렵지만, 앞으로도 더 열심히 공부할래. 한국에 가면 시어머니랑 대화 많이 할 거야.
     (まだちょっと難しいけど、これからももっと一生懸命勉強するね。韓国に行ったらお義母さんとたくさん話すよ。)

남편: 응, 어머님이 정말 기뻐하실 거야. 우리 앞으로도 포기하지 말고 재미있게 한국어 하자! 화이팅!
     (うん, お母さんが本当に喜ばれると思うよ。俺たちこれからも諦めずに楽しく韓国語をやろう！ファイティン！)

아내: 응! 화이팅! 그동안 고마웠어, 내 선생님!
     (うん！ファイティン！今までありがとう、私の先生！)
```
