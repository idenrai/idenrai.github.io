Using encoded image in background-image

Tech/HTML・CSS

2016-12-05 14:15:18

1. 이미지를 Base64로 인코딩

이미지를 Base64로 변환해 주는 사이트는 검색만 하면 다 나오긴 하는데...

몇군데 찾아보니, 여기가 가장 깔끔한 것 같다.

(1MB이상의 파일은 인코딩 불가)


2. 인코딩한 이미지를 CSS에 집어넣기

그냥 아래와 같이 때려넣으면 된다.

1
2
3
.target {
  background-image: url('data:image/png;base64, 인코딩한 이미지 소스');
}
