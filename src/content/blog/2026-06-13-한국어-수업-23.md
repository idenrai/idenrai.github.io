# 수업 23. 저랑 같이 커피 마실래요?

やわらかい提案や相手の意思を尋ねるときの **~(으)ㄹ래요?** と、申し訳なさを表す表現、そして行動を禁止する **~지 마세요** を学ぶ。

## 1. 약속/거절 관련 어휘（約束・断りの語彙）

| 韓国語 | 日本語 | 例文 |
| ---- | ---- | ---- |
| 바쁘다 | 忙しい | 요즘 회사 일이 너무 `바빠요`. |
| 괜찮다 | 大丈夫だ | 내일은 시간 `괜찮아요`. |
| 미안하다 / 죄송하다 | 申し訳ない / すみません | 늦어서 정말 `미안해요`. |
| 약속이 있다/없다 | 予定がある / ない | 오늘 `약속이 있어서` 못 만나요. |
| 시간이 있다/없다 | 時間がある / ない | 지금 대화할 `시간 있어요?` |

---

## 2. ~(으)ㄹ래요?（〜しますか？ / 〜したいですか？ - 意思・提案）

主に親しい間柄で、相手に何かを一緒にしようと提案したり、相手の意思・好みをやわらかく尋ねるときに使う。（自分自身の意思を相手に伝えるときにも使う）

- パッチムがないとき: **~ㄹ래요?**（例: 마시다 → 마실래요?）
- パッチムがあるとき: **~을래요?**（例: 먹다 → 먹을래요?）
- `ㄹ` パッチムのとき: **~ㄹ래요?**（例: 놀다 → 놀래요? ※ㄹの重複なし）

- 오늘 저녁에 저랑 같이 영화 `볼래요?`（今日の夜、私と一緒に映画を見ますか？）
- A: 뭐 `마실래요?`（何を飲みますか？）
  B: 저는 따뜻한 아메리카노 `마실래요`.（私は温かいアメリカーノを飲みます。）
- 내일 우리 집에 놀러 `올래요?`（明日私の家に遊びに来ませんか？）

---

## 3. ~아/어서 죄송해요 / 미안해요（〜してすみません）

理由を表す `~아/어서`（수업 11）と `죄송하다/미안하다` を結合し、申し訳ない理由を具体的に伝えるときに使う。

- 늦`어서` 정말 죄송합니다.（遅れて本当に申し訳ありません。）
- 약속을 어겨`서` 미안해요.（約束を破ってごめんなさい。）
- 전화가 늦`어서` 미안해요.（電話が遅くなってごめんなさい。）

> **注意**：`~아서/어서`の前に過去形を使わないというルールはここでも同じ。
> ❌ `늦었어서 미안해요` → ⭕ **`늦어서 미안해요`**

---

## 4. ~지 마세요（〜しないでください - 禁止）

相手に特定の行動をしないよう、やわらかく指示したり要請するときに使う。
動詞の語幹に、パッチムの有無に関係なく **~지 마세요** をつける。

- 가다 → `가지 마세요`（行かないでください）
- 걱정하다 → `걱정하지 마세요`（心配しないでください）
- 먹다 → `먹지 마세요`（食べないでください）

- 길이 미끄러우니까 `뛰지 마세요`.（道が滑りやすいので走らないでください。）
- 아직 `결정하지 마세요`.（まだ決めないでください。）
- `걱정하지 마세요`. 제가 도와줄게요.（心配しないでください。私が手伝います。）

---

## 5. 회화 연습

**상황：제안과 거절**（提案と断り）

- 아빠: 유나 씨, 오늘 저녁에 퇴근하고 삼겹살 먹으러 갈래요?（ユナさん、今日の夜仕事帰りにサムギョプサル食べに行きませんか？）
- 유나: 아, 정말 가고 싶지만 오늘은 약속이 있어서 안 돼요.（あ、本当に行きたいんですが、今日は約束（予定）があってだめです。）
- 아빠: 아, 그래요? 무슨 약속이 있어요?（あ、そうですか？ 何の約束があるんですか？）
- 유나: 오늘 친구 생일이라서 저녁에 만나기로 했어요. 같이 못 가서 미안해요.（今日は友達の誕生日なので、夜に会うことにしました。一緒に行けなくてごめんなさい。）
- 아빠: 아니에요, 미안해하지 마세요! 친구랑 즐거운 시간 보내세요.（いいえ、申し訳なく思わないでください！友達と楽しい時間を送ってくださいね。）
- 유나: 고마워요. 내일 저녁에는 시간 괜찮으니까 내일 같이 마실래요?（ありがとうございます。明日の夜は時間が大丈夫なので、明日一緒に飲みますか？）
- 아빠: 좋아요! 내일 만나요.（いいですね！明日会いましょう。）

## 練習

文章の空欄に入る正しい単語を作ってみましょう。

```text
1. 저는 매운 음식을 안 좋아해서 안 ________. (먹다 - 自身の意思)
2. 약속 시간에 늦어서 ________. (미안하다 - 理由と謝罪)
3. 극장 안에서는 사진을 ________ 마세요. (찍다 - 禁止)
4. 오늘 우리 집에 놀러 ________? (오다 - 提案)
```

**정답**

1. 먹을래요
2. 미안해요 (または 죄송해요)
3. 찍지
4. 올래요

## Tips

### '죄송해요'와 '미안해요'의 차이（죄송해요 と 미안해요 の違い）
- **죄송합니다 / 죄송해요**: 目上の人やフォーマルな関係（例: 職場の上司、義両親）に使う。
- **미안합니다 / 미안해요**: 同僚、同年代、目下の人、または親密な関係（例: 家族、友人）に使う。
