# 수업 22. 우리 몇 시에 만날까요?

友人や家族と約束を決めるときに使う**約束と提案の表現**を学ぶ。
「何時に会いましょうか？」「どこで会いましょうか？」など相手の意見を聞く方法をマスターしよう。

## 1. 약속 관련 어휘（約束の語彙）

| 韓国語 | 日本語 | 例文 |
| ---- | ---- | ---- |
| 약속 | 約束・予定 | 오늘 저녁에 `약속`이 있어요. |
| 시간 | 時間・都合 | 약속 `시간`이 몇 시예요? |
| 장소 | 場所 | 만날 `장소`를 정해요. |
| 늦다 | 遅れる | 약속에 `늦지 마세요`. |
| 기다리다 | 待つ | 카페에서 친구를 `기다려요`. |
| 만나다 | 会う | 내일 2시에 `만나요`. |

---

## 2. ~(으)ㄹ까요?（〜しましょうか？ / 提案・意見を求める）

相手に何かを一緒にしようと提案したり、相手の意向を尋ねるときに使う。

- パッチムがないとき: **~ㄹ까요?**（例: 만나다 → 만날까요?）
- パッチムがあるとき: **~을까요?**（例: 먹다 → 먹을까요?）
- `ㄹ` パッチムのとき: **~ㄹ까요?**（例: 만들다 → 만들까요? ※ㄹ脱落なし）

- 우리 몇 시에 `만날까요?`（私たち、何時に会いましょうか？）
- 오늘 저녁에 같이 삼겹살 `먹을까요?`（今晩一緒にサムギョプサルを食べましょうか？）
- 커피 한잔 `마실까요?`（コーヒーを一杯飲みましょうか？）

---

## 3. ~(으)ㅂ시다（〜しましょう / 勧誘・提案への同意）

相手に一緒にある行動をしようと要求・勧誘するときに使う。主にフォーマルな状況や、提案を快諾するときに使う。

- パッチムがないとき: **~ㅂ시다**（例: 가다 → 갑시다）
- パッチムがあるとき: **~읍시다**（例: 읽다 → 읽읍시다）
- `ㄹ` パッチムのとき: **~ㅂ시다**（例: 만들다 → 만듭시다 ※ㄹ脱落）

> ※ 目上の人に対しては使わない。同僚や目下の人に対して使う。目上の人には `~아/어 주세요` または `~(으)시지요` を使う。

- A: 어디에서 만날까요?（どこで会いましょうか？）
  B: 서울역 앞 카페에서 `만납시다`.（ソウル駅の前のカフェで会いましょう。）
- A: 같이 갈까요?（一緒に行きましょうか？）
  B: 네, 같이 `갑시다`.（はい、一緒に行きましょう。）

> 会話では、`만납시다`よりも해요체の`만나요`と言う方が自然。

---

## 4. ~기 때문에（〜だから / 〜のために・理由）

ある行動や状態の確実な原因・理由を表す。`~아서/어서`よりも明確でフォーマルな理由表現。

- 動詞/形容詞の語幹にパッチムの有無に関係なく **~기 때문에** をつける。
  （例: 바쁘다 → 바쁘기 때문에, 비가 오다 → 비가 오기 때문에）
- 名詞の後ろにはそのまま **~때문에** をつける。
  （例: 비 → 비 때문에, 감기 → 감기 때문에）

- 약속이 있`기 때문에` 먼저 가야 해요.（予定があるため、先に行かなければなりません。）
- 비가 오`기 때문에` 날씨가 추워요.（雨が降っているため天気が寒いです。）
- 눈 `때문에` 길이 미끄러워요.（雪のせいで道が滑りやすいです。）

---

## 5. 회화 연습

**상황：주말 약속 정하기**（週末の約束決め）

- 남편: 이번 주 토요일에 우리 영화 보러 갈까요?（今週の土曜日に映画を見に行きましょうか？）
- 아내: 좋아요! 어떤 영화를 볼까요?（いいですね！どんな映画を見ましょうか？）
- 남편: 재미있는 한국 영화가 있어요. 같이 봐요.（面白い韓国映画があります。一緒に見ましょう。）
- 아내: 네, 몇 시에 만날까요?（はい、何時に会いましょうか？）
- 남편: 영화가 2시에 시작하기 때문에 1시 반에 만납시다.（映画が2時に始まるので、1時半に会いましょう。）
- 아내: 어디에서 만날까요? 극장 앞 카페가 어때요?（どこで会いましょうか？映画館の前のカフェはどうですか？）
- 남편: 좋아요. 거기서 기다릴게요. 늦지 마세요!（いいですね。そこで待ってます。遅れないでくださいね！）

## 練習

文章の意味が通るように、カッコの中の単語を正しい形に書き換えてください。

```text
1. 주말에 우리 같이 쇼핑 ________? (하다 - 제안)
2. 비가 ________ 때문에 오늘은 집에서 쉴래요. (오다 - 이유)
3. 2층 식당에서 점심을 ________. (먹다 - 권유/동의)
4. 차가 막히기 ________ 약속에 늦었어요. (때문)
```

**정답**

1. 할까요?
2. 오기
3. 먹읍시다 (または 먹어요)
4. 때문에

## Tips

### 약속을 거절하거나 변경할 때（約束を断ったり変更するとき）

相手からの提案を断る場合、以下のように婉曲に断ったり申し訳なさを表現する。
- **미안해요. 그날은 선약이 있어요.**（すみません。その日は先約があります。）
- **제가 요새 좀 바쁘기 때문에 다음 주에 만나요.**（私が最近少し忙しいので、来週会いましょう。）
