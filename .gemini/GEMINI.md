# 자동화 배포 (Automated Deployment)
- 블로그에 강의 등 새로운 콘텐츠를 작성한 후에는 반드시 `git add`, `git commit`, `git push`를 실행하여 Github에 푸시해야 합니다. (작업이 완료된 후 항상 자동으로 진행)
- 커밋 메시지는 `.agent/rules/git-commit-rules.md`의 규칙을 따릅니다.
