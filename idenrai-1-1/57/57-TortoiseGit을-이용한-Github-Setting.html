2. TortoiseGit 설치


3. 로컬 저장소 만들기

탐색기를 열고 저장소로 사용할 적절한 위치로 이동

이후 탐색기 빈공간에 우측마우스를 클릭하여 TortoiseGit > Settings 메뉴를 호출

Git 항목을 선택한 후 아래와 같이 본인의 이름과 이메일 주소를 입력 후 저장

저장소로 사용할 폴더 생성 후, 마우스 우클릭으로 Git Create repository here... 메뉴를 호출


4. 로컬 저장소에 저장

해당 폴더에서 우측마우스를 눌러 Git Commit -> "master".. 메뉴를 선택

Commit 창이 보이면 수정된 내용과 적용할 파일(체크박스)를 선택 하고 [OK] 버튼을 클릭하여 저장소에 저장

이후 Push 버튼을 눌러 원격저장소에도 저장


5. 원격 저장소에 저장 (GitHub 반영)

Push 버튼을 클릭 후, 원격지정보를 등록하기 위해 [Manage] 버튼을 클릭

원격지 이름(Remote)와 원격지 주소(URL)만 입력하고 [Add New/Save] 버튼을 클릭하여 원격지 정보를 저장

원격지 정보를 등록하면 Destination 목록에 위의 원격지 이름이 등록됨
