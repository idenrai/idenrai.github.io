# 수업 24. 제 취미는 요리하는 것이에요

自分の趣味を説明するときに動詞を名詞に変える **~는 것** と、二つの行動を同時に行うときに使う **~(으)면서** を学ぶ。

## 1. 취미 관련 어휘（趣味の語彙）

| 韓国語 | 日本語 | 例文 |
| ---- | ---- | ---- |
| 취미 | 趣味 | `취미`가 뭐예요? |
| 독서 (책 읽기) | 読書 (本を読むこと) | 제 취미는 `독서`예요. |
| 등산 | 登山 / ハイキング | 주말에 `등산`을 가요. |
| 요리 (요리하다) | 料理 (料理する) | 저는 `요리`하는 걸 좋아해요. |
| 여행 (여행하다) | 旅行 (旅行する) | 매년 해외`여행`을 가요. |
| 사진 찍기 | 写真撮影 | 경치가 좋아서 `사진을 찍었어요`. |
| 영화 감상 | 映画鑑賞 | 집에서 넷플릭스로 `영화 감상`을 해요. |

---

## 2. ~는 것（〜すること - 動詞の名詞化）

動詞の後ろに `~는 것` をつけて、動詞を名詞の形にする。（パッチムの有無は関係ない）
これを活用すると、「〜するのが好きです」「〜するのが趣味です」などの表現が作れる。

- 보다 → `보는 것`
- 먹다 → `먹는 것`
- 요리하다 → `요리하는 것`

> ※ 会話体では `~는 것` が **~는 거** に、`~는 것은` が **~는 건** に、`~는 것을` が **~는 걸** に省略されてよく使われる。

- 제 취미는 `요리하는 것`이에요.（私の趣味は料理をすることです。）
  - 会話体: 제 취미는 `요리하는 거`예요.
- 저는 음악 `듣는 것을` 좋아해요.（私は音楽を聞くことが好きです。）
  - 会話体: 저는 음악 `듣는 걸` 좋아해요.
- 사진 `찍는 것은` 재미있어요.（写真を撮ることは面白いです。）
  - 会話体: 사진 `찍는 건` 재미있어요.

---

## 3. ~(으)면서（〜しながら - 同時進行）

一人の人が二つ以上の行動を同時に行うときに使う。

- パッチムがないとき: **~면서**（例: 보다 → 보면서）
- パッチムがあるとき: **~으면서**（例: 먹다 → 먹으면서）
- `ㄹ` パッチムのとき: **~면서**（例: 만들다 → 만들면서 ※ㄹ脱落なし）

- 음악을 들`으면서` 공부를 해요.（音楽を聞きながら勉強をします。）
- 저는 텔레비전을 보`면서` 밥을 먹어요.（私はテレビを見ながらご飯を食べます。）
- 이야기하`면서` 커피를 마실까요?（話しながらコーヒーを飲みましょうか？）

---

## 4. 회화 연습

**상황：취미에 대해 이야기하기**（趣味についての会話）

- 아빠: 유나 씨는 주말에 보통 뭐 해요?（ユナさんは週末に普通何をしますか？）
- 유나: 저는 집에서 맛있는 음식을 요리하는 걸 좋아해요.（私は家で美味しい料理を作るのが好きです。）
- 아빠: 와, 요리가 취미군요! 요리하면서 음악도 들어요?（わあ、料理が趣味なんですね！料理しながら音楽も聞きますか？）
- 유나: 네, 신나는 음악을 들으면서 요리하면 기분이 좋아져요. 아빠는 취미가 뭐예요?（はい、楽しい音楽を聞きながら料理をすると気分が良くなります。パパの趣味は何ですか？）
- 아빠: 저는 산책하면서 경치 사진을 찍는 것을 좋아해요.（私は散歩しながら景色の写真を撮るのが好きです。）
- 유나: 멋지네요! 다음에 사진 찍으러 갈 때 저도 같이 가요.（素敵ですね！今度写真を撮りに行くとき、私も一緒に行きましょう。）

## 練習

文章の空欄に入る正しい単語を作ってみましょう。

```text
1. 제 취미는 한국 드라마를 ________ 것입니다. (보다 - 名詞化)
2. 커피를 ________ 신문을 읽어요. (마시다 - 同時進行)
3. 주말에는 집에서 맛있는 것을 ________ 것을 좋아해요. (만들다 - 名詞化)
4. 운전________ 전화를 하면 위험해요. (운전하다 - 同時進行)
```

**정답**

1. 보는
2. 마시면서
3. 만드는 (만들다 + 는 것 = 만드는 것 ※ㄹ脱落)
4. 하면서 (운전하면서)

## Tips

### 자주 쓰이는 취미 관련 표현（よく使われる趣味関連の表現）

- **~을/를 좋아해요**（〜が好きです）
  - 책 읽는 것을 좋아해요.
- **~에 소질이 있어요 / 잘해요**（〜に素質があります / 得意です）
  - 노래하는 걸 잘해요. (노래를 잘 부릅니다)
