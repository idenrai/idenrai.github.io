# 수업 11. 왜요? 그래서요?

今回は**文をつなげる表現**を学ぶ。  
「〜して、〜」「〜だから、〜」など、2つの文を自然につなげることで、一気に表現の幅が広がる。

---

## 1. ~고（〜して / 〜で）

2つの動作や状態を並列につなぐ。日本語の「〜して」「〜で」に相当。

```text
동사/형용사 어간 + 고
```

```text
밥을 먹고 커피를 마셔요.
→ ご飯を食べてコーヒーを飲みます。

샤워하고 잠을 자요.
→ シャワーを浴びて寝ます。

크고 예뻐요.
→ 大きくてきれいです。（수업 8에서 학습）
```

> `~고`は前後の順番を表すこともあるが、単純な並列にも使える。  
> 時制は文末だけで表す。

```text
밥을 먹고 커피를 마셨어요.  → ご飯を食べてコーヒーを飲みました。（過去は文末だけ）
```

---

## 2. ~아서/어서（〜して / 〜だから）

**理由**または**先行動作**を表す。日本語の「〜なので」「〜して（それから）」に相当。

```text
동사/형용사 어간 + 아서/어서
```

- 語幹の最後の母音が`ㅏ/ㅗ` → `아서`
- それ以外 → `어서`
- `하다` → `해서`

### 理由・原因（〜なので）

```text
피곤해서 일찍 잤어요.       → 疲れたので早く寝ました。
바빠서 못 갔어요.            → 忙しくて行けませんでした。
맛있어서 많이 먹었어요.     → おいしくてたくさん食べました。
비가 와서 집에 있었어요.    → 雨が降ったので家にいました。
```

### 先行動作（〜して、それから）

```text
역에 가서 친구를 만났어요.  → 駅に行って友達に会いました。
집에 와서 밥을 먹었어요.    → 家に帰ってご飯を食べました。
```

> **重要**：`~아서/어서`は`~고`と違い、**時制を変えてはいけない**。  
> ❌ `피곤했어서 잤어요` → ⭕ `피곤해서 잤어요`

---

## 3. ~고 と ~아서/어서 の違い

| | ~고 | ~아서/어서 |
|--|-----|-----------|
| 意味 | 〜して（並列・順序） | 〜なので / 〜して（理由・原因） |
| 時制変化 | 文末のみ | **前節は変えない** |
| 例 | 밥을 먹고 잤어요. | 배가 불러서 잤어요. |

```text
샤워하고 잤어요.             → シャワーを浴びて寝ました。（順序）
피곤해서 잤어요.             → 疲れたので寝ました。（理由）
```

---

## 4. 그래서 / 그런데 / 그리고（接続詞）

文と文をつなぐ接続詞。会話の流れを作るのに必須。

| 韓国語 | 意味 | 使う場面 |
|--------|------|----------|
| 그래서 | だから / それで | 前の文が理由 |
| 그런데 | でも / ところで | 逆接・話題転換 |
| 그리고 | そして / それから | 並列・追加 |
| 그러면 / 그럼 | それなら / じゃあ | 条件・話の展開 |
| 그래도 | それでも | 逆接（譲歩） |

```text
배가 고파요. 그래서 밥을 먹었어요.
→ お腹がすいています。だからご飯を食べました。

오늘 바빠요. 그런데 내일은 괜찮아요.
→ 今日は忙しいです。でも明日は大丈夫です。

한국어를 공부했어요. 그리고 드라마도 봤어요.
→ 韓国語を勉強しました。そしてドラマも見ました。

비가 와요. 그래도 갈 거예요.
→ 雨が降っています。それでも行くつもりです。
```

---

## 5. 회화 연습（会話練習）

```text
왜 한국어를 공부해요?
（なぜ韓国語を勉強しているんですか？）

  → 한국 드라마를 좋아해서요.
    → 韓国ドラマが好きだからです。

  → 한국에 가고 싶어서요.
    → 韓国に行きたいからです。

  → 한국 가족이 있어서요.
    → 韓国に家族がいるからです。
```

> 理由だけ答えるときは`~아서/어서요`で文末を止めるのが自然。

---

## 練習

두 문장을 ~고 또는 ~아서/어서로 연결해 보세요.  
（2つの文を〜고または〜아서/어서でつなげてみましょう。）

```text
1. 밥을 먹었어요. + 커피를 마셨어요.   → ~고로 연결
2. 피곤했어요.    + 일찍 잤어요.        → ~아서/어서로 연결
3. 마트에 갔어요. + 장을 봤어요.        → ~아서/어서로 연결
4. 날씨가 좋아요. + 기분이 좋아요.      → ~아서/어서로 연결
```

**정답**

```text
1. 밥을 먹고 커피를 마셨어요.
2. 피곤해서 일찍 잤어요.
3. 마트에 가서 장을 봤어요.
4. 날씨가 좋아서 기분이 좋아요.
```

---

## Tips

### ~아서/어서 앞에 과거형을 쓰지 않는 이유

`~아서/어서`는 앞뒤 문장이 **인과관계**로 묶여 있기 때문에, 시제는 문장 전체의 끝에서 한 번만 표현한다.

```text
❌ 피곤했어서 잤어요.
⭕ 피곤해서 잤어요.

❌ 바빴어서 못 갔어요.
⭕ 바빠서 못 갔어요.
```

이 규칙は`~아서/어서`特有のもの。`~고`には適用されない。
