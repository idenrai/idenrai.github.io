# 수업 19. 요즘 너무 행복해요!

今回は**감정·상태 표현**（感情・状態の表現）を学ぶ。  
自分の気持ちを韓国語で伝えられると、会話がぐっと豊かになる。

---

## 1. 기본 감정 어휘 (基本の感情語彙)

| 한국어 | 일본어 | 활용형 |
|--------|--------|--------|
| 기쁘다 | 嬉しい | 기뻐요 |
| 슬프다 | 悲しい | 슬퍼요 |
| 행복하다 | 幸せだ | 행복해요 |
| 불행하다 | 不幸だ | 불행해요 |
| 화가 나다 | 怒る | 화가 나요 |
| 무섭다 | 怖い | 무서워요 |
| 외롭다 | 寂しい | 외로워요 |
| 그립다 | 懐かしい・恋しい | 그리워요 |
| 부끄럽다 | 恥ずかしい | 부끄러워요 |
| 설레다 | ときめく・わくわくする | 설레요 |

※ `무섭다`, `외롭다`, `그립다`, `부끄럽다`는 ㅂ불규칙！(수업 17 복습)

```text
오늘 너무 기뻐요!               → 今日すごく嬉しいです！
엄마가 그리워요.                → お母さんが恋しいです。
한국에 가서 설레요.             → 韓国に行くのでわくわくします。
```

---

## 2. 신체·컨디션 표현 (体・コンディション)

```text
피곤해요.            → 疲れています。
졸려요.              → 眠いです。
배고파요.            → お腹が空いています。
배불러요.            → お腹がいっぱいです。
아파요.              → 痛いです・具合が悪いです。
괜찮아요.            → 大丈夫です。
힘들어요.            → 辛い・しんどいです。
스트레스를 받아요.   → ストレスを受けています。
```

### 어디가 아파요? (どこが痛いですか？)

```text
머리가 아파요.       → 頭が痛いです。
배가 아파요.         → お腹が痛いです。
목이 아파요.         → 喉が痛いです。
허리가 아파요.       → 腰が痛いです。
다리가 아파요.       → 足が痛いです。
```

---

## 3. ~아/어서 (〜なので・〜して)

原因・理由、または前の動作に続く行動を表す接続表現。

```text
동사/형용사 어간 + 아/어서
```

```text
피곤해서 일찍 잘 거예요.         → 疲れているので早く寝るつもりです。
배고파서 밥을 먹었어요.          → お腹が空いたのでご飯を食べました。
기뻐서 눈물이 났어요.            → 嬉しくて涙が出ました。
한국어가 재미있어서 매일 공부해요. → 韓国語が面白いので毎日勉強します。
시어머니를 만나서 기뻐요.        → お義母さんに会えて嬉しいです。
```

※ `~아/어서`는 과거형과 함께 쓰지 않는다.  
→ ❌ ~~피곤했어서~~ ✅ 피곤해서

## 4. 감정 표현 심화 (感情表現の深掘り)

### 정도 표현 (程度の表現)

| 한국어 | 일본어 |
|--------|--------|
| 너무 | 〜すぎる・とても |
| 정말 / 진짜 | 本当に |
| 조금 / 좀 | 少し |
| 별로 | あまり（〜でない） |
| 엄청 | めちゃくちゃ |

```text
너무 행복해요!          → すごく幸せです！
별로 안 피곤해요.       → あまり疲れていません。
엄청 설레요!            → めちゃくちゃわくわくします！
진짜 기뻐요!            → 本当に嬉しいです！
```

## 5. 회화 연습

**상황：한국 방문을 앞두고**（韓国訪問を前にして）

```text
남편: 다음 달에 한국 가는 거 기대돼요?
     （来月韓国に行くの、楽しみですか？）

아내: 네! 정말 설레요. 근데 시어머니 만나는 게 좀 긴장돼요.
     （はい！本当にわくわくします。でもお義母さんに会うのが少し緊張します。）

남편: 왜요? 걱정하지 마세요. 장모님도 처음엔 긴장했잖아요.
     （なぜですか？心配しないでください。お義母さんも最初は緊張してたじゃないですか。）

아내: 그래도 한국어를 못해서 걱정이에요.
     （でも韓国語ができないから心配です。）

남편: 지금 열심히 공부하고 있잖아요. 분명히 기뻐하실 거예요.
     （今一生懸命勉強してるじゃないですか。きっと喜ばれますよ。）

아내: 맞아요! 그리워서 빨리 보고 싶어요.
     （そうですよね！恋しくて早く会いたいです。）
```

## 練習

**빈칸에 알맞은 감정 표현을 넣어 보세요。（空欄に合う感情表現を入れてみましょう。）**

1. 시험에 합격해서 너무 \_\_\_\_\_\_.  (嬉しくて)
2. 혼자 있어서 \_\_\_\_\_\_.  (寂しい)
3. 내일 여행을 가서 \_\_\_\_\_\_.  (わくわくする)
4. 많이 걸어서 \_\_\_\_\_\_.  (疲れた)

**정답 (答え)**

```text
1. 기뻐요.
2. 외로워요.
3. 설레요.
4. 피곤해요.
```

## Tips

### 감정 표현의 ㅂ불규칙 정리

감정 어휘에 ㅂ불규칙이 많다 — 모아서 외워두면 편리！

| 기본형 | 해요체 |
|--------|--------|
| 기쁘다 | 기뻐요 |
| 슬프다 | 슬퍼요 |
| 무섭다 | 무서워요 |
| 외롭다 | 외로워요 |
| 그립다 | 그리워요 |
| 부끄럽다 | 부끄러워요 |

### ~고 싶다 vs ~아/어서

```text
보고 싶어요.              → 会いたいです。（希望）
보고 싶어서 왔어요.       → 会いたくて来ました。（理由）
```
