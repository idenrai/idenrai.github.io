# 수업 14. 비싸지만 맛있어요!

今回は**逆接の接続表現**を学ぶ。  
「高いけど美味しい」「忙しいけど行く」など、二つの文を「〜だけど」「〜なのに」でつなぐ表現。

---

## 1. ~지만（〜ですが・〜けれど）

二つの文を逆接でつなぐ最も基本的な表現。

```text
동사/형용사 어간 + 지만
```

語幹の形に関わらず、そのまま `지만` をつける。（母音語幹・子音語幹共通）

| 原型 | ~지만 | 意味 |
|------|-------|------|
| 비싸다 | 비싸지만 | 高いけど |
| 맛있다 | 맛있지만 | 美味しいけど |
| 바쁘다 | 바쁘지만 | 忙しいけど |
| 좋다 | 좋지만 | 良いけど |
| 어렵다 | 어렵지만 | 難しいけど |
| 알다 | 알지만 | 知っているけど |

```text
비싸지만 맛있어요.          → 高いけど美味しいです。
바쁘지만 갈 거예요.         → 忙しいけど行くつもりです。
알지만 말하기 싫어요.       → 知ってるけど言いたくないです。
어렵지만 재미있어요.        → 難しいけど面白いです。
한국어를 공부하지만 아직 어려워요. → 韓国語を勉強しているけど、まだ難しいです。
```

---

## 2. ~는데 / ~은데 / ~ㄴ데（〜なのに・〜ですが）

`~지만`と似ているが、少し話の流れをつなぐ「文脈提供」のニュアンスが強い。  
「〜なんですが（どうですか？）」のように、相手の反応を促す使い方も多い。

```text
동사 현재형: 어간 + 는데
형용사:
  - 자음 어간 + 은데
  - 모음 어간 + ㄴ데
명사: 명사 + 인데
```

| 表現 | 活用 | 意味 |
|------|------|------|
| 가다 | 가는데 | 行くんですが |
| 먹다 | 먹는데 | 食べるんですが |
| 바쁘다 | 바쁜데 | 忙しいんですが |
| 좋다 | 좋은데 | 良いんですが |
| 학생이다 | 학생인데 | 学生なんですが |

```text
배가 고픈데 뭐 먹을 거예요?     → お腹が空いてるんですが、何食べますか？
바쁜데 잠깐 얘기해도 돼요?      → 忙しいところ、ちょっと話してもいいですか？
이게 맛있는데 한번 먹어 봐요.   → これ美味しいんですが、一度食べてみてください。
저는 일본인인데 한국어를 배우고 있어요. → 私は日本人ですが、韓国語を勉強しています。
```

---

## 3. ~지만 vs ~는데 ニュアンス比較

| 표현 | 뉘앙스 | 예 |
|------|--------|----|
| ~지만 | 明確な逆接「〜けれど」 | 비싸지만 살 거예요. (高いけど買います) |
| ~는데 | 状況説明「〜んですが」 | 비싼데 살 거예요? (高いんですが、買いますか?) |

`~는데`는 逆接だけでなく、状況・背景の説明にも使える。

```text
저는 한국어를 공부하는데, 같이 공부해요! → 私は韓国語を勉強してるんですが、一緒にやりましょう！
오늘은 바쁜데, 내일은 어때요?           → 今日は忙しいんですが、明日はどうですか？
```

## 4. 회화 연습

**상황：남편과 주말 계획**（夫と週末の計画）

```text
아내: 이번 주말에 뭐 할 거예요?
      (今週末、何するつもりですか？)

남편: 친구를 만날 거예요. 같이 갈래요?
      (友達に会うつもりです。一緒に行きますか？)

아내: 가고 싶지만 집이 너무 지저분해요.
      (行きたいけど、家が散らかりすぎてて。)

남편: 청소를 다 하고 오면 되잖아요!
      (掃除を全部してから来ればいいじゃないですか！)

아내: 청소는 해야 하는데 귀찮아요…
      (掃除はしなきゃいけないんですが、めんどくさいです…)

남편: 같이 하면 빠르지만 혼자 하면 힘들어요.
      (一緒にやれば早いけど、一人でやると大変ですよ。)
```

## 練習

**다음을 ~지만을 사용해서 연결하세요。（~지만を使ってつなげましょう。）**

1. 맛있다 + 너무 달다  
2. 한국어가 어렵다 + 재미있다  
3. 비가 오다 + 나가고 싶다  
4. 바쁘다 + 전화할게요  

**정답 (答え)**

```text
1. 맛있지만 너무 달아요.      → 美味しいけど甘すぎます。
2. 한국어가 어렵지만 재미있어요. → 韓国語は難しいけど面白いです。
3. 비가 오지만 나가고 싶어요.  → 雨が降ってるけど出かけたいです。
4. 바쁘지만 전화할게요.        → 忙しいけど電話します。
```

## Tips

### ~지만 と ~는데 の使い分けポイント

- **~지만** → 「でも・けれど」という逆接の気持ちが強いとき
- **~는데** → 前の内容を背景・状況として伝えて、後に続けるとき

どちらも使える場面は多いが、日本語の「〜けど」に近い感覚で使い始めて、徐々に使い分けを体得するのが近道。

### 便利な組み合わせ表現

| 表현 | 의미 |
|------|------|
| 좋지만... | いいんだけど… |
| 알지만... | 知ってるけど… |
| 하고 싶지만... | したいけど… |
| 가고 싶은데... | 行きたいんだけど… |
| 시간이 없는데... | 時間がないんだけど… |
