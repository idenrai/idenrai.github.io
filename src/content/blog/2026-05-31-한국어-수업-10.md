# 수업 10. 한국어 할 수 있어요?

今回は**可能・不可能**を表す表現を学ぶ。  
「〜できます」「〜できません」は日常会話で非常に頻繁に使う。

---

## 1. ~(으)ㄹ 수 있어요 / 없어요（〜できます / できません）

動詞の語幹に`(으)ㄹ 수 있어요`をつける。

```text
동사 어간 + (으)ㄹ 수 있어요
```

- 語幹が母音または`ㄹ`で終わる → `ㄹ 수 있어요`
- 語幹が子音で終わる → `을 수 있어요`

| 原型 | ~ㄹ/을 수 있어요 | 意味 |
|------|-----------------|------|
| 가다 | 갈 수 있어요 | 行けます |
| 오다 | 올 수 있어요 | 来られます |
| 먹다 | 먹을 수 있어요 | 食べられます |
| 마시다 | 마실 수 있어요 | 飲めます |
| 하다 | 할 수 있어요 | できます |
| 보다 | 볼 수 있어요 | 見られます |
| 운전하다 | 운전할 수 있어요 | 運転できます |
| 수영하다 | 수영할 수 있어요 | 泳げます |

### 없어요（できません）

`있어요`を`없어요`に変えるだけ。

```text
갈 수 없어요.            → 行けません。
먹을 수 없어요.          → 食べられません。
할 수 없어요.            → できません。
```

---

## 2. 못（〜できない）との違い

「できない」を表すには`못`と`~ㄹ/을 수 없어요`の二通りがある。

| 表現 | ニュアンス | 例 |
|------|-----------|-----|
| 못 + 동사 | 端的に「できない」 | 못 가요. |
| ~ㄹ/을 수 없어요 | 丁寧・理由が含まれる感じ | 갈 수 없어요. |

```text
못 먹어요.               → 食べられません。（アレルギーなど）
먹을 수 없어요.          → 食べられません。（理由があって）

오늘 못 가요.            → 今日行けません。
오늘은 갈 수 없어요.     → 今日は行けません。
```

> 日常会話では`못`の方が自然でよく使われる。`~ㄹ 수 없어요`は少し丁寧な響き。

---

## 3. ~ㄹ/을 수 있어요?（〜できますか？）

文末を上げるだけで疑問文になる。

```text
한국어 할 수 있어요?     → 韓国語できますか？
  → 네, 조금 할 수 있어요.    → はい、少しできます。
  → 아직 잘 못해요.            → まだうまくできません。

이거 먹을 수 있어요?     → これ食べられますか？
  → 네, 먹을 수 있어요.        → はい、食べられます。
  → 아니요, 매워서 못 먹어요.  → いいえ、辛くて食べられません。
```

---

## 4. 과거형（過去形）

`있어요 → 있었어요`に変える。

```text
어렸을 때 수영할 수 있었어요.
→ 小さいころ泳げました。

예전에는 매운 거 먹을 수 있었어요.
→ 以前は辛いものが食べられました。
```

---

## 5. 회화 연습（会話練習）

### 무엇을 할 수 있어요?（何ができますか？）

```text
요리할 수 있어요?
  → 네, 간단한 건 할 수 있어요.  → はい、簡単なものはできます。
  → 별로 못해요.                  → あまりできません。

운전할 수 있어요?
  → 네, 할 수 있어요.             → はい、できます。
  → 아니요, 면허가 없어요.        → いいえ、免許がありません。
```

### 같이 ~할 수 있어요?（一緒に〜できますか？）

```text
내일 같이 갈 수 있어요?  → 明日一緒に行けますか？
  → 네, 갈 수 있어요.    → はい、行けます。
  → 내일은 좀 어려워요.  → 明日はちょっと難しいです。
```

## 練習

다음 문장을 한국어로 만들어 보세요.（以下の文を韓国語で作ってみましょう。）

```text
1. 日本語が話せますか？
2. はい、少し話せます。
3. 辛いものは食べられません。
4. 明日来られますか？
5. 以前はもっとうまくできました。
```

**정답**

```text
1. 일본어 할 수 있어요? / 일본어 말할 수 있어요?
2. 네, 조금 할 수 있어요.
3. 매운 건 못 먹어요. / 매운 걸 먹을 수 없어요.
4. 내일 올 수 있어요?
5. 예전에는 더 잘할 수 있었어요.
```

## Tips

### 잘 / 조금 / 별로（上手に / 少し / あまり）

できる度合いを表す副詞と組み合わせると自然な会話になる。

```text
잘 할 수 있어요.         → うまくできます。
조금 할 수 있어요.       → 少しできます。
별로 못해요.             → あまりできません。
전혀 못해요.             → 全然できません。
```

### 수업 1~10 핵심 문법 누적 정리

| 文法 | 形 | 意味 |
|------|----|------|
| 현재 | ~아요/어요 | 〜します |
| 과거 | ~았어요/었어요 | 〜しました |
| 희망 | ~고 싶어요 | 〜したいです |
| 가능 | ~ㄹ/을 수 있어요 | 〜できます |
| 불가능 | ~ㄹ/을 수 없어요 / 못 | 〜できません |
| 목적 | ~(으)러 가다 | 〜しに行く |
