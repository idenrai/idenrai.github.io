# 수업 4. 처음 뵙겠습니다

挨拶と丁寧な表現を学ぶ。初対面の場や目上の人に対して使う表現が中心。

## 1. 기본 인사（基本の挨拶）

### 만날 때（会うとき）

| 韓国語 | 意味 |
| ---- | ---- |
| 안녕하세요. | こんにちは（丁寧。朝・昼・夜すべてに使える） |
| 안녕. | やあ（友人同士） |
| 어서 오세요. | いらっしゃいませ |

### 헤어질 때（別れるとき）

| 韓国語 | 意味 | 使う場面 |
| ---- | ---- | ---- |
| 안녕히 계세요. | さようなら | 自分が**去る**側 → 残る相手に |
| 안녕히 가세요. | さようなら | 自分が**残る**側 → 去る相手に |
| 또 봐요. | またね | 親しい間柄 |

> 日本語は「さようなら」一言だが、韓国語は**誰が動くか**で使い分ける。まず`안녕히 가세요`を覚えるだけでOK。

---

## 2. 처음 만날 때 / 오랫만에 만날 때

### 처음 만날 때（初対面のとき）

- `처음 뵙겠습니다.`（はじめまして。）
- 저는 [이름]`이라고 해요.`（私は[名前]`と申します。`）
- `잘 부탁드립니다.`（よろしくお願いします。）

> `뵙겠습니다`は`보다`（見る）の謙譲語。

### 오랫만에 만날 때（久しぶりに会うとき）

- `안녕하셨어요?`（お変わりありませんでしたか？）
- `오랫만이에요.`（お久しぶりです。）
- `보고 싶었어요.`（会いたかったです。）
- `잘 지내셨어요?`（お元気でしたか？）

> `안녕하셨어요?`は`안녕하세요`の過去形。久しぶりの目上への挨拶として自然。

---

## 3. 감사와 사과（感謝と謝罪）

### 감사（感謝）

| 韓国語 | 意味 | ニュアンス |
| ---- | ---- | ---- |
| 감사합니다. | ありがとうございます | フォーマル |
| 고마워요. | ありがとう | 友人・同等の相手 |
| 고마워. | ありがとう | 親しい間柄 |

> 시어머니（義母）には必ず`감사합니다`を使う。

### 사과（謝罪）

| 韓国語 | 意味 |
| ---- | ---- |
| 죄송합니다. | 申し訳ございません（フォーマル） |
| 미안해요. | ごめんなさい（カジュアル） |

### 괜찮아요（大丈夫です）

謝られたときの返し方や、体調を聞かれたときにも使える万能フレーズ。

- A: 죄송합니다.（申し訳ありません。）
- B: `괜찮아요.`（`大丈夫ですよ。`）

---

## 4. 식사 표현（食事の表現）

韓国の家庭では食事の挨拶がとても大切。

| タイミング | 韓国語 | 意味 |
| ---- | ---- | ---- |
| 食べる前 | 잘 먹겠습니다. | いただきます。 |
| 食べた後 | 잘 먹었습니다. | ごちそうさまでした。 |
| 料理を褒める | 맛있어요! | おいしいです！ |
| もっとほしい | 더 주세요. | もっとください。 |
| お腹いっぱい | 배불러요. | お腹がいっぱいです。 |

> `잘 먹겠습니다 / 잘 먹었습니다`は日本語の「いただきます/ごちそうさま」に完全対応。

---

## 5. 기타 유용한 표현（その他の便利なフレーズ）

| 韓国語 | 意味 |
| ---- | ---- |
| 네. | はい。 |
| 아니요. | いいえ。 |
| 알겠어요. | わかりました。 |
| 모르겠어요. | わかりません。 |
| 잠깐만요. | 少々お待ちください。 |
| 천천히 말해 주세요. | ゆっくり話してください。 |
| 다시 한번 말해 주세요. | もう一度言ってください。 |

> 最後の二つは特に重要。聞き取れなかったときにすぐ使えるようにしよう。

## 練習

오랫만에 시어머니 댁을 방문する場面を想像しながら、以下の順番で言ってみましょう。

```text
--- （到着、玄関で） ---
1. 안녕하셨어요?
2. 오랫만이에요. 보고 싶었어요.
--- （食事が出てきたら） ---
3. 감사합니다. 잘 먹겠습니다.
4. 맛있어요!
--- （お腹がいっぱいになったら） ---
5. 배불러요. 잘 먹었습니다.
--- （帰るとき） ---
6. 감사합니다. 또 올게요.
7. 안녕히 계세요.
```

## Tips

### 존댓말의 레벨（丁寧さのレベル）

| レベル | 語尾の特徴 | 使う相手 |
| ---- | ---- | ---- |
| 합쇼체（最丁寧） | ~합니다 / ~습니다 | 公式の場・初対面の目上 |
| 해요체（丁寧） | ~아요 / ~어요 | 日常の目上・知人 |
| 해체（タメ口） | ~아 / ~어 | 親しい友人 / 家族 |

시어머니には**해요체**または**합쇼체**を使う。今回の表現(`감사합니다`, `잘 먹겠습니다`)は합쇼체。
