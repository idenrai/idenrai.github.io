# 수업 3. 뭐 해요?

今回は韓国語の**動詞の丁寧形**を学ぶ。  
これを覚えると「食べます」「行きます」「します」など、日常のほぼすべての動作を表現できるようになる。

---

## 1. 해요체（ヘヨ体）とは

韓国語の丁寧な会話体を**해요체**という。  
動詞の原型（~다）から`다`を取り、語幹に`아요 / 어요`をつけて作る。

> 強의 1で学んだ`예요/이에요`は名詞につく。  
> 今回の`아요/어요`は**動詞・形容詞**につく。

---

## 2. 아요 / 어요 の区分

語幹の最後の母音によって決まる。

| 条件 | 語尾 | 例 |
|------|------|----|
| 語幹の最後の母音が `ㅏ` または `ㅗ` | 아요 | 가`다` → 가`아요` → `가요` |
| それ以外 | 어요 | 먹`다` → 먹`어요` |
| `하다`動詞 | 해요（固定） | 공부하`다` → 공부`해요` |

> `가아요`のように母音が重なる場合は縮約して`가요`になる。これは자연な発音のため。

---

## 3. よく使う動詞一覧

### 이동（移動）

| 原型 | 해요체 | 意味 |
|------|--------|------|
| 가다 | 가요 | 行きます |
| 오다 | 와요 | 来ます |
| 나가다 | 나가요 | 出かけます / 出ます |
| 들어오다 | 들어와요 | 入ってきます |

### 음식（食べ物・飲み物）

| 原型 | 해요체 | 意味 |
|------|--------|------|
| 먹다 | 먹어요 | 食べます |
| 마시다 | 마셔요 | 飲みます |
| 만들다 | 만들어요 | 作ります |

### 일상（日常）

| 原型 | 해요체 | 意味 |
|------|--------|------|
| 자다 | 자요 | 寝ます |
| 일어나다 | 일어나요 | 起きます |
| 씻다 | 씻어요 | 洗います / 洗顔します |
| 입다 | 입어요 | 着ます |
| 보다 | 봐요 | 見ます |
| 읽다 | 읽어요 | 読みます |
| 듣다 | 들어요 | 聞きます |

### 하다 動詞（よく使うもの）

| 원형 | 해요체 | 意味 |
|------|--------|------|
| 공부하다 | 공부해요 | 勉強します |
| 일하다 | 일해요 | 働きます |
| 요리하다 | 요리해요 | 料理します |
| 운동하다 | 운동해요 | 運動します |
| 전화하다 | 전화해요 | 電話します |

---

## 4. 을/를（〜を）

動詞の目的語につく助詞。

| 条件 | 助詞 | 例 |
|------|------|----|
| 前の名詞が母音で終わる | 를 | 커피`를` 마셔요. |
| 前の名詞が子音で終わる | 을 | 밥`을` 먹어요. |

```
커피를 마셔요.     → コーヒーを飲みます。
밥을 먹어요.       → ご飯を食べます。
책을 읽어요.       → 本を読みます。
음악을 들어요.     → 音楽を聞きます。
드라마를 봐요.     → ドラマを見ます。
```

---

## 5. 기본 문장 패턴（基本の文型）

```
[주어(主語)] + [목적어(目的語)] + [동사(動詞)]
```

韓国語は日本語と同じく**動詞が文末に来る**。

```
저는 밥을 먹어요.              → 私はご飯を食べます。
저는 커피를 마셔요.            → 私はコーヒーを飲みます。
저는 지금 공부해요.            → 私は今勉強しています。
저는 회사에 가요.              → 私は会社に行きます。
남편은 지금 자요.              → 夫は今寝ています。
```

> `지금`（今）のような副詞は動詞の前に置く。

---

## 6. 질문과 대답（質問と答え）

### 뭐 해요?（何をしていますか？）

```
뭐 해요?                       → 何をしていますか？ / 何をしますか？
  → 공부해요.                  → 勉強しています。
  → 밥 먹어요.                 → ご飯を食べています。
  → 그냥 쉬어요.               → ただ休んでいます。
```

> `밥 먹어요`のように을/를を省略することも多い。会話ではよく省略される。

### 어디 가요?（どこへ行きますか？）

```
어디 가요?                     → どこへ行きますか？
  → 학교에 가요.               → 学校に行きます。
  → 슈퍼에 가요.               → スーパーに行きます。
  → 그냥 집에 있어요.          → 特に、家にいます。
```

## 練習

以下の단어を使って文章を作ってみましょう。

```
아침（朝）, 저녁（夜）, 주말（週末）
밥（ご飯）, 커피（コーヒー）, 물（水）, 영화（映画）, 음악（音楽）
헬스장（ジム）, 카페（カフェ）, 도서관（図書館）
```

**例**
- 저는 아침에 커피를 마셔요.（私は朝にコーヒーを飲みます。）
- 주말에 영화를 봐요.（週末に映画を見ます。）

## Tips

### 現在・進行・未来を同じ形で表す

韓国語の`아요/어요`は文脈によって**現在・進行・未来**を表す。

| 文 | 意味（文脈による） |
|----|----------------|
| 밥을 먹어요. | ご飯を食べます（習慣）/ 食べています（進行） |
| 내일 가요. | 明日行きます（未来） |

日本語のように形が変わらないので、むしろシンプル。

### 助詞まとめ（강의 1〜3）

| 意味 | 母音終わり | 子音終わり |
|------|-----------|-----------|
| は | 는 | 은 |
| が | 가 | 이 |
| を | 를 | 을 |
| に（場所） | 에 | 에 |
