# 수업 21. 이거 다른 걸로 바꿔 주세요

商品の交換や返品など、ショッピングでトラブルがあったときに使う表現を学ぶ。また、行動の順序を表す文法も一緒に覚える。

## 1. 교환 및 환불 어휘（交換・払い戻しの語彙）

| 韓国語 | 日本語 | 例文 |
| ---- | ---- | ---- |
| 교환 | 交換 | 다른 색으로 `교환`하고 싶어요. |
| 환불 | 払い戻し | `환불`해 주세요. |
| 영수증 | レシート・領収書 | `영수증`이 있어요. |
| 점원 | 店員 | `점원`이 친절해요. |
| 거스름돈 | お釣り | `거스름돈` 여기 있습니다. |
| 바꾸다 | 換える | 이걸로 `바꿔 주세요`. |

---

## 2. ~아/어 주다（〜してあげる / 〜してもらう）

動詞の後ろについて、他人のためにある行動をする（依頼・お願い）ときに使う。

- 母音が `ㅏ, ㅗ` の場合 → **~아 주다**（例: 사다 → 사 주다）
- それ以外の母音 → **~어 주다**（例: 바꾸다 → 바꿔 주다, 빌리다 → 빌려 주다）
- `하다` の場合 → **~해 주다**（例: 환불하다 → 환불해 주다）

お願いをするときは、主に **`~아/어 주세요`** またはより丁寧な **`~아/어 주시겠어요?`** を使う。

- 이거 다른 걸로 `바꿔 주세요`.（これ、他のものに換えてください。）
- 사이즈를 `확인해 주시겠어요?`（サイズを確認していただけますか？）
- 영수증을 `보여 주세요`.（レシートを見せてください。）

---

## 3. ~기 전에 / ~(으)ㄴ 후에（〜する前に / 〜した後に）

行動の順序を表す表現。

### 1) ~기 전에（〜する前に）

動詞の語幹の後ろに `~기 전에` をつける。（パッチムの有無は関係ない）

- 가다 → `가기 전에`（行く前に）
- 먹다 → `먹기 전에`（食べる前に）

- 사기 전에 입어 보세요.（買う前に着てみてください。）
- 집에 가기 전에 영수증을 받았어요.（家に帰る前にレシートをもらいました。）

### 2) ~(으)ㄴ 후에（〜した後に）

動詞の語幹にパッチムがない、または `ㄹ` パッチムのときは `~ㄴ 후에`、パッチムがあるときは `~은 후에` を使う。

- 사다 → `산 후에`（買った後に）
- 먹다 → `먹은 후에`（食べた後に）
- 만들다 → `만든 후에`（作った後に ※ㄹ脱落）

- 밥을 먹은 후에 약을 드세요.（ご飯を食べた後に薬を飲んでください。）
- 물건을 산 후에 봉투에 넣었어요.（商品を買った後に袋に入れました。）

---

## 4. 회화 연습

**상황：가게에서 물건 교환하기**（お店での商品の交換）

- 아내: 저기요, 어제 이 셔츠를 샀는데 사이즈가 좀 작아요.（すみません、昨日このシャツを買ったんですが、サイズが少し小さいです。）
- 점원: 아, 그러세요? 영수증 가지고 오셨어요?（あ、そうですか？ レシートは持って来られましたか？）
- 아내: 네, 여기 있어요. 다른 사이즈로 교환해 주세요.（はい、ここにあります。他のサイズに交換してください。）
- 점원: 네, 어떤 사이즈로 바꿔 드릴까요? 한 단계 큰 사이즈로 드릴까요?（はい、どのサイズに換えましょうか？ ワンサイズ大きいものにしましょうか？）
- 아내: 네, 그걸로 보여 주세요. 입어 본 후에 결정할게요.（はい、それを見せてください。着てみた後に決めます。）
- 점원: 네, 여기 있습니다. 피팅룸은 저기 왼쪽에 있어요.（はい、こちらにございます。試着室はあちらの左側にあります。）

## 練習

空欄に合う表現を入れて文章を完成させてみましょう。

```text
1. 한국에 ________ (가다) 전에 한국어를 공부해요.
2. 영화를 ________ (보다) 후에 저녁을 먹을 거예요.
3. 너무 매워요. 물 좀 ________ (주다).
4. 사이즈가 안 맞아요. 이걸로 ________ (바꾸다 + 주다).
```

**정답**

1. 가기
2. 본
3. 주세요
4. 바꿔 주세요

## Tips

### 교환(交換)과 환불(払い戻し)의 조건

韓国では通常、商品を買った日から **7日以内** にレシートを持って行けば交換や払い戻しが可能。
タグ（Tag）を外したり、商品を使用した後は交換や払い戻しが難しくなるので注意！
- **태그를 떼기 전에 입어 보세요.**（タグを取る前に着てみてください。）
