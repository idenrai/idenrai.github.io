# 수업 18. 이거 얼마예요?

今回は**쇼핑 실전 표현**（ショッピングの実践表現）を学ぶ。  
韓国のショッピングモール、東大門、明洞など — 値段の聞き方から試着・値引き交渉まで完全マスター。

---

## 1. 가격 묻기 (値段を聞く)

```text
이거 얼마예요?           → これいくらですか？
저거 얼마예요?           → あれいくらですか？
합계가 얼마예요?         → 合計いくらですか？
좀 깎아 주세요.          → 少し負けてください。
너무 비싸요.             → 高すぎます。
좀 싸게 해 주세요.       → 少し安くしてください。
```

### 숫자 (数字) — 韓国語の値段は漢数詞

| 숫자 | 한국어 |
|------|--------|
| 100 | 백 |
| 1,000 | 천 |
| 10,000 | 만 |
| 100,000 | 십만 |
| 1,000,000 | 백만 |

```text
오천 원이에요.           → 5,000ウォンです。
이만 원이에요.           → 20,000ウォンです。
삼만오천 원이에요.       → 35,000ウォンです。
```

---

## 2. 쇼핑할 때 자주 쓰는 표현 (よく使う表現)

```text
입어 봐도 돼요?          → 試着してもいいですか？
신어 봐도 돼요?          → 試し履きしてもいいですか？
다른 색 있어요?          → 他の色はありますか？
다른 사이즈 있어요?      → 他のサイズはありますか？
이거 주세요.             → これをください。
카드 돼요?               → カード使えますか？
영수증 주세요.           → 領収書ください。
```

### 색깔 (色)

| 한국어 | 일본어 |
|--------|--------|
| 빨간색 | 赤 |
| 파란색 | 青 |
| 노란색 | 黄色 |
| 초록색 | 緑 |
| 흰색 | 白 |
| 검은색 | 黒 |
| 분홍색 | ピンク |
| 베이지색 | ベージュ |

### 사이즈 표현

```text
작아요.      → 小さいです。（小さすぎます。）
커요.        → 大きいです。（大きすぎます。）
딱 맞아요.   → ぴったりです。
조금 작은데 큰 사이즈 있어요?  → 少し小さいんですが、大きいサイズはありますか？
```

---

## 3. ~아/어 보다 (〜してみる)

試してみる、やってみるという表現。수업 15에서 잠깐 등장했던 표현！

```text
동사 어간 + 아/어 봐요 (현재 권유)
동사 어간 + 아/어 봤어요 (과거)
```

```text
입어 봐요.              → 着てみてください。
먹어 봐요.              → 食べてみてください。
한번 해 봐요!           → 一度やってみてください！
이 식당 가 봤어요?      → このお店行ってみましたか？
한국어 공부해 봤어요.   → 韓国語を勉強してみました。
```

## 4. 회화 연습

**상황：명동 쇼핑**（明洞でのショッピング）

```text
아내: 이 가방 얼마예요?
     （このバッグいくらですか？）

직원: 오만 원이에요.
     （50,000ウォンです。）

아내: 좀 비싸네요. 깎아 주세요.
     （ちょっと高いですね。負けてください。）

직원: 사만오천 원에 드릴게요.
     （45,000ウォンにしますよ。）

아내: 다른 색도 있어요?
     （他の色もありますか？）

직원: 네, 빨간색이랑 검은색 있어요.
     （はい、赤と黒があります。）

아내: 검은색으로 주세요. 카드 돼요?
     （黒いのをください。カード使えますか？）

직원: 네, 됩니다!
     （はい、使えます！）
```

## Tips

### 한국 쇼핑 문화 tip

- **정가제 매장** (百貨店・大型チェーン) では値引き交渉不可。  
- **시장·노점** (市場・屋台) では `깎아 주세요` が効果的。  
- 明洞の路面店では「お得なセット提案」が多い — `세트로 사면 싸요` (セットで買うと安いよ)。

### 편리한 표현 정리 (便利な表現まとめ)

| 표현 | 의미 |
|------|------|
| 구경 좀 할게요. | ちょっと見るだけです。 |
| 생각해 볼게요. | 考えてみます。 |
| 다음에 올게요. | また来ます。 |
| 이거 포장해 주세요. | これを包んでください。 |

### ~에 드릴게요 (〜でお渡しします)

値段を提示するときの丁寧な表現。

```text
사만 원에 드릴게요.      → 40,000ウォンにしますよ。
```
