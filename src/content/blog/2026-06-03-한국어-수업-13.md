# 수업 13. 뭐 할 거예요?

今回は**未来・予定・意図**を表す表現を学ぶ。  
「〜するつもりです」「〜しようと思っています」など、これを覚えると会話の時制が現在・過去・未来すべてカバーできる。

---

## 1. ~ㄹ/을 거예요（〜するつもりです / 〜でしょう）

未来の予定や意志を表す最も基本的な表現。

```text
동사/형용사 어간 + (으)ㄹ 거예요
```

- 語幹が母音またはㄹで終わる → `ㄹ 거예요`
- 語幹が子音で終わる → `을 거예요`

| 原型 | ~ㄹ/을 거예요 | 意味 |
|------|--------------|------|
| 가다 | 갈 거예요 | 行くつもりです |
| 오다 | 올 거예요 | 来るつもりです |
| 먹다 | 먹을 거예요 | 食べるつもりです |
| 마시다 | 마실 거예요 | 飲むつもりです |
| 자다 | 잘 거예요 | 寝るつもりです |
| 공부하다 | 공부할 거예요 | 勉強するつもりです |
| 만나다 | 만날 거예요 | 会うつもりです |

```text
내일 한국에 갈 거예요.      → 明日韓国に行くつもりです。
주말에 집에 있을 거예요.    → 週末は家にいるつもりです。
뭐 먹을 거예요?             → 何を食べるつもりですか？
```

---

## 2. ~(으)려고 해요（〜しようと思っています）

まだ決定していないが、そうしようと考えている意図を表す。  
`~ㄹ 거예요`より少し「検討中」のニュアンスが強い。

```text
동사 어간 + (으)려고 해요
```

- 語幹が母音またはㄹで終わる → `려고 해요`
- 語幹が子音で終わる → `으려고 해요`

```text
내년에 한국에 가려고 해요.  → 来年韓国に行こうと思っています。
한국어를 배우려고 해요.     → 韓国語を習おうと思っています。
다이어트를 하려고 해요.     → ダイエットをしようと思っています。
뭐 먹으려고 해요?           → 何を食べようと思っていますか？
```

---

## 3. ~ㄹ 거예요 vs ~(으)려고 해요

| | ~ㄹ 거예요 | ~(으)려고 해요 |
|--|-----------|--------------|
| ニュアンス | 予定・決定済み | 意図・検討中 |
| 確度 | 高い | やや低い |
| 例 | 내일 갈 거예요. | 내일 가려고 해요. |

```text
내일 병원에 갈 거예요.      → 明日病院に行くつもりです。（予定）
내일 병원에 가려고 해요.    → 明日病院に行こうと思っています。（検討中）
```

---

## 4. 예정 관련 표현（予定関連の表現）

### 언제 ~ㄹ 거예요?（いつ〜するつもりですか？）

```text
언제 한국에 갈 거예요?      → いつ韓国に行くつもりですか？
  → 7월에 갈 거예요.        → 7月に行くつもりです。
  → 아직 모르겠어요.        → まだわかりません。

언제 결혼할 거예요?         → いつ結婚するつもりですか？
  → 이미 결혼했어요!        → もう結婚しました！（笑）
```

### 계획이 있어요?（予定はありますか？）

```text
이번 주말에 계획이 있어요?  → 今週末予定はありますか？
  → 네, 가족이랑 나갈 거예요.   → はい、家族と出かけるつもりです。
  → 아니요, 특별한 계획 없어요. → いいえ、特に予定はありません。
```

---

## 5. 부정형（否定形）

```text
안 갈 거예요.               → 行かないつもりです。
먹지 않을 거예요.           → 食べないつもりです。
가려고 하지 않아요.         → 行こうとは思っていません。
```

## 練習

다음 문장을 한국어로 만들어 보세요.（以下の文を韓国語で作ってみましょう。）

```text
1. 週末に何をするつもりですか？
2. 友達に会うつもりです。
3. 来月引っ越ししようと思っています。
4. 今日は早く寝るつもりです。
5. まだ決めていません。
```

**정답**

```text
1. 주말에 뭐 할 거예요?
2. 친구를 만날 거예요.
3. 다음 달에 이사하려고 해요.
4. 오늘은 일찍 잘 거예요.
5. 아직 안 정했어요. / 아직 모르겠어요.
```

## Tips

### 시제 총정리（時制まとめ）

수업 7, 9, 10, 13을 통해 시제가 완성되었다。

| 時制 | 形 | 例 |
|------|----|----|
| 현재 | ~아요/어요 | 먹어요 |
| 과거 | ~았어요/었어요 | 먹었어요 |
| 미래/예정 | ~ㄹ/을 거예요 | 먹을 거예요 |
| 희망 | ~고 싶어요 | 먹고 싶어요 |
| 의도 | ~(으)려고 해요 | 먹으려고 해요 |

### 구어체 줄임

`~ㄹ/을 거예요`は口語でよく縮約される。

```text
갈 거예요 → 갈 거야（タメ口）
먹을 거예요 → 먹을 거야（タメ口）
```
