# 수업 17. 더워서 창문을 열었어요

韓国語の不規則活用（불규칙 활용）を学ぶ。
日常でよく使う基本的な単語に不規則変化が多いので、代表的なものを暗記しよう。

## 1. ㅂ 불규칙（ㅂ不規則）

語幹の最後が `ㅂ` で終わる形容詞（一部の動詞）は、母音と出会うと `ㅂ` が脱落し、代わりに `우`（または `오`）が追加される。

- `ㅂ` → 脱落 + `우` + `어요` = `워요`
- 돕다（助ける）など一部は `와요`

| 原型 | 해요체（現在） | 過去形 | 意味 |
| ---- | ---- | ---- | ---- |
| 덥다 | 더워요 | 더웠어요 | 暑い |
| 춥다 | 추워요 | 추웠어요 | 寒い |
| 맵다 | 매워요 | 매웠어요 | 辛い |
| 무겁다 | 무거워요 | 무거웠어요 | 重い |
| 가볍다 | 가벼워요 | 가벼웠어요 | 軽い |
| 어렵다 | 어려워요 | 어려웠어요 | 難しい |
| 쉽다 | 쉬워요 | 쉬웠어요 | 簡単だ |

- 오늘은 날씨가 `더워요`.（今日は天気が暑いです。）
- 이 책은 너무 `어려워요`.（この本はとても難しいです。）

> **例外**：`입다`（着る）、`좁다`（狭い）、`잡다`（掴む）などは規則通り変化する。
> ex) 입다 → 입어요

---

## 2. ㄷ 불규칙（ㄷ不規則）

語幹の最後が `ㄷ` で終わる動詞は、母音と出会うと `ㄷ` が `ㄹ` に変わる。

- `ㄷ` → `ㄹ` + `아요/어요` = `라요/러요`

| 原型 | 해요체（現在） | 過去形 | 意味 |
| ---- | ---- | ---- | ---- |
| 듣다 | 들어요 | 들었어요 | 聞く |
| 걷다 | 걸어요 | 걸었어요 | 歩く |
| 묻다 | 물어요 | 물었어요 | 尋ねる |

- 매일 한국 음악을 `들어요`.（毎日韓国の音楽を聴きます。）
- 역까지 `걸어서` 가요.（駅まで歩いて行きます。）
- 모르면 선생님한테 `물어보세요`.（わからなければ先生に聞いてみてください。）

> **例外**：`받다`（もらう）、`닫다`（閉める）、`믿다`（信じる）などは規則通り変化する。
> ex) 닫다 → 닫아요（閉めます）

---

## 3. 르 불규칙（르不規則）

語幹の最後が `르` で終わる動詞・形容詞は、母音と出会うと `르` が脱落し、`ㄹ` が一つ増える。

- 直前の母音が `ㅏ/ㅗ` → `ㄹ라요`
- 直前の母音がそれ以外 → `ㄹ러요`

| 原型 | 해요체（現在） | 過去形 | 意味 |
| ---- | ---- | ---- | ---- |
| 빠르다 | 빨라요 | 빨랐어요 | 速い |
| 다르다 | 달라요 | 달랐어요 | 違う |
| 모르다 | 몰라요 | 몰랐어요 | わからない |
| 부르다 | 불러요 | 불렀어요 | 呼ぶ / 歌う |

- KTX는 아주 `빨라요`.（KTXはとても速いです。）
- 노래를 `불렀어요`.（歌を歌いました。）
- 저는 `몰라요`.（私はわかりません。）

---

## 4. ㅅ 불규칙（ㅅ不規則）

語幹の最後が `ㅅ` で終わる動詞は、母音と出会うと `ㅅ` が脱落する。
ただし、後ろの形（아요/어요など）はそのまま残る。

| 原型 | 해요체（現在） | 過去形 | 意味 |
| ---- | ---- | ---- | ---- |
| 낫다 | 나아요 | 나았어요 | 治る / よりマシだ |
| 짓다 | 지어요 | 지었어요 | 建てる / 作る |
| 붓다 | 부어요 | 부었어요 | 腫れる / 注ぐ |

- 감기가 `나았어요`.（風邪が治りました。）
- 아침에 얼굴이 `부어요`.（朝、顔が腫れます。）

> **例外**：`웃다`（笑う）、`씻다`（洗う）、`벗다`（脱ぐ）などは規則通り変化する。
> ex) 손을 씻어요（手を洗います）

## 5. 회화 연습

**상황：남편이 창문을 여는 상황**（夫が窓を開ける状況）

- 남편: 오늘 날씨가 좀 더워요.（今日は少し天気が暑いですね。）
- 아내: 네, 덥네요. 창문을 열까요?（はい、暑いですね。窓を開けましょうか？）
- 남편: 제가 열게요. 어제 밖에서 많이 걸었어요?（私が開けます。昨日外でたくさん歩きましたか？）
- 아내: 네, 걸어서 다리가 부었어요.（はい、歩いたので足が腫れました。）
- 남편: 따뜻한 물로 씻고 푹 쉬세요.（温かいお湯で洗ってゆっくり休んでください。）

## 練習

原型を해요체に変えてみましょう。

```text
1. 춥다
2. 닫다
3. 듣다
4. 모르다
5. 웃다
```

**정답**

1. 추워요 (ㅂ不規則)
2. 닫아요 (規則)
3. 들어요 (ㄷ不規則)
4. 몰라요 (르不規則)
5. 웃어요 (規則)

## Tips

### 헷갈리는 규칙 vs 불규칙（まぎらわしい規則と不規則）

同じような形でも不規則と規則が混ざっている。よく使うものをペアで覚えると良い。

- 걷다（歩く）→ `걸어요` 🆚 받다（もらう）→ `받아요`
- 덥다（暑い）→ `더워요` 🆚 입다（着る）→ `입어요`
- 낫다（治る）→ `나아요` 🆚 씻다（洗う）→ `씻어요`
