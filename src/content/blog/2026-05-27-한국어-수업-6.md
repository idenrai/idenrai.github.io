# 수업 6. 지금 몇 시예요?

수업 5で学んだ数字を使って、時刻・日付・曜日を表現する。

## 1. 시간（時刻）

### 몇 시예요?（何時ですか？）

- `지금 몇 시예요?`（今何時ですか？）

**시（時）** は고유어 숫자、**분（分）** は한자어 숫자を使う。

| 時刻 | 韓国語 |
| ---- | ---- |
| 1時 | 한 시 |
| 2時 | 두 시 |
| 3時30分 | 세 시 삼십 분 |
| 6時15分 | 여섯 시 십오 분 |
| 12時 | 열두 시 |

> `반`（半）を使うと便利。`두 시 반` = 2時半

### 오전 / 오후（午前 / 午後）

- `오전 열 시예요.`（午前10時です。）
- `오후 두 시 반이에요.`（午後2時半です。）

### 몇 시에（〜時に）

時刻に`에`をつけると「〜時に」になる。

- 몇 시에 일어나요?（何時に起きますか？）
  - 아침 `일곱 시에` 일어나요.（朝`7時に`起きます。）
- 몇 시에 자요?（何時に寝ますか？）
  - 밤 `열한 시에` 자요.（夜`11時に`寝ます。）

---

## 2. 날짜（日付）

### 오늘 며칠이에요?（今日は何日ですか？）

**월（月）** も **일（日）** も한자어 숫자を使う。

- `오늘 며칠이에요?`（今日は何日ですか？）
  - `오월 이십칠일이에요.`（5月27日です。）

### 月の言い方

| 月 | 韓国語 |
| ---- | ---- |
| 1月 | 일월 |
| 2月 | 이월 |
| 3月 | 삼월 |
| 4月 | 사월 |
| 5月 | 오월 |
| 6月 | **유월** ※ |
| 7月 | 칠월 |
| 8月 | 팔월 |
| 9月 | 구월 |
| 10月 | **시월** ※ |
| 11月 | 십일월 |
| 12月 | 십이월 |

> 6月は`육월`でなく**`유월`**、10月は`십월`でなく**`시월`**。発音しやすくするための例外。

---

## 3. 요일（曜日）

| 曜日 | 韓国語 |
| ---- | ---- |
| 月曜日 | 월요일 |
| 火曜日 | 화요일 |
| 水曜日 | 수요일 |
| 木曜日 | 목요일 |
| 金曜日 | 금요일 |
| 土曜日 | 토요일 |
| 日曜日 | 일요일 |

- `오늘 무슨 요일이에요?`（今日は何曜日ですか？）
  - `수요일이에요.`（水曜日です。）
- `무슨 요일에 쉬어요?`（何曜日に休みますか？）
  - 토요일`이랑` 일요일에 쉬어요.（土曜日`と`日曜日に休みます。）

## 4. 날짜 관련 단어（日付関連の単語）

| 韓国語 | 意味 |
| ---- | ---- |
| 어제 | 昨日 |
| 오늘 | 今日 |

| 다음 주 | 来週 |
| 지난달 | 先月 |
| 이번 달 | 今月 |
| 다음 달 | 来月 |

### 시간 관련（時間帯）

| 韓国語 | 意味 |
| ---- | ---- |
| 아침 | 朝 |
| 점심 | 昼 |
| 저녁 | 夕方・夜 |
| 밤 | 夜（深夜寄り） |
| 새벽 | 夜明け・深夜 |

## 5. 빈도 표현（頻度の表現）

| 韓国語 | 意味 | 例文 |
| ---- | ---- | ---- |
| 항상 / 늘 | いつも | |
| 매일 | 毎日 | 매일 운동해요.（毎日運動します。） |
| 자주 | よく・頻繁に | 자주 영화를 봐요.（よく映画を見ます。） |
| 가끔 | たまに | 가끔 커피를 마셔요.（たまにコーヒーを飲みます。） |
| 별로 | あまり（〜ない） | 별로 안 먹어요.（あまり食べません。） |
| 전혀 | 全然（〜ない） | 전혀 안 가요.（全然行きません。） |
