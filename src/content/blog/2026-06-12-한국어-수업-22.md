# 수업 22. 우리 몇 시에 만날까요?

오늘은 친구나 가족과 약속을 정할 때 쓰는 **약속과 제안 표현**을 배운다.
"몇 시에 만날까요?", "어디에서 만날까요?" 등 상대방의 의견을 묻고 제안하는 법을 알아보자!

---

## 1. 약속 관련 어휘 (約束の語彙)

| 한국어 | 일본어 | 예문 |
|--------|--------|------|
| 약속 | 약속 / 予定 | 오늘 저녁에 약속이 있어요. |
| 시간 | 시간 / 都合 | 약속 시간이 몇 시예요? |
| 장소 | 場所 | 만날 장소를 정해요. |
| 늦다 | 遅れる | 약속에 늦지 마세요. |
| 기다리다 | 待つ | 카페에서 친구를 기다려요. |
| 만나다 | 会う | 내일 2시에 만나요. |

---

## 2. ~(으)ㄹ까요? (〜합시다/〜할까요? - 提案・意見を求める)

상대방에게 어떤 일을 같이 하자고 제안하거나, 상대방의 의사를 물어볼 때 사용한다.

- 받침이 없을 때: **~ㄹ까요?** (예: 만나다 → 만날까요?)
- 받침이 있을 때: **~을까요?** (예: 먹다 → 먹을까요?)
- 'ㄹ' 받침으로 끝날 때: **~ㄹ까요?** (예: 만들다 → 만들까요?)

```text
우리 몇 시에 만날까요?
  → 私たち、何時に会いましょうか？

오늘 저녁에 같이 삼겹살 먹을까요?
  → 오늘 저녁에 같이 삼겹살 먹을까요?

커피 한잔 마실까요?
  → コーヒーを一杯飲みましょうか？
```

---

## 3. ~(으)ㅂ시다 (〜합시다 - 勧유・提案への同意)

대화 상대방에게 어떤 행동을 함께 할 것을 요청하거나 권유할 때 사용한다. 주로 격식 있는 상황이나 공손하게 제안을 받아들일 때 쓴다.

- 받침이 없을 때: **~ㅂ시다** (예: 가다 → 갑시다)
- 받침이 있을 때: **~읍시다** (예: 읽다 → 읽읍시다)
- 'ㄹ' 받침으로 끝날 때: **~ㅂ시다** (예: 만들다 → 만듭시다 - 'ㄹ' 탈락)

※ 나보다 나이가 많거나 지위가 높은 사람에게는 잘 쓰지 않고, 동료나 아랫사람에게 주로 쓴다. (높은 사람에게는 "~아/어 주시겠어요?" 또는 "~(으)시지요"를 쓴다.)

```text
A: 어디에서 만날까요?
B: 서울역 앞 카페에서 만납시다. (→ 서울역 앞 카페에서 만나요가 더 자연스러운 대화체이기도 함)

A: 같이 갈까요?
B: 네, 같이 갑시다.
```

---

## 4. ~기 때문에 (〜だから / 〜のために - 理由)

어떤 행동이나 상태의 확실한 원인이나 이유를 나타낸다. `~아서/어서`보다 더 명확하고 격식 있는 이유 표현이다.

- 동사/형용사 어간 뒤에 받침과 상관없이 **~기 때문에**를 붙인다. (예: 바쁘다 → 바쁘기 때문에, 비가 오다 → 비가 오기 때문에)
- 명사 뒤에는 **~때문에**를 붙인다. (예: 비 → 비 때문에, 감기 → 감기 때문에)

```text
약속이 있기 때문에 먼저 가야 해요.
  → 予定があるため、先に行かなければなりません。

비가 오기 때문에 날씨가 추워요.
  → 雨が降っているため天気が寒いです。

눈 때문에 길이 미끄러워요.
  → 雪のせいで道が滑りやすいです。
```

## 5. 회화 연습

**상황: 주말 약속 정하기 (週末の約束決め)**

```text
남편: 이번 주 토요일에 우리 영화 보러 갈까요?
     (今週의 토요일에 우리 영화 보러 갈까요?)

아내: 좋아요! 어떤 영화를 볼까요?
     (いいですね！どんな映画を見ましょうか？)

남편: 재미있는 한국 영화가 있어요. 같이 봐요.
     (面白い韓国映画があります。一緒に見ましょう。)

아내: 네, 몇 시에 만날까요?
     (はい、何時に会いましょうか？)

남편: 영화가 2시에 시작하기 때문에 1시 반에 만납시다.
     (映画が2時に始まるので、1시 반에 만나자.)

아내: 어디에서 만날까요? 극장 앞 카페가 어때요?
     (どこで会いましょうか？映画館の前のかふぇはどうですか？)

남편: 좋아요. 거기서 기다릴게요. 늦지 마세요!
     (いいね。そこで待ってるね。遅れないでね！)
```

## 練習

**문장의 의미가 통하도록 괄호 안의 단어를 올바른 형태로 바꾸어 쓰세요. (文章の意味が通るようにカッコの中の単語を正しい形に書き換えてください。)**

1. 주말에 우리 같이 쇼핑 ________? (하다 - 제안)
2. 비가 ________ 때문에 오늘은 집에서 쉴래요. (오다 - 이유)
3. 2층 식당에서 점심을 ________. (먹다 - 권유/동의)
4. 차가 막히기 ________ 약속에 늦었어요. (때문)

**정답 (答え)**
1. 할까요
2. 오기
3. 먹읍시다 (또는 먹어요)
4. 때문에 (막히기 때문에)

## Tips

### 약속을 거절하거나 변경할 때
상대방이 제안했을 때 사정이 있어서 약속을 못 잡는 경우, 다음과 같이 완곡하게 거절하거나 미안함을 표현한다.
- **미안해요. 그날은 선약이 있어요.** (すみません。その日は先約があります。)
- **제가 요새 좀 바쁘기 때문에 다음 주에 만나요.** (私が最近ちょっと忙しいので、来週会いましょう。)
