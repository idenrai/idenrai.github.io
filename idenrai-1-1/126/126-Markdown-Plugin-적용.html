그간 티스토리에 쓰는 글과는 별도로

Qiita나 Github, Jekyll Github Page를 파서 조금씩 글을 써 왔다.


티스토리야 원래 이용하던 블로그고,

Qiita에는 회사에서 연구발표를 하게 될 때마다 프레젠테이션용으로 적어 왔다.

그리고 Github Page의 경우는 이 세곳의 내용을 최종적으로 전부 옮겨두려는 생각으로 개설했다.


다만 티스토리의 글을 다른 곳으로 옮기기엔

마크다운으로 다시 작성해야 한다는 것이 너무나도 귀찮아서...

미루고 또 미루다 보니 아직까지 손도 안대고 있었다.


오늘 친구가 티스토리 초대장을 받은 김에 블로그를 해 보겠다기에

할거면 처음부터 마크다운으로 글을 쓰라고 추천하면서

티스토리에 마크다운을 적용하는 법을 검색해 보니...


티스토리 유저인 MakerJ님이 개발한 티스토리용 마크다운 플러그인이 있더라.

(적용 방법은 이곳을 참조하도록 하자)

마크다운 문법이 조금 다를 수 있으니, 여기서 직접 타이핑하며 확인하도록 하자.


그래서 바로 적용하고, Qiita의 글을 복붙해서 포스팅해 보니, 완전 잘 돌아간다.

티스토리에 이렇게 마크다운이 쉽게 적용되니,

차후 블로그를 옮기거나 할 경우 조금은 편하게 작업할 수 있을 것 같다.


다만 치명적인 문제가 하나...

Qiita의 글은 일본어로 적어두었기에, 한글로 번역을 해야 한다...


결국 귀찮음의 굴레에서는 벗어날 수 없었다.
