# 수업 16. 지하철역이 어디예요?

今回は**길 묻기와 교통 표현**（道の聞き方・交通表現）を学ぶ。  
韓国旅行で必ず使う「〜はどこですか？」「〜まで何で行きますか？」の実践フレーズ。

---

## 1. 위치 묻기 (場所を尋ねる)

```text
〜이/가 어디예요?        → 〜はどこですか？
〜에 어떻게 가요?        → 〜にどうやって行きますか？
〜까지 얼마나 걸려요?    → 〜までどのくらいかかりますか？
여기서 멀어요?           → ここから遠いですか？
```

### 장소 단어 (場所の単語)

| 한국어 | 일본어 |
|--------|--------|
| 지하철역 | 地下鉄の駅 |
| 버스 정류장 | バス停 |
| 공항 | 空港 |
| 기차역 | 駅（電車） |
| 편의점 | コンビニ |
| 화장실 | トイレ |
| 출구 | 出口 |
| 입구 | 入口 |
| 근처 | 近く |

```text
지하철역이 어디예요?              → 地下鉄の駅はどこですか？
가장 가까운 편의점이 어디예요?    → 一番近いコンビニはどこですか？
공항까지 얼마나 걸려요?           → 空港までどのくらいかかりますか？
```

---

## 2. 방향·위치 표현 (方向・位置の表現)

```text
왼쪽으로 가세요.         → 左に行ってください。
오른쪽으로 가세요.       → 右に行ってください。
직진하세요.              → まっすぐ行ってください。
저 건물 앞에 있어요.     → あのビルの前にあります。
신호등 지나서 있어요.    → 信号を過ぎたところにあります。
```

| 한국어 | 일본어 |
|--------|--------|
| 왼쪽 | 左 |
| 오른쪽 | 右 |
| 앞 | 前 |
| 뒤 | 後ろ |
| 옆 | 横 |
| 위 | 上 |
| 아래 | 下 |
| 맞은편 | 向かい側 |

---

## 3. 교통수단 표현 (交通手段の表現)

```text
지하철을 타세요.         → 地下鉄に乗ってください。
버스로 가세요.           → バスで行ってください。
택시를 타세요.           → タクシーに乗ってください。
걸어서 가세요.           → 歩いて行ってください。
〜호선을 타세요.         → 〜号線に乗ってください。
〜에서 내리세요.         → 〜で降りてください。
〜에서 갈아타세요.       → 〜で乗り換えてください。
```

```text
2호선을 타세요.                   → 2号線に乗ってください。
시청역에서 내리세요.              → 市庁駅で降りてください。
종각역에서 1호선으로 갈아타세요.  → 鍾閣駅で1号線に乗り換えてください。
여기서 걸어서 10분이에요.         → ここから歩いて10分です。
```

## 4. 회화 연습

**상황：서울에서 길을 묻는 상황**（ソウルで道を聞く場面）

```text
아내: 저기요, 경복궁이 어디예요?
     （すみません、景福宮はどこですか？）

행인: 3호선 지하철을 타세요. 경복궁역에서 내리면 돼요.
     （3号線の地下鉄に乗ってください。景福宮駅で降りればいいです。）

아내: 여기서 가까워요?
     （ここから近いですか？）

행인: 걸어가기엔 좀 멀어요. 지하철로 5분 정도예요.
     （歩くには少し遠いです。地下鉄で5分くらいです。）

아내: 지하철역이 어디예요?
     （地下鉄の駅はどこですか？）

행인: 저 신호등 지나서 왼쪽으로 가면 있어요.
     （あの信号を過ぎて左に行くとあります。）

아내: 감사합니다!
     （ありがとうございます！）
```

## 練習

**다음 질문에 한국어로 대답해 보세요。（次の質問に韓国語で答えてみましょう。）**

1. 지하철역을 찾고 있다. 어떻게 물어볼까요？
2. 지하철 2호선을 타서 강남역에서 내리라고 안내해 보세요。
3. 화장실이 왼쪽에 있다고 알려주세요。
4. 여기서 걸어서 3분이라고 말해 보세요。

**정답 (答え)**

```text
1. 지하철역이 어디예요?
2. 2호선을 타세요. 강남역에서 내리세요.
3. 화장실은 왼쪽에 있어요.
4. 여기서 걸어서 3분이에요.
```

## Tips

### ~(으)로 가면 돼요 （〜に行けばいいです）

「〜すればいい」という自然なアドバイス表現。

```text
오른쪽으로 가면 돼요.       → 右に行けばいいです。
이 버스를 타면 돼요.         → このバスに乗ればいいです。
2호선으로 갈아타면 돼요.     → 2号線に乗り換えればいいです。
```

### 한국 교통 앱 tip

韓国旅行では **카카오맵 (Kakao Maps)** または **네이버 지도 (Naver Maps)** が使いやすい。  
日本語表示にも対応しており、地下鉄の乗り換え案内も詳しく出る。

### 긴급 표현 (緊急表現)

```text
길을 잃어버렸어요.          → 道に迷いました。
도와주세요.                 → 助けてください。
여기가 어디예요?            → ここはどこですか？
```
