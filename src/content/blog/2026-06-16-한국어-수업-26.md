# 수업 26. 비가 오면 집에서 쉴 거예요

仮定や条件を表す **~(으)면** と、ある出来事が起きる時間を表す **~(으)ㄹ 때** を学ぶ。

## 1. 조건/날씨 관련 어휘（条件・天気の語彙）

| 韓国語 | 日本語 | 例文 |
| ---- | ---- | ---- |
| 만약 | もしも | `만약` 내일 비가 오면 약속을 취소해요. |
| 날씨 | 天気 | 오늘 `날씨`가 참 맑네요. |
| 비 / 눈 | 雨 / 雪 | 하늘에서 `눈`이 내려요. |
| 바람 | 風 | `바람`이 강하게 불어요. |
| 기분 | 気分 / 気持ち | `기분`이 좋을 때 노래를 불러요. |

---

## 2. ~(으)면（〜すれば / 〜したら - 仮定・条件）

ある状況を仮定したり、前の文の行動が後ろの文の条件になる時に使う。（日本語の `〜ば`、`〜たら`、`〜と` に対応する）

- パッチムがないとき: **~면**（例: 가다 → 가면）
- パッチムがあるとき: **~으면**（例: 먹다 → 먹으면）
- `ㄹ` パッチムのとき: **~면**（例: 살다 → 살면 ※ㄹ脱落なし）

- 돈이 있`으면` 가방을 살 거예요.（お金があればバッグを買います。）
- 질문이 있`으면` 언제든지 말해 주세요.（質問があればいつでも言ってください。）
- 날씨가 좋`으면` 산책합시다.（天気が良ければ散歩しましょう。）

> **만약（もしも）と一緒に使う**
> 仮定の意味をより明確にしたいとき、文の頭に `만약` または `만일` をつける。
> - `만약` 내일 눈이 오면 좋겠어요.（もし明日雪が降ったらいいですね。）

---

## 3. ~(으)ㄹ 때（〜の時 - 時間）

ある行動や状態が持続する時点や、時間的な状況を表す時に使う。（名詞の後ろには直接 **때** をつける）

- パッチムがないとき: **~ㄹ 때**（例: 가다 → 갈 때）
- パッチムがあるとき: **~을 때**（例: 먹다 → 먹을 때）
- `ㄹ` パッチムのとき: **~ㄹ 때**（例: 만들다 → 만들 때）
- 名詞の後ろ: **名詞 + 때**（例: 방학 → 방학 때, 학생 → 학생 때）
  ※ 아침(朝)には 보통 '아침에' が自然。

- 비가 `올 때` 우산을 써요.（雨が降る時に傘をさします。）
- 밥을 `먹을 때` 조용히 하세요.（ご飯を食べる時、静かにしてください。）
- 저는 슬플 `때` 매운 음식을 먹어요.（私は悲しい時、辛い食べ物を食べます。）
- 방학 `때` 한국 여행을 갈 거예요.（休み（長期休暇）の時に韓国旅行に行くつもりです。）

---

## 4. 회화 연습

**상황：주말 날씨에 따른 계획**（週末の天気に合わせた予定）

- 남편: 유나 씨, 주말에 뭐 할 거예요? 날씨가 좋을까요?（ユナさん、週末に何をしますか？ 天気は良いでしょうか？）
- 아내: 일기예보를 보니까 토요일에 비가 온대요.（天気予報を見たら、土曜日に雨が降るそうです。）
- 남편: 그래요? 비가 오면 야외에서 등산을 못 하겠네요.（そうですか？ 雨が降ったら屋外で登山ができませんね。）
- 아내: 네, 비가 오면 그냥 집에서 영화를 보면서 쉴래요.（はい、雨が降ったらただ家で映画を見ながら休みます。）
- 남편: 그럼 일요일에는 어때요? 일요일엔 갠다고 하던데.（じゃあ、日曜日はどうですか？ 日曜日は晴れると言っていましたが。）
- 아내: 일요일에 날씨가 맑으면 공원에 피크닉 가요. 기분이 좋을 때 밖에서 노는 게 최고예요!（日曜日に天気が良ければ公園にピクニックに行きましょう。気分が良い時に外で遊ぶのが最高です！）
- 남편: 좋아요! 맛있는 도시락을 만들어서 갑시다.（いいですね！美味しいお弁当を作って行きましょう。）

## 練習

文章の空欄に入る正しい単語を作ってみましょう。

```text
1. 모르는 단어가 ________ 사전을 찾으세요. (있다 - 条件)
2. 한국에 ________ 비행기표를 사야 해요. (가다 - 時間)
3. 너무 피곤________ 물을 많이 마시거나 주무세요. (하다 - 条件)
4. 어릴 ________ 무엇을 좋아했어요? (어리다 - 時間)
```

**정답**

1. 있으면
2. 갈 때
3. 하면 (피곤하면)
4. 때 (어릴 때 - '어리다'の形容詞語幹 '어리' + ㄹ 때)
