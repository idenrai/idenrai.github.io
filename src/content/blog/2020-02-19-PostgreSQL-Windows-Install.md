대략 3년만에 컴퓨터에 PostgreSQL 설치할라 했더니 계속 설치가 안되더라.

어제 한 30분 헤매다가, 아침에 출근하자마자 기억나서 설치하고 적어둔다.

**1\. 설치 파일 다운로드**

[다운로드](https://www.enterprisedb.com/downloads/postgres-postgresql-downloads)

**2\. 폴더 이동**

Windows에서 PostgreSQL을 설치할 때엔, **설치 파일을 C:\\로 이동시켜야 한다**.

Download폴더나 데스크탑에서 실행해봐야 실행 안됨.

**3\. 설치**

C:\\ 바로 밑으로 이동시킨 설치 파일을 실행하여 PostgreSQL을 설치.
