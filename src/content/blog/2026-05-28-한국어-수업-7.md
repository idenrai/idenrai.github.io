# 수업 7. 어제 뭐 했어요?

今回は**過去形**を学ぶ。  
現在形（~아요/어요）を覚えていれば、過去形への変換はシンプル。

---

## 1. 과거형 만들기（過去形の作り方）

現在形の`아요/어요`を`았어요/었어요`に変えるだけ。

| 現在形 | 過去形 | 意味 |
|--------|--------|------|
| 가요 | 갔어요 | 行きました |
| 와요 | 왔어요 | 来ました |
| 먹어요 | 먹었어요 | 食べました |
| 마셔요 | 마셨어요 | 飲みました |
| 봐요 | 봤어요 | 見ました |
| 자요 | 잤어요 | 寝ました |
| 읽어요 | 읽었어요 | 読みました |
| 들어요 | 들었어요 | 聞きました |

> ルール：現在形の`요`を取って`ㅆ어요`をつける。  
> 母音で終わる場合は縮約が起きる。`가요 → 가 + ㅆ어요 → 갔어요`

---

## 2. 하다 동사의 과거형（하다動詞の過去形）

`하다`動詞は`해요 → 했어요`と変化する。

| 現在形 | 過去形 | 意味 |
|--------|--------|------|
| 공부해요 | 공부했어요 | 勉強しました |
| 일해요 | 일했어요 | 働きました |
| 요리해요 | 요리했어요 | 料理しました |
| 운동해요 | 운동했어요 | 運動しました |
| 전화해요 | 전화했어요 | 電話しました |
| 청소해요 | 청소했어요 | 掃除しました |

---

## 3. 이에요/예요의 과거형（〜です の過去形）

| 現在形 | 過去形 | 使う場面 |
|--------|--------|----------|
| 예요 | 였어요 | 母音終わりの名詞 |
| 이에요 | 이었어요 | 子音終わりの名詞 |

```text
학생이에요.    → 학생이었어요.    （学生でした。）
의사예요.      → 의사였어요.      （医者でした。）
```

---

## 4. 있었어요 / 없었어요

| 現在形 | 過去形 | 意味 |
|--------|--------|------|
| 있어요 | 있었어요 | いました / ありました |
| 없어요 | 없었어요 | いませんでした / ありませんでした |

```text
집에 있었어요.           → 家にいました。
열쇠가 없었어요.         → 鍵がありませんでした。
```

---

## 5. 과거형 회화（過去形の会話）

### 어제 뭐 했어요?（昨日何をしましたか？）

```text
어제 뭐 했어요?
  → 친구를 만났어요.     → 友達に会いました。
  → 집에서 쉬었어요.     → 家で休みました。
  → 드라마를 봤어요.     → ドラマを見ました。
  → 그냥 집에 있었어요.  → ただ家にいました。
```

### 어디 갔어요?（どこへ行きましたか？）

```text
어디 갔어요?
  → 마트에 갔어요.       → スーパーに行きました。
  → 친정에 갔어요.       → 実家に行きました。
  → 아무 데도 안 갔어요. → どこにも行きませんでした。
```

### 맛있었어요?（おいしかったですか？）

```text
맛있었어요?
  → 네, 너무 맛있었어요. → はい、とてもおいしかったです。
  → 별로였어요.          → 微妙でした。
```

---

## 6. 부정 과거형（否定の過去形）

`안`または`못`を動詞の前に置く。

```text
안 갔어요.               → 行きませんでした。
못 먹었어요.             → 食べられませんでした。
공부 안 했어요.          → 勉強しませんでした。
```

> `안`：〜しなかった（意志・事実）  
> `못`：〜できなかった（不可能）

## 練習

다음 문장을 과거형으로 바꿔 보세요.（以下の文を過去形に変えてみましょう。）

```text
1. 커피를 마셔요.         →
2. 집에 있어요.           →
3. 운동해요.              →
4. 영화를 봐요.           →
5. 학생이에요.            →
```

**정답**

```text
1. 커피를 마셨어요.
2. 집에 있었어요.
3. 운동했어요.
4. 영화를 봤어요.
5. 학생이었어요.
```

## Tips

### 불규칙 동사 예고（不規則動詞の予告）

一部の動詞は活用するときに語幹が変わる。今は全部覚えなくてよいが、以下はよく使うので頭に入れておこう。

| 原型 | 現在形 | 過去形 | 変化 |
|------|--------|--------|------|
| 듣다 | 들어요 | 들었어요 | ㄷ → ㄹ |
| 걷다 | 걸어요 | 걸었어요 | ㄷ → ㄹ |
| 돕다 | 도와요 | 도왔어요 | ㅂ → 와 |
| 맵다 | 매워요 | 매웠어요 | ㅂ → 워 |

不規則活用は今後詳しく取り上げる。今は「こういうものがある」という認識でOK。
