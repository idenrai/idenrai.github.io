# 수업 28. 한국어를 배우러 가요

오늘은 어떤 이동의 목적을 나타내는 **~(으)러 가다/오다** 표현과,
행동의 계획이나 목적을 나타내는 **~(으)려고** 표현을 배운다.

---

## 1. 목적 관련 어휘 (目的의 語彙)

| 한국어 | 일본어 | 예문 |
|--------|--------|------|
| 은행 | 銀行 | 돈을 찾으러 은행에 가요. |
| 우체국 | 郵便局 | 편지를 보내러 우체국에 왔어요. |
| 시장 | 시장 / 市場 | 시장에 야채를 사러 가요. |
| 환전 (환전하다) | 両替 (両替する) | 은행에 환전하러 갈 거예요. |
| 보내다 | 送る / 発送する | 친구에게 소포를 보내려고 해요. |

---

## 2. ~(으)러 가다/오다 (〜しに行く/来ます - 移動の目的)

어떤 장소에 이동하는 구체적인 목적을 나타낼 때 사용한다. 뒤에는 반드시 `가다, 오다, 다니다`와 같은 **이동 동사**가 온다.

- 받침이 없을 때: **~러 가다/오다** (예: 공부하다 → 공부하러 가다)
- 받침이 있을 때: **~으러 가다/오다** (예: 찾다 → 찾으러 가다)
- 'ㄹ' 받침으로 끝날 때: **~러 가다/오다** (예: 만들다 → 만들러 가다 - 'ㄹ' 유지)

```text
한국에 한국어를 배우러 가요.
  → 韓国に韓国語を学びに行きます。

도서관에 책을 읽으러 왔어요.
  → 図書館に本を読みに行きました（来ました）。

돈을 찾으러 은행에 다녀요.
  → お金をおろしに（探しに）銀行に通っています。
```

---

## 3. ~(으)려고 (〜しようと / 〜するために - 計画・意図)

어떤 일을 할 주어의 계획이나 의도, 목적을 나타낸다. (뒤에 이동 동사 외에도 다양한 일반 동사가 올 수 있다.)
`~(으)려고 해요` 형태로 쓰면 미래 계획을 나타내는 종결 어미가 된다. (수업 13 복습)

- 받침이 없을 때: **~려고** (예: 사다 → 사려고)
- 받침이 있을 때: **~으려고** (예: 먹다 → 먹으려고)
- 'ㄹ' 받침으로 끝날 때: **~려고** (예: 만들다 → 만들려고)

```text
한국 여행을 가려고 돈을 모아요.
  → 韓国旅行に行くためにお金を貯めています。

건강해지려고 매일 운동을 합니다.
  → 건강해지려고 매일 운동을 합니다.

아내에게 선물을 주려고 시계를 샀어요.
  → 妻にプレゼントをあげるために時計を買いました。
```

※ 주의: `~(으)려고` 뒤에는 `~(으)세요`, `~(으)ㄹ까요?`, `~(으)ㅂ시다` 같은 청유/명령형을 쓸 수 없다.
- ❌ ~~도서관에 책을 읽으려고 갑시다.~~ → ✅ **도서관에 책을 읽으러 갑시다.** (이동 목적일 때는 `~러`를 쓴다.)

---

## 4. 회화 연습

**상황: 은행에 가기 전에 나눈 대화 (銀行に行く前の会話)**

```text
남편: 유나 씨, 지금 외출해요? 어디 가요?
     (ユナさん、今出かけますか? どこへ行きますか？)

아내: 네, 돈을 환전하러 은행에 가요.
     (はい, 돈을 환전하러 은행에 가요.)

남편: 아, 다음 달에 한국 여행 가려고 환전하는 거예요?
     (あ, 내년에 한국 여행 가려고 환전하는 거예요?)

아내: 맞아요. 여행할 때 쓰려고 한국 돈으로 좀 바꾸어 두려고요.
     (そうです。旅行の時に使うために韓国のお金に少し両替しておこうと思いまして。)

남편: 그렇군요. 우체국은 안 가요? 저번에 보낼 소포가 있다고 들었는데.
     (そうですか。郵便局には行きませんか？この前送る荷物があると言っていましたが。)

아내: 소포는 내일 보내려고 해요. 오늘은 은행만 갈 거예요.
     (荷物は明日送るつもりです。今日は銀行だけ行きます。)

남편: 알았어요. 잘 다녀와요!
     (分かりました。行ってらっしゃ이!)
```

## 練習

**문장의 빈칸에 들어갈 알맞은 단어를 만들어 보세요. (文章의 空欄に入る正しい단어를 만들어 보세요.)**

1. 친구를 ________ 카페에 가요. (만나다 - 이동 목적)
2. 한국 회사에 ________ 한국어를 열심히 공부해요. (취직하다 - 목적)
3. 밥을 ________ 식당에 다녀왔어요. (먹다 - 이동 목적)
4. 부모님께 ________ 선물을 고르고 있어요. (보내다 - 의도/목적)

**정답 (答え)**
1. 만나러
2. 취직하려고
3. 먹으러
4. 보내려고

## Tips

### '~러 가다'와 '~려고'의 쉬운 구분
- 목적지에 **이동하는 동작(가다/오다/다니다)**이 문장의 핵심인 경우: **~(으)러 가다/오다**
- 이동에 상관없이 구체적인 **행동의 목적이나 이유**를 나타내며 돈을 모으거나 공부를 하는 행위 등일 경우: **~(으)려고**
