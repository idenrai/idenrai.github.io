# 수업 9. 뭐 먹고 싶어요?

今回は**希望・欲求**を表す表現を学ぶ。  
「〜したい」「〜が欲しい」など、日常会話で最もよく使う表現の一つ。

---

## 1. ~고 싶어요（〜したいです）

動詞の語幹に`고 싶어요`をつけるだけ。語幹の形に関係なく常に同じ形。

```text
동사 어간 + 고 싶어요
```

| 原型 | ~고 싶어요 | 意味 |
|------|-----------|------|
| 먹다 | 먹고 싶어요 | 食べたいです |
| 마시다 | 마시고 싶어요 | 飲みたいです |
| 가다 | 가고 싶어요 | 行きたいです |
| 오다 | 오고 싶어요 | 来たいです |
| 자다 | 자고 싶어요 | 寝たいです |
| 쉬다 | 쉬고 싶어요 | 休みたいです |
| 보다 | 보고 싶어요 | 見たいです / 会いたいです |
| 사다 | 사고 싶어요 | 買いたいです |
| 배우다 | 배우고 싶어요 | 習いたいです |

> `보고 싶어요`は「見たい」だけでなく、人に対して使うと「会いたい」の意味になる。  
> `보고 싶었어요` → 「会いたかったです」（수업 4で学んだ挨拶と繋がる！）

---

## 2. ~고 싶지 않아요（〜したくないです）

否定形は`싶어요`を`싶지 않아요`に変える。

```text
가고 싶지 않아요.        → 行きたくないです。
먹고 싶지 않아요.        → 食べたくないです。
```

口語では`고 싶어요`の縮約形もよく使う。

```text
먹고 싶어요 → 먹고 싶어
가고 싶어요 → 가고 싶어
```

---

## 3. ~고 싶었어요（〜したかったです）

過去の希望は`싶어요 → 싶었어요`に変える。

```text
한국에 가고 싶었어요.    → 韓国に行きたかったです。
더 자고 싶었어요.        → もっと寝たかったです。
보고 싶었어요.           → 会いたかったです。
```

---

## 4. 원해요 / 갖고 싶어요（欲しいです）

物が欲しいときの表現。

```text
[명사] + 이/가 갖고 싶어요    → 〜が欲しいです
[명사] + 을/를 원해요          → 〜を欲しいです / 望んでいます
```

```text
새 가방이 갖고 싶어요.   → 新しいバッグが欲しいです。
커피 한 잔을 원해요.     → コーヒー一杯が欲しいです。
```

---

## 5. ~(으)러 가다 / 오다（〜しに行く / 来る）

목적（目的）を表す表現。「〜しに行く」という場面で使う。

```text
동사 어간 + (으)러 가다/오다
```

- 語幹が母音または`ㄹ`で終わる → `러`
- 語幹が子音で終わる → `으러`

| 例文 | 意味 |
|------|------|
| 밥 먹으러 가요. | ご飯を食べに行きます。 |
| 커피 마시러 와요. | コーヒーを飲みに来ます。 |
| 쇼핑하러 가요. | 買い物をしに行きます。 |
| 한국어 배우러 다녀요. | 韓国語を習いに通っています。 |

---

## 6. 회화 연습（会話練習）

### 뭐 먹고 싶어요?（何が食べたいですか？）

```text
뭐 먹고 싶어요?
  → 삼겹살 먹고 싶어요.  → サムギョプサルが食べたいです。
  → 아무거나 괜찮아요.   → 何でもいいです。
  → 별로 안 먹고 싶어요. → あまり食べたくないです。
```

### 주말에 뭐 하고 싶어요?（週末に何をしたいですか？）

```text
주말에 뭐 하고 싶어요?
  → 그냥 집에서 쉬고 싶어요.     → ただ家で休みたいです。
  → 맛있는 거 먹으러 가고 싶어요. → おいしいものを食べに行きたいです。
  → 쇼핑하러 가고 싶어요.         → 買い物に行きたいです。
```

## 練習

다음 문장을 한국어로 만들어 보세요.（以下の文を韓国語で作ってみましょう。）

```text
1. コーヒーが飲みたいです。
2. 韓国に行きたいです。
3. もっと寝たかったです。
4. 映画を見に行きます。
5. 何が食べたいですか？
```

**정답**

```text
1. 커피를 마시고 싶어요. / 커피가 마시고 싶어요.
2. 한국에 가고 싶어요.
3. 더 자고 싶었어요.
4. 영화 보러 가요.
5. 뭐 먹고 싶어요?
```

## Tips

### 보고 싶어요의 특별한 쓰임

`보다`（見る）に`고 싶어요`をつけた`보고 싶어요`は、  
人を目的語にすると「会いたい」という意味になる。

```text
영화가 보고 싶어요.      → 映画が見たいです。
엄마가 보고 싶어요.      → お母さんに会いたいです。
보고 싶었어요.           → 会いたかったです。（수업 4의 인사와 연결！）
```

### 수업 1~9 핵심 표현 정리

| 場面 | 표현 |
|------|------|
| 〜です | 이에요/예요 |
| 〜します | 아요/어요 |
| 〜しました | 았어요/었어요 |
| 〜したいです | 고 싶어요 |
| 〜しに行きます | (으)러 가요 |
