# 수업 9. 뭐 먹고 싶어요?

希望・欲求を表す表現を学ぶ。「〜したい」は日常会話で最もよく使う表現の一つ。

## 1. ~고 싶어요（〜したいです）

動詞の語幹に`고 싶어요`をつける。語幹の形に関係なく常に同じ形。

| 原型 | ~고 싶어요 | 意味 |
| ---- | ---- | ---- |
| 먹다 | 먹고 싶어요 | 食べたいです |
| 마시다 | 마시고 싶어요 | 飲みたいです |
| 가다 | 가고 싶어요 | 行きたいです |
| 오다 | 오고 싶어요 | 来たいです |
| 자다 | 자고 싶어요 | 寝たいです |
| 쉬다 | 쉬고 싶어요 | 休みたいです |
| 보다 | 보고 싶어요 | 見たいです / 会いたいです |
| 사다 | 사고 싶어요 | 買いたいです |
| 배우다 | 배우고 싶어요 | 習いたいです |

> `보고 싶어요`は「見たい」だけでなく、人に対して使うと「会いたい」の意味になる。
> 수업 4で学んだ`보고 싶었어요`（会いたかったです）とつながる。

---

## 2. 否定・過去

### 否定：~고 싶지 않아요（〜したくないです）

`싶어요`を`싶지 않아요`に変える。

- 가고 `싶지 않아요.`（行きたくないです。）
- 먹고 `싶지 않아요.`（食べたくないです。）

### 過去：~고 싶었어요（〜したかったです）

`싶어요`を`싶었어요`に変える。

- 한국에 `가고 싶었어요.`（韓国に行きたかったです。）
- 더 `자고 싶었어요.`（もっと寝たかったです。）

### タメ口：~고 싶어

`요`を取るだけ。

- 먹고 `싶어`.（食べたい。）
- 가고 `싶어`.（行きたい。）

---

## 3. 갖고 싶어요 / 원해요（欲しいです）

物が欲しいときの表現。

- [명사] + `이/가 갖고 싶어요`（〜が欲しいです）
- [명사] + `을/를 원해요`（〜を望んでいます / 欲しいです）

> `갖고 싶어요`は会話で自然。`원해요`はやや硬い。

- 새 가방이 `갖고 싶어요.`（新しいカバンが欲しいです。）
- 커피 한 잔을 `원해요.`（コーヒー一杯が欲しいです。）

---

## 4. ~(으)러 가다/오다（〜しに行く / 来る）

目的を表す表現。「〜しに行く」の場面で使う。

- 語幹が母音またはㄹで終わる → `러`
- 語幹が子音で終わる → `으러`

| 例文 | 意味 |
| ---- | ---- |
| 밥 먹으러 가요. | ご飯を食べに行きます。 |
| 커피 마시러 와요. | コーヒーを飲みに来ます。 |
| 쇼핑하러 가요. | 買い物をしに行きます。 |
| 한국어 배우러 다녀요. | 韓国語を習いに通っています。 |

---

## 5. 실전 표현

### 뭐 먹고 싶어요?（何が食べたいですか？）

- 삼겹살 먹고 싶어요.（サムギョプサルが食べたいです。）
- 아무거나 괜찮아요.（何でもいいです。）
- 별로 안 먹고 싶어요.（あまり食べたくないです。）

### 주말에 뭐 하고 싶어요?（週末に何をしたいですか？）

- 그냥 집에서 쉬고 싶어요.（ただ家で休みたいです。）
- 맛있는 거 먹으러 가고 싶어요.（おいしいものを食べに行きたいです。）
- 쇼핑하러 가고 싶어요.（買い物に行きたいです。）

---

## 練習

以下の文を韓国語にしてみましょう。

```text
1. コーヒーが飲みたいです。
2. 韓国に行きたいです。
3. もっと寝たかったです。
4. 映画を見に行きます。
5. 何が食べたいですか？
```

**정답**

1. 커피를 마시고 싶어요. / 커피가 마시고 싶어요.
2. 한국에 가고 싶어요.
3. 더 자고 싶었어요.
4. 영화 보러 가요.
5. 뭐 먹고 싶어요?
