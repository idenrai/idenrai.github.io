# 수업 5. 몇 살이에요?

韓国語の数字は**2種類**ある。場面によって使い分ける。日本語の「いち、に、さん」と「ひとつ、ふたつ」の使い分けに似ている。

## 1. 한자어 숫자（漢数字系）

日本語の「いち・に・さん」に相当する。漢字由来の数字。

| 数字 | 韓国語 | 読み |
| ---- | ---- | ---- |
| 0 | 영 / 공 | ヨン / コン |
| 1 | 일 | イル |
| 2 | 이 | イ |
| 3 | 삼 | サム |
| 4 | 사 | サ |
| 5 | 오 | オ |
| 6 | 육 | ユク |
| 7 | 칠 | チル |
| 8 | 팔 | パル |
| 9 | 구 | ク |
| 10 | 십 | シプ |
| 100 | 백 | ペク |
| 1,000 | 천 | チョン |
| 10,000 | 만 | マン |

### 십 단위（10の位）の作り方

日本語と同じ構造で、組み合わせるだけ。

- 11 = `십일`、20 = `이십`、35 = `삼십오`、99 = `구십구`

### 使う場面

- 날짜（日付） → `오월 이십육일`（5月26日）
- 전화번호（電話番号） → `공일공-일이삼사-오육칠팔`
- 가격（値段） → `삼만 원`（3万ウォン）
- 층（フロア） → `삼층`（3階）

---

## 2. 고유어 숫자（固有語系）

日本語の「ひとつ・ふたつ」に相当する。韓国固有の数字。

| 数字 | 韓国語 | 読み |
| ---- | ---- | ---- |
| 1 | 하나 | ハナ |
| 2 | 둘 | トゥル |
| 3 | 셋 | セッ |
| 4 | 넷 | ネッ |
| 5 | 다섯 | タソッ |
| 6 | 여섯 | ヨソッ |
| 7 | 일곱 | イルゴプ |
| 8 | 여덟 | ヨドル |
| 9 | 아홉 | アホプ |
| 10 | 열 | ヨル |
| 20 | 스물 | スムル |
| 30 | 서른 | ソルン |
| 40 | 마흔 | マフン |
| 50 | 쉰 | シュイン |

> 고유어 숫자は99まで。100以上は한자어を使う。

### 使う場面

- 나이（年齢） → `서른다섯 살`（35歳）
- 시간（時間の「時」） → `두 시`（2時）
- 개수（個数） → 사과 `세 개`（りんご3個）

---

## 3. 어느 숫자를 쓸까?（どちらを使う？）

| 場面 | 使う数字 | 例 |
| ---- | ---- | ---- |
| 年齢 | 고유어 | 스물다섯 살 |
| 時（時刻の「時」） | 고유어 | 세 시 |
| 分・秒 | 한자어 | 삼십 분 |
| 値段・番号・日付 | 한자어 | 오천 원 |
| 個数（〜개, 〜명） | 고유어 | 두 명, 세 개 |

> 一番ハマりやすい落とし穴：**時刻**は「時」が고유어、「分」が한자어。
> ex) `두 시 삼십 분`（2時30分）

---

## 4. 나이 묻기（年齢の聞き方）

- `나이가 어떻게 되세요?`（お年はおいくつですか？ — 丁寧）
- `몇 살이에요?`（何歳ですか？ — 普通）

### 答え方

고유어 숫자 + `살`で年齢を表す。

- `스물다섯 살이에요.`（25歳です。）
- `서른 살이에요.`（30歳です。）
- `마흔두 살이에요.`（42歳です。）

---

## 5. 가격 묻기（値段の聞き方）

- `이거 얼마예요?`（これはいくらですか？）

### 答え方（한자어 + 원）

- `천 원이에요.`（1,000ウォンです。）
- `오천 원이에요.`（5,000ウォンです。）
- `삼만 오천 원이에요.`（35,000ウォンです。）

## 練習

以下の数字を韓国語で言ってみましょう。

```text
【한자어で】

【고유어で】

```

## Tips

### 고유어 숫자 + 단위 = 縮約に注意

고유어 숫자は単位詞の前で形が変わるものがある。暗記必須。

| 元の形 | 単位の前 | 例 |
| ---- | ---- | ---- |
| 하나 | 한 | 한 명（1人） |
| 둘 | 두 | 두 개（2個） |
| 셋 | 세 | 세 시（3時） |
| 넷 | 네 | 네 살（4歳） |
| 스물 | 스무 | 스무 살（20歳） |
