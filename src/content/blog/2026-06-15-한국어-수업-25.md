# 수업 25. 한국에 가 본 적이 있어요?

過去の経験を尋ねたり表現したりするときに使う **~(으)ㄴ 적이 있다/없다** と、試みや経験を表す **~아/어 보다** を学ぶ。

## 1. 경험 관련 어휘（経験の語彙）

| 韓国語 | 日本語 | 例文 |
| ---- | ---- | ---- |
| 경험 | 経験 | 여러 가지 `경험`을 했어요. |
| 타다 | (乗り物に)乗る | 비행기를 `탄 적이 있어요`. |
| 가 보다 | 行ってみる | 한국 제주도에 `가 봤어요`. |
| 먹어 보다 | 食べてみる | 비빔밥을 `먹어 본 적이 있어요`. |
| 만나다 | 会う | 한국 연예인을 `만난 적이 있어요`. |

---

## 2. ~(으)ㄴ 적이 있다/없다（〜したことがある/ない - 経験）

過去にある事をした経験が「ある」または「ない」ことを表すときに使う。

- パッチムがないとき: **~ㄴ 적이 있다/없다**（例: 가다 → 간 적이 있다）
- パッチムがあるとき: **~은 적이 있다/없다**（例: 먹다 → 먹은 적이 있다）
- `ㄹ` パッチムのとき: **~ㄴ 적이 있다/없다**（例: 만들다 → 만든 적이 있다 ※ㄹ脱落）

- 한국에 `간 적이 있어요?`（韓国に行ったことがありますか？）
- 저는 매운 음식을 `먹은 적이 없어요`.（私は辛い食べ物を食べたことがありません。）
- 말을 `탄 적이 있어요?`（馬に乗ったことがありますか？）

---

## 3. ~아/어 보다（〜してみる - 試み・経験）

수업 18で学んだ `~아/어 보다` の過去形である **~아/어 봤어요** は、すでに試みて完了した「経験」を具体的に強調するときによく使われる。
前に学んだ `~(으)ㄴ 적이 있다/없다` と結合して **~아/어 본 적이 있다/없다（〜してみたことがある/ない）** の形でも非常によく使われる。

- 제주도에 `가 봤어요`.（済州島に行ってみました。/ 行ったことがあります。）
- 제주도에 `가 본 적이 있어요`.（済州島に行ってみたことがあります。※より経験を強調）
- 김치를 `먹어 봤어요?`（キムチを食べてみましたか？）

---

## 4. 회화 연습

**상황：과거 경험 묻기**（過去の経験についての会話）

- 남편: 유나 씨는 한국의 제주도에 가 본 적이 있어요?（ユナさんは韓国の済州島に行ってみたことがありますか？）
- 아내: 아니요, 제주도에는 가 본 적이 없어요. 서울에만 가 봤어요.（いいえ、済州島には行ったことがありません。ソウルにだけ行ってみました。）
- 남편: 그래요? 제주도는 정말 아름다운 섬이에요. 다음에 꼭 같이 가 봐요.（そうですか？ 済州島は本当に美しい島です。今度ぜひ一緒に行ってみましょう。）
- 아내: 네, 좋아요. 참, 남편은 한라산(제주도의 산)에 올라가 봤어요?（はい、いいですね。そういえば、夫はハルラ山(済州島の山)に登ってみましたか？）
- 남편: 네, 등산해 본 적이 있어요. 조금 힘들었지만 경치가 아주 좋았어요.（はい、登山したことがあります。少し大変でしたが景色がとても良かったです。）
- 아내: 부럽네요. 저도 한라산에 가 보고 싶어요.（羨ましいですね。私もハルラ山に行ってみたいです。）

## 練習

文章の空欄に入る正しい単語を作ってみましょう。

```text
1. 저는 한국 비행기를 ________ 적이 없어요. (타다 - 経験)
2. 비빔밥은 진짜 맛있으니까 한 번 ________ 보세요. (먹다 - 試み)
3. 예전에 한국 친구를 ________ 적이 있어요. (만나다 - 経験)
4. 김치를 맛________ 적이 있습니까? (보다 - 経験)
```

**정답**

1. 탄
2. 먹어
3. 만난
4. 본 (맛본 - '맛보다' の活用形)

## Tips

### '가 봤어요' と '간 적이 있어요' の微妙な違い

どちらも「行った（経験がある）」を意味するが、会話では **가 봤어요** の方が軽く、日常的によく使われる。
**간 적이 있어요** は、もう少し客観的な事実（経験の有無）を表すニュアンスが強い。
