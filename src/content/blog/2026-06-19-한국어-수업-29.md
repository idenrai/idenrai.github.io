# 수업 29. 고마워! 친구 사이의 반말 표현

友人や恋人、目下の人など、とても親しくて打ち解けた間で使う **반말（パンマル・タメ口）** を学ぶ。
韓国語でタメ口を使うときのルールと、よく使われる基本語彙を整理しよう！

## 1. 인칭 대명사 및 호칭의 변화（人称代名詞と呼称の変化）

敬語（존댓말）では自分をへりくだり相手を高めて言うが、タメ口（반말）では直接的な代名詞を使う。

| 韓国語 (敬語) | 韓国語 (タメ口) | 日本語 |
| ---- | ---- | ---- |
| 저 (제) | **나 (내)** | 私 (僕 / 俺) |
| 당신 / 씨 (이름) | **너 (네 / 네가)** | 君 / お前 / あなた |
| 저희 | **우리** | 私たち |
| 예 / 네 | **응** | うん |
| 아니요 | **아니** | ううん / いや |

> ※ `네가（君が）`は会話では「니가（ニガ）」と発音することが多い。（나(내)と너(네)の発音が似ていて混乱を防ぐため）

---

## 2. 해요체에서 반말 만들기（해요体 → タメ口）

해요체（〜요）から単に **「요」を取る** と、最も基本的なタメ口になる！

| 해요体 | タメ口 (반말) | 意味 |
| ---- | ---- | ---- |
| 가요 | **가** | 行く / 行って |
| 먹어요 | **먹어** | 食べる / 食べて |
| 공부해요 | **공부해** | 勉強する / 勉強して |
| 예뻐요 | **예뻐** | 綺麗だ / 可愛い |
| 맛있어요 | **맛있어** | 美味しい |
| 아니에요 | **아니야** | 違う / 違って |
| 예요 / 이에요 | **야 / 이야** | 〜だよ |

- 名詞にパッチムがないとき: **~야**（例: 친구야, 유나야）
- 名詞にパッチムがあるとき: **~이야**（例: 남편이야, 선생님이야）

- A: 이거 뭐`야`?（これ何？）
  B: 이건 내 선물`이야`.（これは私のプレゼントだよ。）
- A: 지금 어디 `가`?（今どこ行くの？）
  B: 나 지금 마트에 `가`.（私今スーパーに行くよ。）

---

## 3. 다양한 반말 종결어미（様々なタメ口の語尾）

敬語の提案、命令、質問の形に対応するタメ口のルール。

### 1) 제안 / 청유 (~하자 - 〜しよう)
敬語の `~(으)ㅂ시다 / ~아요` に対応し、動詞の後ろに **~자** をつける。

- 가다 → `가자`（行こう）
- 먹다 → `먹자`（食べよう）

- 우리 내일 만나`자`.（私たち明日会おう。）

### 2) 질문 (~니? / ~어/아? - 〜する？)
単に語尾を上げて言うか、質問のニュアンスを込めて **~니?** または **~어/아?** を使う。

- 하다 → 하`니`? / 해?
- 먹다 → 먹`니`? / 먹어?

- 밥 먹었`어`? / 밥 먹었`니`?（ご飯食べた？）

### 3) 명령 (~아/어라 - 〜しろ / 〜しなさい)
単に해요体から「요」を抜くより、少し命令や指示の度合いを強めるときに使う。

- 보다 → 봐 / 봐`라`
- 자다 → 자 / 자`라`

- 숙제 빨리 해`라`!（宿題早くしなさい！）

---

## 4. 자주 쓰는 반말 일상 표현（よく使うタメ口の日常表現）

- `안녕!` → やぁ！ / バイバイ！ (会うときと別れるとき両方使う)
- `잘 자!` → おやすみ！
- `고마워.` → ありがとう。
- `미안해.` → ごめんね。
- `맞아.` → その通り（だね）。
- `진짜?` → 本当？
- `축하해!` → おめでとう！

---

## 5. 회화 연습

**상황：아침에 부부가 나눈 대화**（朝の夫婦の会話）

- 남편: 유나야, 잘 잤어?（ユナ、よく眠れた？）
- 아내: 응, 잘 잤어. 자기는? 오늘 몇 시에 출근해?（うん、よく眠れた。あなたは？今日何時に出勤する？）
- 남편: 나 오늘은 오전 10시에 출근해. 그래서 조금 더 자도 돼.（俺、今日は午前10時に出勤する。だからもう少し寝てもいいよ。）
- 아내: 진짜? 잘됐다! 그럼 아침 같이 먹자. 내가 아침 만들게.（本当？よかった！じゃあ朝ごはん一緒に食べよう。私が朝ごはん作るね。）
- 남편: 고마워! 역시 유나가 최고야.（ありがとう！やっぱりユナが最高だよ。）
- 아내: 아침 다 먹은 후에 같이 청소하자.（朝ごはん全部食べた後に一緒に掃除しよう。）
- 남편: 응, 알았어. 내가 도와줄게.（うん、分かった。手伝うよ。）

## 練習

敬語の文章を正しいタメ口の文章に書き換えてください。

```text
1. 저는 내일 학교에 가요. → 나 내일 학교에 ________.
2. 같이 점심 먹읍시다. → 같이 점심 ________.
3. 고맙습니다. 주말 잘 보내세요. → 고마워. 주말 잘 ________.
4. 이것은 제 핸드폰이에요. → 이건 내 핸드폰________.
```

**정답**

1. 가
2. 먹자
3. 보내 / 보내라 (友達同士では '잘 보내!' が自然)
4. 이야

## Tips

### 반말을 쓸 때 주의할 점!（タメ口を使う時の注意点！）
韓国では初対面の人にタメ口を使うと大きな無礼になる。
年齢が自分より下であっても、相手の同意なしにすぐタメ口を使うのではなく、通常は **"말 편하게 하셔도 돼요"（言葉を崩してもいいですよ）** または **"우리 이제 말 놓을까요?"（私たちもうタメ口にしましょうか？）** という提案を経てからタメ口を使う。
夫婦間やとても親しい友人間では、タメ口を使うことでより強い親近感を感じることができる！
