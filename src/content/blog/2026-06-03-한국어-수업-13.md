# 수업 13. 뭐 할 거예요?

未来・予定・意図を表す表現を学ぶ。これで時制が現在・過去・未来すべてカバーできる。

## 1. ~ㄹ/을 거예요（〜するつもりです / 〜でしょう）

未来の予定や意志を表す最も基本的な表現。

- 語幹が母音またはㄹで終わる → `ㄹ 거예요`
- 語幹が子音で終わる → `을 거예요`

| 原型 | ~ㄹ/을 거예요 | 意味 |
| ---- | ---- | ---- |
| 가다 | 갈 거예요 | 行くつもりです |
| 오다 | 올 거예요 | 来るつもりです |
| 먹다 | 먹을 거예요 | 食べるつもりです |
| 마시다 | 마실 거예요 | 飲むつもりです |
| 자다 | 잘 거예요 | 寝るつもりです |
| 공부하다 | 공부할 거예요 | 勉強するつもりです |
| 만나다 | 만날 거예요 | 会うつもりです |

- 내일 한국에 `갈 거예요.`（明日韓国に行くつもりです。）
- 주말에 집에 `있을 거예요.`（週末は家にいるつもりです。）
- 뭐 `먹을 거예요?`（何を食べるつもりですか？）

---

## 2. ~(으)려고 해요（〜しようと思っています）

まだ決定していないが検討中、という意図を表す。
`~ㄹ 거예요`より「まだ考え中」のニュアンスが強い。

- 語幹が母音またはㄹで終わる → `려고 해요`
- 語幹が子音で終わる → `으려고 해요`

- 내년에 한국에 `가려고 해요.`（来年韓国に行こうと思っています。）
- 한국어를 `배우려고 해요.`（韓国語を習おうと思っています。）
- 다이어트를 `하려고 해요.`（ダイエットをしようと思っています。）

---

## 3. ~ㄹ 거예요 vs ~(으)려고 해요

| | ~ㄹ 거예요 | ~(으)려고 해요 |
| ---- | ---- | ---- |
| ニュアンス | 予定・決定済み | 意図・検討中 |
| 確度 | 高い | やや低い |
| 例 | 내일 갈 거예요. | 내일 가려고 해요. |

---

## 4. 否定形

- `안` 갈 거예요.（行かないつもりです。）
- 먹`지 않을 거예요.`（食べないつもりです。）
- 가`려고 하지 않아요.`（行こうとは思っていません。）

---

## 5. 실전 표현

### 언제 ~ㄹ 거예요?（いつ〜するつもりですか？）

- 언제 한국에 `갈 거예요?`（いつ韓国に行くつもりですか？）
  - 7월에 갈 거예요.（7月に行くつもりです。）
  - 아직 모르겠어요.（まだわかりません。）

### 계획이 있어요?（予定はありますか？）

- 이번 주말에 `계획이 있어요?`（今週末予定はありますか？）
  - 네, 가족이랑 나갈 거예요.（はい、家族と出かけるつもりです。）
  - 아니요, 특별한 계획 없어요.（いいえ、特に予定はありません。）

---

## 練習

以下の文を韓国語にしてみましょう。

```text
1. 週末に何をするつもりですか？
2. 友達に会うつもりです。
3. 来月引っ越ししようと思っています。
4. 今日は早く寝るつもりです。
5. まだ決めていません。
```

**정답**

1. 주말에 뭐 할 거예요?
2. 친구를 만날 거예요.
3. 다음 달에 이사하려고 해요.
4. 오늘은 일찍 잘 거예요.
5. 아직 안 정했어요. / 아직 모르겠어요.

## Tips

### 시제 총정리（時制まとめ）

수업 7, 9, 10, 13を通して時制が完成した。

| 時制 | 形 | 例 |
| ---- | ---- | ---- |
| 현재 | ~아요/어요 | 먹어요 |
| 과거 | ~았어요/었어요 | 먹었어요 |
| 미래/예정 | ~ㄹ/을 거예요 | 먹을 거예요 |
| 희망 | ~고 싶어요 | 먹고 싶어요 |
| 의도 | ~(으)려고 해요 | 먹으려고 해요 |

### 口語での縮約

`~ㄹ/을 거예요`は口語でよく縮約される。

- 갈 거예요 → `갈 거야`（タメ口）
- 먹을 거예요 → `먹을 거야`（タメ口）
