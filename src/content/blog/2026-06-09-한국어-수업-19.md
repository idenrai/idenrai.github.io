# 수업 19. 머리가 아파서 약국에 갔어요

病院・薬局での表現（병원·약국 표현）を学ぶ。
いざという時に自分の症状を正確に伝えられるようにしよう。

## 1. 신체 부위와 증상（体と症状）

「〜が痛いです」は `[신체 부위] + 이/가 아파요` と言う。

| 신체 부위 (体の部位) | 예문 (例文) |
| -------------------- | ----------- |
| 머리 (頭) | 머리가 아파요. (頭が痛いです) |
| 배 (お腹) | 배가 아파요. (お腹が痛いです) |
| 목 (喉/首) | 목이 아파요. (喉が痛いです) |
| 허리 (腰) | 허리가 아파요. (腰が痛いです) |
| 다리 (足/脚) | 다리가 아파요. (足が痛いです) |
| 이 / 이빨 (歯) | 이가 아파요. (歯が痛いです) |

### 그 외의 증상（その他の症状）

- `감기에 걸렸어요.`（風邪をひきました。）
- `열이 나요.`（熱が出ます/あります。）
- `기침을 해요.`（咳が出ます/します。）
- `콧물이 나요.`（鼻水が出ます。）
- `소화가 안 돼요.`（消化不良です/胃もたれします。）
- `토할 것 같아요.`（吐きそうです。）

---

## 2. 병원과 약국（病院と薬局）

### 증상 설명하기（症状を説明する）

病院や薬局に行くと、まずこう聞かれる。
- 의사/약사: `어디가 아프세요?`（どこが痛いですか？） / `어떻게 오셨어요?`（どうされましたか？）

수업 11で学んだ `~아서/어서`（〜なので・〜して）を使って理由を伝えるのが自然。
- `열이 나서` 왔어요.（熱が出て来ました。）
- `머리가 아파서` 약국에 갔어요.（頭が痛くて薬局に行きました。）
- `배가 아프고 설사를 해서` 병원에 왔어요.（お腹が痛くて下痢をしているので病院に来ました。）

---

## 3. 약국에서 약 사기（薬局で薬を買う）

処方箋（처방전）なしで買える一般的な薬の言い方。

| 韓国語 | 日本語 |
| ---- | ---- |
| 두통약 | 頭痛薬 |
| 감기약 | 風邪薬 |
| 소화제 | 消化剤 / 胃薬 |
| 해열제 | 解熱剤 |
| 진통제 | 鎮痛剤 |
| 파스 | 湿布 |
| 연고 | 軟膏 |
| 밴드 | 絆創膏 |

- `소화제 주세요.`（胃薬をください。）
- `머리가 아파서 그러는데, 진통제 좀 주세요.`（頭が痛いんですが、鎮痛剤をください。）

---

## 4. 복약 지도（薬の飲み方の説明）

韓国の薬局では、薬の飲み方を袋に書いたり、直接説明してくれたりする。

- `하루에 세 번 드세요.`（1日に3回飲んでください。）
- `식후 삼십 분에 드세요.`（食後30分に飲んでください。）
- `식전에 드세요.`（食前に飲んでください。）
- `두 알씩 드세요.`（2錠ずつ飲んでください。）

> 韓国では薬を「食べる（먹다）」と言うのが一般的だが、薬剤師など専門家は丁寧語の「드시다（召し上がる）」を使って `약을 드세요` と言う。

---

## 5. 회화 연습

**상황：약국에서**（薬局で）

- 약사: 어떻게 오셨어요?（どうされましたか？）
- 유나: 감기에 걸려서 왔어요. 목이 많이 아프고 기침을 해요.
       (風邪をひいて来ました。喉がとても痛くて咳が出ます。)
- 약사: 열도 나요?（熱もありますか？）
- 유나: 아니요, 열은 없어요. 콧물만 조금 나요.
       (いいえ、熱はありません。鼻水だけ少し出ます。)
- 약사: 그럼 이 감기약을 드세요. 식후 30분에 두 알씩 드시면 돼요. 졸릴 수 있으니까 운전은 하지 마세요.
       (それではこの風邪薬を飲んでください。食後30分に2錠ずつお飲みになればいいです。眠くなることがあるので運転はしないでください。)
- 유나: 네, 감사합니다. 얼마예요?（はい、ありがとうございます。いくらですか？）

---

## 練習

自分の症状に合わせて言ってみましょう。

```text
1. どこが痛いですか？ (薬局での質問)
2. 頭が痛くて来ました。
3. 風邪薬をください。
4. 1日3回、食後30分に飲んでください。
```

**정답**

1. 어디가 아프세요?
2. 머리가 아파서 왔어요.
3. 감기약 주세요.
4. 하루에 세 번, 식후 삼십 분에 드세요.

## Tips

### 아프다 の 불규칙（으不規則）

`아프다`（痛い）は、語幹の最後が `으` で終わる。`으` は母音と出会うと脱落する。
- 아프 + 아요 → `아파요`

他にも以下のような単語がある。
- 바쁘다（忙しい） → `바빠요`
- 예쁘다（きれいだ） → `예뻐요`
- 슬프다（悲しい） → `슬퍼요`
