# 수업 24. 제 취미는 요리하는 것이에요

오늘은 자신의 취미를 설명할 때 동사를 명사로 바꾸는 **~는 것** 표현과,
두 가지 행동을 동시에 할 때 사용하는 **~(으)면서** 표현을 배운다.

---

## 1. 취미 관련 어휘 (趣味の語彙)

| 한국어 | 일본어 | 예문 |
|--------|--------|------|
| 취미 | 趣味 | 취미가 뭐예요? |
| 독서 (책 읽기) | 読書 (本を読むこと) | 제 취미는 독서예요. |
| 등산 | 登山 / 하이킹 | 주말에 등산을 가요. |
| 요리 (요리하다) | 料理 (料理する) | 저는 요리하는 걸 좋아해요. |
| 여행 (여행하다) | 旅行 (旅行する) | 매년 해외여행을 가요. |
| 사진 찍기 | 写真撮影 | 경치가 좋아서 사진을 찍었어요. |
| 영화 감상 | 映画鑑賞 | 집에서 넷플릭스로 영화 감상을 해요. |

---

## 2. ~는 것 (〜すること - 動詞の名詞化)

동사 뒤에 `~는 것`을 붙여서 동사를 명사 형태로 만든다. (받침 유무 상관없음)
이를 활용하면 "~하는 것을 좋아해요", "~하는 것이 취미예요" 같은 표현을 만들 수 있다.

- 보다 → 보는 것
- 먹다 → 먹는 것
- 요리하다 → 요리하는 것

※ 구어체(대화체)에서는 `~는 것`이 **~는 거**로, `~는 것은`이 **~는 건**, `~는 것을`이 **~는 걸**로 줄여서 자주 쓰인다.

```text
제 취미는 요리하는 것이에요. (또는 요리하는 거예요.)
  → 私の趣味は料理をすることです。

저는 음악 듣는 것을 좋아해요. (또는 듣는 걸 좋아해요.)
  → 私は 음악 듣는 걸 좋아해요.

사진 찍는 것은 재미있어요. (또는 찍는 건 재미있어요.)
  → 写真を撮ることは面白いです。
```

---

## 3. ~(으)면서 (〜하면서 - 同時進行)

한 사람이 두 가지 이상의 행동을 동시에 할 때 사용한다.

- 받침이 없을 때: **~면서** (예: 보다 → 보면서)
- 받침이 있을 때: **~으면서** (예: 먹다 → 먹으면서)
- 'ㄹ' 받침으로 끝날 때: **~면서** (예: 만들다 → 만들면서)

```text
음악을 들으면서 공부를 해요.
  → 音楽を聞きながら勉強をします。

저는 텔레비전을 보면서 밥을 먹어요.
  → 私は 텔레비전을 보면서 밥을 먹어요.

이야기하면서 커피를 마실까요?
  → 話しながらコーヒーを飲みましょうか？
```

---

## 4. 회화 연습

**상황: 취미에 대해 이야기하기 (趣味についての会話)**

```text
남편: 유나 씨는 주말에 보통 뭐 해요?
     (ユナさんは週末に普通何をしますか？)

아내: 저는 집에서 맛있는 음식을 요리하는 걸 좋아해요.
     (저는 집에서 맛있는 음식을 요리하는 걸 좋아해요.)

남편: 와, 요리가 취미군요! 요리하면서 음악도 들어요?
     (わあ、料理が趣味なんですね！ 요리하면서 음악도 들어요?)

아내: 네, 신나는 음악을 들으면서 요리하면 기분이 좋아져요. 남편은 취미가 뭐예요?
     (はい、楽しい音楽を聞きながら料理をすると気分が良くなります。夫の趣味は何ですか？)

남편: 저는 산책하면서 경치 사진을 찍는 것을 좋아해요.
     (저는 산책하면서 경치 사진을 찍는 것을 좋아해요.)

아내: 멋지네요! 다음에 사진 찍으러 갈 때 저도 같이 가요.
     (素敵ですね！今度写真を撮りに行くとき、私も一緒に行きましょう。)
```

## 練習

**문장의 빈칸에 들어갈 알맞은 단어를 만들어 보세요. (文章의 空欄に入る正しい単語を作ってみましょう。)**

1. 제 취미는 한국 드라마를 ________ 것입니다. (보다 - 명사화)
2. 커피를 ________ 신문을 읽어요. (마시다 - 동시행동)
3. 주말에는 집에서 맛있는 것을 ________ 것을 좋아해요. (만들다 - 명사화, 'ㄹ' 탈락 주의)
4. 운전________ 전화를 하면 위험해요. (운전하다 - 동시행동)

**정답 (答え)**
1. 보는
2. 마시면서
3. 만드는 (만들다 + 는 것 = 만드는 것)
4. 하면서 (운전하면서)

## Tips

### 자주 쓰이는 취미 관련 표현
- **~을/를 좋아해요** (〜が好きです)
  - 책 읽는 것을 좋아해요.
- **~에 소질이 있어요 / 잘해요** (〜に素質があります / 得意です)
  - 노래하는 걸 잘해요. (노래를 잘 불러요)
