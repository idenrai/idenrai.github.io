# 수업 12. 이거 먹어도 돼요?

今回は**許可・禁止・義務**を表す表現を学ぶ。  
「〜してもいいですか？」「〜してはいけません」「〜しなければなりません」は日常のあらゆる場面で必要になる。

---

## 1. ~아도/어도 돼요（〜してもいいです）

許可を求めたり、許可を与えるときに使う。

```text
동사 어간 + 아도/어도 돼요
```

| 原型 | ~아도/어도 돼요 | 意味 |
|------|----------------|------|
| 먹다 | 먹어도 돼요 | 食べてもいいです |
| 마시다 | 마셔도 돼요 | 飲んでもいいです |
| 가다 | 가도 돼요 | 行ってもいいです |
| 앉다 | 앉아도 돼요 | 座ってもいいです |
| 사용하다 | 사용해도 돼요 | 使ってもいいです |
| 찍다 | 찍어도 돼요 | 撮ってもいいです |

```text
이거 먹어도 돼요?        → これ食べてもいいですか？
  → 네, 먹어도 돼요.     → はい、食べてもいいです。
  → 아니요, 안 돼요.     → いいえ、だめです。

여기 앉아도 돼요?        → ここに座ってもいいですか？
  → 네, 앉으세요.        → はい、どうぞ座ってください。
```

---

## 2. ~으면 안 돼요（〜してはいけません）

禁止を表す表現。

```text
동사 어간 + (으)면 안 돼요
```

- 語幹が母音またはㄹで終わる → `면 안 돼요`
- 語幹が子音で終わる → `으면 안 돼요`

```text
여기서 담배 피우면 안 돼요.  → ここでタバコを吸ってはいけません。
늦으면 안 돼요.              → 遅れてはいけません。
사진 찍으면 안 돼요.         → 写真を撮ってはいけません。
소리가 크면 안 돼요.         → 音が大きくてはいけません。
```

---

## 3. ~아야/어야 해요（〜しなければなりません）

義務・必要性を表す表現。

```text
동사 어간 + 아야/어야 해요
```

- 語幹の最後の母音が`ㅏ/ㅗ` → `아야 해요`
- それ以外 → `어야 해요`
- `하다` → `해야 해요`

| 原型 | ~아야/어야 해요 | 意味 |
|------|----------------|------|
| 가다 | 가야 해요 | 行かなければなりません |
| 먹다 | 먹어야 해요 | 食べなければなりません |
| 자다 | 자야 해요 | 寝なければなりません |
| 공부하다 | 공부해야 해요 | 勉強しなければなりません |
| 일하다 | 일해야 해요 | 働かなければなりません |

```text
내일 일찍 일어나야 해요.     → 明日早く起きなければなりません。
약을 먹어야 해요.            → 薬を飲まなければなりません。
지금 가야 해요.              → 今行かなければなりません。
```

---

## 4. 세 가지 표현 비교（3つの表現の比較）

| 表現 | 意味 | 例 |
|------|------|----|
| ~아도/어도 돼요 | 〜してもいいです（許可） | 가도 돼요. |
| ~(으)면 안 돼요 | 〜してはいけません（禁止） | 가면 안 돼요. |
| ~아야/어야 해요 | 〜しなければなりません（義務） | 가야 해요. |

```text
약을 먹어도 돼요.            → 薬を飲んでもいいです。
약을 먹으면 안 돼요.         → 薬を飲んではいけません。
약을 먹어야 해요.            → 薬を飲まなければなりません。
```

## 5. 회화 연습（会話練習）

### 식당에서（レストランで）

```text
여기 앉아도 돼요?            → ここに座ってもいいですか？
이거 포장해도 돼요?          → これ持ち帰りできますか？
카드 돼요?                   → カード使えますか？
```

### 가정에서（家庭で）

```text
지금 자도 돼요?              → 今寝てもいいですか？
  → 응, 자도 돼.             → うん、寝ていいよ。（タメ口）
  → 아직 안 자도 돼.         → まだ寝なくていいよ。

손 씻어야 해요.              → 手を洗わなければなりません。
빨리 먹어야 해요.            → 早く食べなければなりません。
```

### ~지 않아도 돼요（〜しなくてもいいです）

義務の否定 = 「しなくていい」は`안 해도 돼요`または`지 않아도 돼요`を使う。

```text
안 가도 돼요.                → 行かなくてもいいです。
걱정 안 해도 돼요.           → 心配しなくてもいいです。
다 먹지 않아도 돼요.         → 全部食べなくてもいいです。
```

## Tips

### 안 돼요 vs 없어요

```text
안 돼요.                     → だめです / できません（禁止・不可）
없어요.                      → ありません / いません（存在しない）
```

```text
카드 안 돼요.                → カードは使えません。（禁止・非対応）
카드 없어요.                 → カードがありません。（持っていない）
```

### 구어체 줄임（会話での縮約）

口語では`해야 해요`を縮めることが多い。

```text
가야 해요 → 가야 해 / 가야 돼（口語）
먹어야 해요 → 먹어야 해 / 먹어야 돼（口語）
```

`가야 돼요`も非常によく使われる自然な表現。
