# 수업 21. 이거 다른 걸로 바꿔 주세요

오늘은 쇼핑할 때 물건을 바꾸거나 환불하는 **교환·환불 표현**을 배운다.
그리고 행동의 순서를 나타내는 **~기 전에 / ~(으)ㄴ 후에**도 함께 알아보자!

---

## 1. 교환 및 환불 어휘 (交換・払戻しの語彙)

| 한국어 | 일본어 | 예문 |
|--------|--------|------|
| 교환 (교환하다) | 交換 (交換する) | 다른 색으로 교환하고 싶어요. |
| 환불 (환불하다) | 払い戻し (返金する) | 환불해 주세요. |
| 영수증 | 領収書 / レシート | 영수증이 있어요. |
| 점원 | 店員 | 점원이 친절해요. |
| 거스름돈 | お釣り | 거스름돈 여기 있습니다. |
| 바꾸다 | 바인딩 / 換える | 이걸로 바꿔 주세요. |

---

## 2. ~아/어 주다 (〜해 주다 - 依頼・お願い)

동사 뒤에 붙어 남을 위해 어떤 행동을 해 주거나, 다른 사람에게 행동을 부탁할 때 사용한다.

- 어간 모음이 `ㅏ, ㅗ` 일 때: **~아 주다** (예: 사다 → 사 주다)
- 그 외의 모음일 때: **~어 주다** (예: 바꾸다 → 바꿔 주다, 빌리다 → 빌려 주다)
- `하다`로 끝날 때: **~해 주다** (예: 환불하다 → 환불해 주다)

부탁할 때는 주로 **~아/어 주세요** 또는 **~아/어 주시겠어요?** (더 정중한 표현) 형태로 쓴다.

```text
이거 다른 걸로 바꿔 주세요.
  → これ、他のものに換えてください。

사이즈를 확인해 주시겠어요?
  → サイズを確認していただけますか？

영수증을 보여 주세요.
  → 領収書を見せてください.
```

---

## 3. ~기 전에 / ~(으)ㄴ 후에 (〜する前に / 〜한 후에)

행동의 시간 순서를 나타낼 때 사용하는 표현이다.

### 1) ~기 전에 (〜する前に)
동사 어간 뒤에 `~기 전에`를 붙인다. (받침 유무 상관없음)

- 가다 → 가기 전에
- 먹다 → 먹기 전에

```text
사기 전에 입어 보세요.
  → 買う前に着てみてください。

집에 가기 전에 영수증을 받았어요.
  → 家에 가기 전에 레시트를 받았습니다.
```

### 2) ~(으)ㄴ 후에 (〜한 후에)
동사 어간에 받침이 없거나 'ㄹ' 받침일 때는 `~ㄴ 후에`, 받침이 있을 때는 `~은 후에`를 쓴다.

- 사다 → 산 후에
- 먹다 → 먹은 후에
- 만들다 → 만든 후에 ('ㄹ' 탈락)

```text
밥을 먹은 후에 약을 드세요.
  → ご飯を食べた後に薬을 드세요.

물건을 산 후에 봉투에 넣었어요.
  → 品物を買った後に袋に入れました。
```

## 4. 회화 연습

**상황: 가게에서 물건 교환하기 (店での商品の交換)**

```text
아내: 저기요, 어제 이 셔츠를 샀는데 사이즈가 좀 작아요.
     (すみません、昨日 이 셔츠를 샀는데 사이즈가 좀 작아요.)

점원: 아, 그러세요? 영수증 가지고 오셨어요?
     ( 아, 그러세요? 領収書を持って来られましたか？)

아내: 네, 여기 있어요. 다른 사이즈로 교환해 주세요.
     (はい、ここにあります。他のサイズに交換してください。)

점원: 네, 어떤 사이즈로 바꿔 드릴까요? 한 단계 큰 사이즈로 드릴까요?
     (はい、どのサイズに換えましょうか？ワンサイズ大きいものにしましょうか？)

아내: 네, 그걸로 보여 주세요. 입어 본 후에 결정할게요.
     (はい、それを見せてください。着てみた後に決めます。)

점원: 네, 여기 있습니다. 피팅룸은 저기 왼쪽에 있어요.
     (はい、こちらにございます。フィッティングルームはあちらの左側にあります。)
```

## 練習

**빈칸에 알맞은 표현을 넣어 문장을 완성해 보세요. (空欄に合う表現を入れて文章を完成させてみましょう。)**

1. 한국에 ________ (가다) 전에 한국어를 공부해요.
2. 영화를 ________ (보다) 후에 저녁을 먹을 거예요.
3. 너무 매워요. 물 좀 ________ (주다) 주세요.
4. 사이즈가 안 맞아요. 이걸로 ________ (바꾸다) 주세요.

**정답 (答え)**
1. 가기
2. 본
3. 갖다 (또는 '물 좀 주세요') / 주다 활용 시: '물 좀 **줘요 / 주세요**'
4. 바꿔

## Tips

### 교환(交換)과 환불(払戻し)의 조건
한국에서는 보통 물건을 산 날로부터 **7일 이내**에 영수증을 가지고 가야 교환이나 환불이 가능하다.
태그(Tag)를 제거했거나 물건을 사용한 후에는 교환이나 환불이 어려울 수 있으니 주의하자!
